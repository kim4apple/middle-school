## 一、项目概述

参照 math7.html 的完整结构模板，创建单文件 HTML 页面 math8.html，涵盖八年级数学上下两册全部知识点。

## 二、技术方案

| 项目 | 方案 |
|------|------|
| **交付物** | 单文件 `math8.html` |
| **模板来源** | 复制 `math7.html` 的 CSS/JS，替换内容 |
| **公式渲染** | Katex 本地文件（共用 katex/ 目录） |
| **几何图示** | 内嵌 SVG（全等三角形、平行四边形、函数图象等） |

## 三、内容范围（10 章）

### 上册（5 章）
- 第11章：三角形
- 第12章：全等三角形（含几何证明入门）
- 第13章：轴对称
- 第14章：整式乘法与因式分解
- 第15章：分式

### 下册（5 章）
- 第16章：二次根式
- 第17章：勾股定理
- 第18章：平行四边形
- 第19章：一次函数
- 第20章：数据的分析
