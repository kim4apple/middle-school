第2章 · 空气·氧气

九年级上册 | 实验驱动 · 物质性质 | 4个化学方程式 + 空气成分计算

🧪 实验一：红磷燃烧测定空气中氧气含量
1在集气瓶中加入少量水，将水面上方空间分为5等份，夹紧弹簧夹。
2点燃燃烧匙内的红磷，迅速伸入集气瓶中，塞紧瓶塞。观察：红磷燃烧产生大量白烟（P₂O₅固体小颗粒），放出热量。
3火焰熄灭、冷却至室温后，打开弹簧夹——烧杯中的水沿导管倒吸进入集气瓶，进入水的体积约占集气瓶内空气体积的1/5。

📋 结论：氧气约占空气体积的1/5。红磷燃烧消耗了集气瓶中的氧气，瓶内压强减小，水被大气压压入瓶内，进入水的体积 = 消耗掉的氧气体积。

🧪 实验：氧气的实验室制取（高锰酸钾分解法）

📋 操作要点：

① 试管口略向下倾斜（防止冷凝水倒流炸裂试管）

② 导管伸入集气瓶的口部即可（不宜伸入太长）

③ 实验结束先撤导管再熄灭酒精灯（防止水倒吸）

④ 用排水法收集——氧气不易溶于水

🧪 实验三：氧气的实验室制取（两套方案对比）
方案方案A：高锰酸钾分解方案B：过氧化氢分解
原理2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑2H₂O₂ → 2H₂O + O₂↑（MnO₂作催化剂）
装置固固加热型（试管口略向下倾斜，管口放一团棉花防止粉末进入导管）固液不加热型（长颈漏斗下端管口必须伸入液面以下——液封防漏气）
收集方法排水法（O₂不易溶于水）或向上排空气法（O₂密度比空气略大）排水法或向上排空气法
优点无需催化剂，原料稳定不需加热，操作简便，反应速率可控（通过控制H₂O₂的滴加速度）

📋 误差分析（红磷实验）：结果偏小——①红磷不足（氧气未被全部消耗）②装置漏气（外界空气进入）③未冷却就打开弹簧夹（气体受热膨胀）。结果偏大——①燃烧匙伸入过慢（瓶内空气受热逸出）②弹簧夹未夹紧（红磷燃烧时空气受热从导管逸出）。

🧠 空气是混合物——成分比例精确记忆

空气不是单一物质，而是由多种气体组成的混合物。各成分按体积分数计：

成分体积分数性质与用途
氮气 N₂78%化学性质不活泼，用作保护气、食品充氮防腐、液氮冷冻
氧气 O₂21%供给呼吸、支持燃烧；是化学性质最活泼的空气主要成分
稀有气体0.94%He、Ne、Ar、Kr、Xe——通电发不同颜色光，作霓虹灯、航标灯；化学性质极不活泼
二氧化碳 CO₂0.03%光合作用原料；含量升高导致温室效应
其他气体和杂质0.03%水蒸气、稀有气体之外的少量气体、尘埃等

空气成分的测定无法用一种物质一次性完成——氧气的含量用燃烧法（红磷/铜粉），这说明混合物中各组分保持各自的性质，这是混合物区别于化合物的关键。

📖 核心概念

空气的组成

空气是混合物，主要成分为 N₂(78%)、O₂(21%)、稀有气体(0.94%)、CO₂(0.03%)、其他气体和杂质(0.03%)。各成分保持各自的化学性质。

氧气的物理性质

通常状况下为无色无味气体，密度比空气略大（1.429 g/L），不易溶于水（可用排水法收集）。液态氧和固态氧为淡蓝色。

氧气的化学性质

氧气化学性质比较活泼，具有氧化性。①与碳反应：C + O₂ → CO₂，空气中发红光，氧气中发白光；②与硫反应：S + O₂ → SO₂，空气中淡蓝色火焰，氧气中蓝紫色火焰；③与铁反应：3Fe + 2O₂ → Fe₃O₄，空气中红热不燃烧，氧气中剧烈燃烧、火星四射、放出大量热、生成黑色固体。

催化剂

在化学反应中能改变其他物质的反应速率，而本身的质量和化学性质在反应前后都没有发生变化的物质。MnO₂不是任何反应的催化剂——催化剂具有专一性。

化合反应

由两种或两种以上物质生成另一种物质的反应。特点："多变一"——A + B + … → X。如：C + O₂ → CO₂，S + O₂ → SO₂，3Fe + 2O₂ → Fe₃O₄。

氧化反应

物质与氧气发生的反应。包括剧烈氧化（如燃烧）和缓慢氧化（如铁生锈、食物腐烂、呼吸作用）。缓慢氧化可能引起自燃——热量积累使温度达到着火点。

⚗️ 化学方程式

① 碳在氧气中燃烧：C + O₂ → CO₂（化合反应，也是氧化反应）

② 硫在氧气中燃烧：S + O₂ → SO₂（化合反应，也是氧化反应）

③ 铁在氧气中燃烧：3Fe + 2O₂ → Fe₃O₄（化合反应，也是氧化反应）

④ 高锰酸钾分解制氧气：2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑（分解反应）

⑤ 过氧化氢分解制氧气：2H₂O₂ → 2H₂O + O₂↑（MnO₂作催化剂，分解反应）

🔢 空气成分计算

核心原理：进入集气瓶中水的体积 = 被消耗掉的氧气体积

例题1：用红磷燃烧法测定空气中氧气含量。集气瓶容积为250 mL，实验结束后进入集气瓶的水的体积约为多少？

解：进入水的体积 = 氧气体积 ≈ 250 mL × 1/5 = 50 mL

例题2：实验中烧杯内的水实际进入45 mL，分析可能的原因。

答：45 mL < 50 mL，结果偏小。可能原因：①红磷量不足，瓶内氧气未完全耗尽；②装置气密性不好，外界空气进入；③未等装置冷却至室温就打开了弹簧夹。

例题3：100 L空气中，氮气、氧气、稀有气体、二氧化碳的体积各约为多少？

解：N₂: 100 L × 78% = 78 L；O₂: 100 L × 21% = 21 L；稀有气体: 100 L × 0.94% = 0.94 L；CO₂: 100 L × 0.03% = 0.03 L

🚨 实验安全与常见错误

🔴 O₂的检验 vs 验满

检验方法：将带火星的木条伸入集气瓶中，若木条复燃，证明是氧气。
验满方法：将带火星的木条放在集气瓶口，若木条复燃，证明已收集满。
关键区别：检验伸入瓶中，验满放在瓶口！

⚠️ 催化剂"一变两不变"

❌ "催化剂不参加反应"——错！催化剂参与反应，但反应后质量不变。
❌ "MnO₂是所有反应的催化剂"——错！催化剂的催化作用有专一性。
❌ "催化剂只加快反应速率"——错！也可以减慢（如防腐剂）。
✅ 催化剂"一变两不变"：改变反应速率，质量和化学性质不变。物理性质可能改变（如颗粒大小变化）。

下列关于催化剂的叙述正确的是（ ）
A. 催化剂在化学反应中不参加反应   B. 催化剂一定能加快反应速率
C. 反应前后催化剂的质量和化学性质都不变   D. MnO₂可以用作任何反应的催化剂

实验室用高锰酸钾制取氧气，下列操作正确的是（ ）
A. 试管口向上倾斜   B. 导管口开始冒气泡时立即收集
C. 实验结束时先熄灭酒精灯，后从水槽中撤出导管   D. 用排水法收集时，当集气瓶口有气泡冒出时证明已收集满
答案：D
🔑 关键思路：A错：试管口应略向下倾斜（防止冷凝水倒流炸裂试管）。B错：刚开始冒出的气泡是空气，应等气泡连续均匀冒出再收集。C错：应先撤导管再灭酒精灯（防止水倒吸炸裂试管）。D对：排水法验满的标志。

九上★★★
某同学用红磷燃烧法测定空气中氧气含量，结果进入集气瓶的水的体积小于1/5。请分析可能的原因（至少写3条）。
答案：①红磷量不足——集气瓶内氧气未被全部消耗完；②装置漏气——外界空气进入集气瓶补充了压强；③未等装置冷却至室温就打开弹簧夹——瓶内气体仍受热膨胀，压强差较小。
⚠️ 易错：答"导管内有水残留"也算偏小原因。答"燃烧匙伸入太慢导致空气逸出"会让结果偏大而非偏小——注意审题方向！

🔬 实验探究设计：证明MnO₂是H₂O₂分解的催化剂

设计思路：验证催化剂的"一变两不变"——必须设计控制变量实验。

① 验证"改变速率"：取两支试管，各加入3 mL 5% H₂O₂溶液。试管A加入少量MnO₂，试管B不加。观察——A迅速产生大量气泡（带火星木条复燃），B几乎无气泡。证明MnO₂加快了H₂O₂分解速率。

② 验证"质量不变"：精确称量MnO₂质量m₁ → 加入过量H₂O₂中反应 → 反应完全后，过滤、洗涤、干燥剩余固体 → 称量得m₂ → 比较m₁和m₂ → m₁ = m₂。

③ 验证"化学性质不变"：将步骤②中回收的MnO₂再次加入新的H₂O₂溶液中，仍能加快H₂O₂分解速率。

📋 结论： MnO₂能改变H₂O₂分解速率，反应前后自身的质量和化学性质不变——MnO₂是H₂O₂分解的催化剂。
