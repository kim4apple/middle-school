# 初中化学自学资料 — 需求规格说明

> 版本: v1.0 | 状态: 需求确认中 | 学科: 化学 | 教材: 人教版九年级

---

## 一、项目目标

为九年级学生创建一套**零基础到中考满分**的化学自学资料，特色：
1. **实验驱动教学** — 从观察到理解，而非从定义到做题
2. **三重表征可视化** — 宏观现象↔微观粒子↔符号语言 三条线并行
3. **3D 分子互动** — 用 Three.js 呈现分子结构、晶体模型、电子转移
4. **单 HTML 文件** — 完全离线可用，支持暗色模式、打印 PDF

---

## 二、章节结构：10 模块实验驱动模型

```
每章（共 12 章，九年级上下册）：

┌─ 1. 🧪 实验观察         experiment-observation
│     现象描述 + 实验视频/动画 + 操作步骤 + 安全警示
├─ 2. 🧠 微观解释         micro-explanation
│     粒子模型图 + Three.js 3D 分子 + 电子转移动画
├─ 3. 📖 核心概念         concept-cards × 4-6
│     在观察和解释之后才给定义（概念才有意义）
├─ 4. ⚗️ 化学方程式       equation-cards × 3-5
│     配平动画 + 反应类型标注 + 条件/箭头规范
├─ 5. 🔢 计算专题         calculation-cards × 2-3
│     相对原子质量/化学方程式计算/溶液浓度/溶解度
├─ 6. 🚨 实验安全+易错    safety-pitfalls
│     操作安全 + 概念混淆 + 方程式书写陷阱
├─ 7. 🎯 中考真题         exam-questions × 3-5
│     含实验探究题、推断题、计算题
├─ 8. 🔬 实验探究设计     experiment-design
│     设计实验方案 + 变量控制 + 误差分析
├─ 9. ✅ 自查清单+间隔复习 checklist + review
└─ 10. ➡️ 下一站预告      next-stop
```

**与数学/物理 13 模块的关键区别**：
- 实验从辅助模块提升到**教学起点**（模块 1）
- 新增**微观解释**（模块 2）——化学独有的粒子视角
- 新增**化学方程式**独立模块（模块 4）——配平是独立技能
- 新增**实验安全+易错**合并模块（模块 6）
- 新增**实验探究设计**（模块 8）——中考必考 8-10 分

## 三、功能需求

### FR-1: 实验观察模块
- FR-1.1: 每个实验有完整的步骤说明（操作→现象→结论）
- FR-1.2: 关键实验用 CSS/Anime.js 动画展示操作流程（如气体制取装置组装）
- FR-1.3: 现象描述用颜色标注（溶液变红/产生气泡/生成沉淀）
- FR-1.4: 安全警示用 🚨 红色边框高亮

### FR-2: 微观解释模块（Three.js）
- FR-2.1: 分子 3D 模型（H₂O, CO₂, CH₄ 正四面体, C₆₀ 足球烯）
- FR-2.2: 晶体结构 3D 展示（NaCl 离子晶体、金刚石网状、石墨层状）
- FR-2.3: 电子转移/共用电子对动画（氧化还原、共价键形成）
- FR-2.4: 化学反应过程的粒子重排动画
- FR-2.5: 3D 模型可旋转、缩放（鼠标拖拽/触屏手势）
- FR-2.6: Three.js 按需加载——只在有 3D 内容的章节加载

### FR-3: 化学方程式模块
- FR-3.1: 方程式配平的步进动画（逐步添加系数）
- FR-3.2: 反应类型标注（化合/分解/置换/复分解）
- FR-3.3: 反应条件用标准符号（△/点燃/催化剂/MnO₂）
- FR-3.4: 常见方程式快速记忆卡片

### FR-4: 计算专题
- FR-4.1: 相对原子质量/相对分子质量计算
- FR-4.2: 化学方程式计算（已知一种物质求另一种）
- FR-4.3: 溶液中溶质质量分数计算
- FR-4.4: 溶解度曲线读图
- FR-4.5: 每道题含完整的 IDEAL 解析（关键思路+步骤+常见错误+检验）

### FR-5: 导航与 UI（复用模板规范）
- FR-5.1: 侧边栏（🏠首页 + 章节列表 + 暗色模式 + 打印 + 展开全部）
- FR-5.2: 暗色模式（localStorage 持久化）
- FR-5.3: 侧边栏隐藏/显示 + 回到顶部
- FR-5.4: 打印 PDF 优化（自动展开所有内容 + 隐藏侧栏）
- FR-5.5: 移动端响应式（768px/480px 断点）
- FR-5.6: 快速导航 TOC

### FR-6: 化学专用视觉
- FR-6.1: 溶液颜色色块（CuSO₄ 蓝色、FeCl₃ 黄色、KMnO₄ 紫色等）
- FR-6.2: 沉淀符号 ↓ 和气体符号 ↑ 在方程式中高亮
- FR-6.3: pH 试纸颜色渐变条
- FR-6.4: 元素周期表可交互区域（初中常用 1-20 号元素）

## 四、非功能需求

### NFR-1: 离线可用
- 单 HTML 文件（每章独立）+ 共享 CSS/JS 库目录
- 不依赖 CDN（Three.js 和 KaTeX 本地化）

### NFR-2: 性能
- Three.js 仅在有 3D 内容的章节按需加载
- 2D 动画优先用 CSS 过渡或轻量 Anime.js
- KaTeX 使用本地文件，不发起网络请求

### NFR-3: 兼容性
- Chrome/Safari/Firefox 现代浏览器
- Three.js WebGL 不支持时降级为静态 SVG 图
- 打印时 3D 模型降级为静态截图

### NFR-4: 文件大小
- 每章 HTML ≤ 2000 行（不含共享库）
- Three.js 场景代码内联，每个 3D 场景 ≤ 100 行 JS
- 共享 CSS 文件 ≤ 600 行

## 六、技术选型（待确认）

## 七、待用户确认的问题

1. **文件结构**：每章独立 HTML（12 个）+ 共享目录？还是上册/下册各一个大文件？
2. **3D 内容密度**：每章 1-2 个 Three.js 场景即可，还是需要更多？
3. **实验动画**：用 Anime.js 做步骤动画，还是用 SVG 序列图更简单？
4. **元素周期表**：需要完整的交互式周期表，还是只展示前 20 号元素？
5. **优先级**：是否先完成上册 7 章（不含 3D）试用，再迭代添加 3D？
