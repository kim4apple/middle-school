第11章 · 盐·化肥

九年级下册 | 复分解反应 · 化肥分类 · 粗盐提纯 | 核心：离子反应的条件与鉴别

★★★★☆ 综合应用 📊 中考高频 推断题和实验探究题的压轴常客
🧪 实验：粗盐提纯（溶解 → 过滤 → 蒸发 → 结晶）
1称量与溶解：用托盘天平称取约 5 g 粗盐，放入烧杯中，加入约 20 mL 水，用玻璃棒搅拌加速溶解。
2过滤：将滤纸对折两次放入漏斗中，用水润湿使其紧贴漏斗壁。将粗盐水沿玻璃棒引流倒入漏斗中——不溶性杂质（泥沙）留在滤纸上，澄清滤液流入烧杯。
3蒸发：将滤液倒入蒸发皿中，用酒精灯加热并用玻璃棒不断搅拌——防止局部温度过高导致液体飞溅。当蒸发皿中出现较多固体时停止加热，利用余热蒸干。
4转移：用玻璃棒将蒸发皿中的 NaCl 晶体转移到指定容器中。

📋 仪器：托盘天平、药匙、烧杯、量筒、玻璃棒、铁架台（带铁圈）、漏斗、滤纸、蒸发皿、酒精灯、坩埚钳。

🟡 Na⁺ · 🟢 Cl⁻
🖱️ 旋转/缩放 · 切换暗色请刷新

NaCl 晶体中，每个 Na⁺ 被 6 个 Cl⁻ 包围，每个 Cl⁻ 也被 6 个 Na⁺ 包围。Na⁺ 和 Cl⁻ 通过离子键（正负电荷吸引力）结合。离子晶体硬度大、熔点高，熔融状态或水溶液中能导电（离子可以自由移动）。

📝 本节必背方程式（6个）
1$Na_2CO_3 + 2HCl \rightarrow 2NaCl + H_2O + CO_2\uparrow$ 检验CO₃²⁻
2$NaHCO_3 + HCl \rightarrow NaCl + H_2O + CO_2\uparrow$ 胃酸治疗
3$CaCO_3 + 2HCl \rightarrow CaCl_2 + H_2O + CO_2\uparrow$ 实验室制CO₂
4$AgNO_3 + NaCl \rightarrow AgCl\downarrow + NaNO_3$ 检验Cl⁻
5$BaCl_2 + Na_2SO_4 \rightarrow BaSO_4\downarrow + 2NaCl$ 检验SO₄²⁻
6$NH_4NO_3 + NaOH \rightarrow NaNO_3 + NH_3\uparrow + H_2O$ 铵态氮肥检验

🔬 本节实验中考考点
  • Cl⁻检验：待测液+AgNO₃溶液→白色沉淀→再加稀HNO₃，沉淀不溶解（排除CO₃²⁻干扰）
  • SO₄²⁻检验：待测液+BaCl₂溶液→白色沉淀→再加稀HNO₃，沉淀不溶解
  • 铵态氮肥检验：加NaOH研磨/加热→湿润红色石蕊试纸变蓝（产生NH₃）
  • CO₃²⁻检验：加稀盐酸→产生的气体通入澄清石灰水→变浑浊
📋 中考真题示例
（2023 · 武汉）某白色固体可能含有Na₂CO₃、NaCl、BaCl₂、Na₂SO₄中的一种或几种。取样加水溶解，得到白色沉淀和无色溶液。向沉淀中加入足量稀硝酸，沉淀部分溶解。
则该固体中一定含有______。
答案：Na₂CO₃、BaCl₂、Na₂SO₄ 白色沉淀→BaCO₃或BaSO₄；部分溶于稀硝酸→既有BaCO₃（溶解）又有BaSO₄（不溶）→所以Na₂CO₃和Na₂SO₄和BaCl₂都有

📖 核心概念

常见盐

NaCl（氯化钠/食盐）：调味品、防腐剂、工业制钠和氯气。Na₂CO₃（碳酸钠/纯碱/苏打）：注意：它是盐不是碱！水溶液呈碱性。用于玻璃、造纸。NaHCO₃（碳酸氢钠/小苏打）：发酵粉主要成分、治疗胃酸过多。CaCO₃（碳酸钙）：大理石/石灰石的主要成分，建筑材料。

复分解反应及条件

定义：两种化合物互相交换成分，生成另外两种化合物的反应。通式：AB + CD → AD + CB。发生条件（三者至少满足其一）：① 生成沉淀（↓）；② 生成气体（↑）；③ 生成水（或弱电解质）。三者都无，反应不能发生。反应物须可溶（有酸参加时例外——酸能与不溶性碱/盐反应）。

化肥——N、P、K

氮肥（N）：促进茎叶生长茂盛，叶色浓绿。如：尿素 CO(NH₂)₂、NH₄NO₃、NH₄HCO₃。磷肥（P）：促进根系发达、籽粒饱满。如：过磷酸钙 Ca(H₂PO₄)₂。钾肥（K）：增强抗倒伏、抗病虫害能力。如：KCl、K₂SO₄。复合肥：含 N+P+K 两种以上，如 KNO₃（K+N）、NH₄H₂PO₄（N+P）。

常见化肥的简易鉴别

① 外观：氮肥、钾肥多为白色晶体，磷肥多为灰白色粉末。② 溶解性：氮肥、钾肥全部溶于水，磷肥部分溶或不溶。③ 加碱——铵态氮肥（NH₄⁺）与碱共热会放出NH₃（刺激性气味，使湿润红色石蕊试纸变蓝）。④ 钾肥加碱无此现象。

粗盐中可溶性杂质的去除

粗盐除含泥沙（不溶性杂质）外，还含 CaCl₂、MgCl₂、Na₂SO₄ 等可溶性杂质。去除方法：依次加入过量 BaCl₂（除 SO₄²⁻）、NaOH（除 Mg²⁺）、Na₂CO₃（除 Ca²⁺ 和多余 Ba²⁺），过滤后再加适量稀盐酸中和多余的 NaOH 和 Na₂CO₃，最后蒸发结晶得纯 NaCl。

⚖️ 复分解反应发生条件判断

复分解反应通式：AB + CD → AD + CB（两种化合物互相交换成分）

发生的条件（满足其一即可）：

✅ 生成沉淀↓（如 AgCl↓、BaSO₄↓、CaCO₃↓、Mg(OH)₂↓、Cu(OH)₂↓）

✅ 生成气体↑（如 CO₂↑、NH₃↑）

✅ 生成水 H₂O（中和反应，如 HCl+NaOH→NaCl+H₂O）

判断口诀：两交换、看产物——有沉淀/气体/水→发生；无→不发生。

⚠️ 碱+盐、盐+盐 反应时，反应物必须都可溶！

🌱 常见化肥简易鉴别（一看二溶三加碱）

化肥外观加水加熟石灰研磨类别
尿素 CO(NH₂)₂白色颗粒溶解无氨味有机氮肥
NH₄HCO₃白色晶体溶解有刺激性氨味↑铵态氮肥
Ca(H₂PO₄)₂灰白色粉末难溶无氨味磷肥
KCl白色晶体溶解无氨味钾肥

核心原理：铵态氮肥(NH₄⁺) + 碱(OH⁻) —加热→ 放出NH₃↑（使湿润红色石蕊试纸变蓝）。尿素是有机物，不含NH₄⁺→加碱不放氨气。

⚗️ 本章核心化学方程式

#化学方程式反应类型考点
1$Na_2CO_3 + 2HCl \rightarrow 2NaCl + H_2O + CO_2\uparrow$复分解反应有气体生成；纯碱与酸反应——检验 CO₃²⁻
2$NaHCO_3 + HCl \rightarrow NaCl + H_2O + CO_2\uparrow$复分解反应小苏打治胃酸过多的原理
3$AgNO_3 + NaCl \rightarrow AgCl\downarrow + NaNO_3$复分解反应白色沉淀（AgCl 不溶于稀硝酸）——检验 Cl⁻
4$BaCl_2 + Na_2SO_4 \rightarrow BaSO_4\downarrow + 2NaCl$复分解反应白色沉淀（BaSO₄ 不溶于稀硝酸）——检验 SO₄²⁻
5$NH_4HCO_3 \xrightarrow{\Delta} NH_3\uparrow + H_2O + CO_2\uparrow$分解反应碳铵分解——铵态氮肥应低温密封保存

🔢 计算专题：化肥中营养元素的质量分数

核心思路：计算化肥中 N、P₂O₅、K₂O 的含量，判断化肥纯度。

【例题】计算 NH₄NO₃（硝酸铵）中氮元素的质量分数。某硝酸铵样品含氮量实际为 33.6%，求样品纯度。

解：$Mr(NH_4NO_3) = 14 \times 2 + 1 \times 4 + 16 \times 3 = 80$
N 的质量分数（理论值）= $\dfrac{14 \times 2}{80} \times 100\% = 35\%$
样品纯度 = $\dfrac{\text{实际 N 含量}}{\text{理论 N 含量}} \times 100\% = \dfrac{33.6\%}{35\%} \times 100\% = 96\%$
答：NH₄NO₃ 中 N 的理论含量为 35%，样品纯度为 96%。

🚨 安全与常见错误

🔴 核心辨析

Na₂CO₃ 俗名"纯碱"，但它是盐不是碱！它的水溶液显碱性是因为 CO₃²⁻ 水解产生 OH⁻（初中不要求理解水解，但须知道它是盐）。NaHCO₃（小苏打）也是盐。
铵态氮肥不能与碱性物质混用——NH₄⁺ + OH⁻ → NH₃↑ + H₂O，NH₃ 逸出导致氮元素损失，降低肥效。

⚠️ 最高频易错

❌ 判断复分解反应是否发生——三个条件（气体/沉淀/水）至少满足其一。
❌ 复分解反应中，反应物一般要求可溶（有酸参加时例外）。
❌ 常见不溶性盐：AgCl（白）、BaSO₄（白）——都不溶于稀硝酸。
❌ "硝"字含 N 如 KNO₃——既是钾肥又是氮肥（复合肥）。
❌ 蒸发操作中，不可将水分完全蒸干再停止加热——防止固体飞溅。

🎯 中考真题精选（盐·化肥 · 近年典型）
九下★☆☆
下列物质中，属于盐的是（ ）
A. NaOH   B. Na₂CO₃   C. H₂SO₄   D. CaO
答案：B
🔑 关键思路：NaOH 是碱、H₂SO₄ 是酸、CaO 是氧化物。Na₂CO₃ 是盐（金属离子 Na⁺ + 酸根离子 CO₃²⁻），虽然俗名"纯碱"。
💡 显示/隐藏解析
九下★★☆
下列各组物质混合后，能发生复分解反应的是（ ）
A. NaCl + KNO₃   B. Na₂CO₃ + HCl   C. Fe + CuSO₄   D. CO₂ + NaOH
答案：B
🔑 关键思路：A 交换后无沉淀/气体/水→不反应；B 生成 CO₂↑ → 发生反应；C 是置换反应不是复分解；D 不是复分解（CO₂ 非酸）。
💡 显示/隐藏解析
九下★★☆
铵态氮肥不能与碱性物质混用的原因是（ ）
A. 降低肥效   B. 生成有毒物质   C. 发生燃烧   D. 生成沉淀
答案：A
🔑 关键思路：铵态氮肥含 NH₄⁺，与碱（OH⁻）反应生成 NH₃ 气体逸出——氮元素损失 → 降低肥效。
💡 显示/隐藏解析
九下★★★
某白色固体可能是 Na₂CO₃、NaCl、Na₂SO₄ 中的一种或几种。(1) 溶于水，滴加 BaCl₂ 溶液，产生白色沉淀；(2) 向沉淀中加入足量稀硝酸，沉淀部分溶解并产生气泡。判断该白色固体的组成，写出相关化学方程式。
答案：一定含 Na₂CO₃ 和 Na₂SO₄，可能含 NaCl。含 Na₂CO₃→与 HCl 产生 CO₂（气泡），生成的 BaCO₃ 溶于稀硝酸；含 Na₂SO₄→生成 BaSO₄ 不溶于稀硝酸（沉淀部分溶解的原因）。方程式：$Na_2CO_3 + BaCl_2 \rightarrow BaCO_3\downarrow + 2NaCl$；$BaCO_3 + 2HNO_3 \rightarrow Ba(NO_3)_2 + H_2O + CO_2\uparrow$；$Na_2SO_4 + BaCl_2 \rightarrow BaSO_4\downarrow + 2NaCl$。
💡 显示/隐藏解析

🔬 实验探究设计：鉴别 (NH₄)₂SO₄ 和 K₂SO₄

设计思路：利用铵根离子（NH₄⁺）的特性——加碱加热放出 NH₃。

方法：取两种白色固体各少量，分别放入两支试管，加入 NaOH 溶液后加热。

现象与判断：

① 试管口放置湿润的红色石蕊试纸——若试纸变蓝→ 该固体为 (NH₄)₂SO₄（产生 NH₃）。
② 无此现象/K₂SO₄ 试纸不变蓝。

化学方程式：$(NH_4)_2SO_4 + 2NaOH \xrightarrow{\Delta} Na_2SO_4 + 2H_2O + 2NH_3\uparrow$

补充：也可直接闻气味——(NH₄)₂SO₄ 与碱研磨后有刺激性气味（NH₃），K₂SO₄ 无气味。
