$-\dfrac{1}{3}$ 的倒数是（ ）
A. $3$   B. $-3$   C. $\dfrac{1}{3}$   D. $-\dfrac{1}{3}$
答案：B

🔑 关键思路：倒数定义——乘积为 $1$ 的两个数。$(-\dfrac{1}{3}) \times (-3) = 1$。

  1. $-\dfrac{1}{3}$ 的倒数为 $\dfrac{1}{-\frac{1}{3}} = -3$

⚠️ 常见错误：混淆"倒数"与"相反数"——相反数是 $\dfrac{1}{3}$。

8年级模拟卷2★★☆
三角形的两边长分别为 $3$ 和 $7$，则第三边的长可能是（ ）
A. $2$   B. $4$   C. $8$   D. $11$
答案：C

🔑 关键思路：三角形两边之和大于第三边，两边之差小于第三边。$|7-3| < x < 7+3$ → $4 < x < 10$。

  1. $4 < x < 10$，选项中只有 $8$ 满足
  2. $2$ 不满足（$2 < 4$），$4$ 不满足（等于左边），$11$ 不满足（$11 > 10$）

⚠️ 常见错误：只记住"两边之和大于第三边"，忘了"两边之差小于第三边"。

7年级模拟卷2★★☆
下列各数中，最大的无理数是（ ）
A. $\sqrt{2}$   B. $\sqrt{3}$   C. $\pi$   D. $\sqrt{5}$
答案：C

🔑 关键思路：比较无理数大小：$\pi \approx 3.14$，$\sqrt{2}\approx 1.414$，$\sqrt{3}\approx 1.732$，$\sqrt{5}\approx 2.236$。$\pi$ 最大。

  1. $\sqrt{2} \approx 1.414$，$\sqrt{3} \approx 1.732$，$\sqrt{5} \approx 2.236$，$\pi \approx 3.142$
  2. $3.142$ 最大 → $\pi$ 最大

⚠️ 常见错误：不熟悉 $\pi$ 的近似值，误选 $\sqrt{5}$。

8年级模拟卷2★★☆
若一次函数 $y=kx+b$ 的图像不经过第二象限，则 $k$ 和 $b$ 的符号是（ ）
A. $k>0$，$b<0$   B. $k>0$，$b>0$   C. $k<0$，$b>0$   D. $k<0$，$b<0$
答案：A

🔑 关键思路：不经过第二象限 → 图像过一、三、四象限或一、三象限。这要求 $k>0$（向上倾斜），且 $b \leq 0$（在 $y$ 轴截距非正）。选项中只有 A 满足。

  1. $k>0$：直线左下→右上，必过一、三象限
  2. $b<0$：与 $y$ 轴交于负半轴，过第四象限
  3. 组合：过一、三、四象限，不过二 ✓

⚠️ 常见错误：混淆 $k$ 符号与象限过不过的关系，凭直觉选。

8年级模拟卷2★★☆
能判定 $\triangle ABC$ 为直角三角形的是（ ）
A. $\angle A + \angle B = 90^\circ$
B. $AB=3$，$BC=4$，$AC=5$
C. A 和 B 都对
D. $\angle A = \angle B = \angle C$
答案：C

🔑 关键思路：A：两锐角互余 → 第三个角为 $90^\circ$，是直角三角形。B：$3^2+4^2=5^2$，满足勾股定理逆定理，是直角三角形。

  1. A：若 $\angle A + \angle B = 90^\circ$，则 $\angle C = 180^\circ - 90^\circ = 90^\circ$ → 直角△ ✓
  2. B：$3^2+4^2=9+16=25=5^2$ → 勾股定理逆定理 → 直角△ ✓
  3. D：$\angle A=\angle B=\angle C=60^\circ$ → 等边三角形，非直角

⚠️ 常见错误：只选 A 或 B 中的一个，忽略了"都对"选项。

8年级模拟卷2★★☆
关于矩形的对角线，下列说法正确的是（ ）
A. 垂直   B. 相等   C. 互相平分   D. B 和 C 都对
答案：D

🔑 关键思路：矩形是特殊的平行四边形，对角线互相平分（所有平行四边形都有此性质）且相等（矩形特有）。矩形对角线不一定垂直（菱形才垂直）。

  1. 矩形是平行四边形 → 对角线互相平分 ✓
  2. 矩形对角线相等（可用全等三角形证明）✓
  3. 矩形对角线不垂直（垂直是菱形的性质）

⚠️ 常见错误：混淆矩形、菱形、正方形的对角线性质。

8年级模拟卷2★★★
方程 $\dfrac{x}{x-2} = \dfrac{3}{x-2} + 1$ 的解是（ ）
A. $x=2$   B. $x=3$   C. 无解   D. $x=2$ 或 $x=3$
答案：C

🔑 关键思路：去分母解分式方程，注意增根。$x=2$ 使分母为零 → 是增根。

  1. 两边同乘 $(x-2)$：$x = 3 + (x-2)$
  2. 化简：$x = x + 1$ → $0 = 1$，矛盾！
  3. 方程无解
  4. 也可从另一角度：$x \neq 2$，但化简后导出矛盾，故无解

⚠️ 常见错误：① 去分母后得到 $x=x+1$ 仍往下推 ② 不检验直接选 $x=2$。

8年级模拟卷2★★★
已知 $a+b=5$，$ab=6$，则 $a^2+b^2$ 的值为（ ）
A. $13$   B. $25$   C. $19$   D. $37$
答案：A

🔑 关键思路：整体代换——利用完全平方公式：$a^2+b^2 = (a+b)^2 - 2ab$。不求 $a$、$b$ 的具体值。

  1. $a^2 + b^2 = (a+b)^2 - 2ab$
  2. $= 5^2 - 2 \times 6 = 25 - 12 = 13$

⚠️ 常见错误：① 展开 $(a+b)^2$ 时漏掉 $2ab$ 项 ② 用求根公式求 $a$,$b$ 再代入（虽然也能做，但整体代换更简洁）。

答案：$a(a+1)(a-1)$

🔑 关键思路：先提公因式 $a$ → 剩下 $a^2-1$ 是平方差公式 $(a+1)(a-1)$。

8年级模拟卷2★★★
分式 $\dfrac{x^2-1}{x^2+x}$ 约分后的结果为 ______（注意注明 $x$ 的取值范围）
答案：$\dfrac{x-1}{x}$（$x \neq 0$ 且 $x \neq -1$）

🔑 关键思路：先因式分解分子分母，再约分。注意约分后定义域不变：原分母 $x^2+x=x(x+1) \neq 0$ → $x \neq 0$ 且 $x \neq -1$。

  1. 分子：$x^2-1 = (x+1)(x-1)$
  2. 分母：$x^2+x = x(x+1)$
  3. 原式 $= \dfrac{(x+1)(x-1)}{x(x+1)} = \dfrac{x-1}{x}$（约去 $x+1$）
  4. 定义域：$x \neq 0$ 且 $x \neq -1$

⚠️ 常见错误：① 约分后忘标 $x \neq -1$（因为约掉的因子 $x+1$ 不能为零）② 只写 $x \neq 0$。

8年级模拟卷2★★☆
一次函数的图像经过点 $(-1, 0)$ 和 $(0, 2)$，求该函数图像与坐标轴围成的三角形面积。
答案：$1$

🔑 关键思路：两点恰好分别在 $x$ 轴和 $y$ 轴上，所以三角形面积直接用 $S=\dfrac{1}{2} \times |x\text{-截距}| \times |y\text{-截距}|$。

  1. $x$ 轴交点：$(-1, 0)$ → $x$-截距 $= -1$，$|OA| = 1$
  2. $y$ 轴交点：$(0, 2)$ → $y$-截距 $= 2$，$|OB| = 2$
  3. 验证：过 $(-1,0)$ 和 $(0,2)$ 的函数为 $y=2x+2$
  4. $S = \dfrac{1}{2} \times 1 \times 2 = 1$

⚠️ 常见错误：① $x$ 截距不取绝对值导致面积为负 ② 忘记乘 $\dfrac{1}{2}$。

【阅读材料】配方法：把 $x^2+px+q$ 写成 $(x+\dfrac{p}{2})^2 + (q-\dfrac{p^2}{4})$ 的形式。
【应用】(1) 求 $x^2-6x+10$ 的最小值及此时 $x$ 的值；
(2) 利用配方法解方程 $x^2-4x-5=0$。
答案：(1) 最小值为 $1$，此时 $x=3$   (2) $x_1=5$，$x_2=-1$

🔑 关键思路：配方——加上一次项系数一半的平方，同时减去同样的数以保持恒等。

  1. (1) $x^2-6x+10 = (x^2-6x+9) + 1 = (x-3)^2 + 1$
  2. $(x-3)^2 \geq 0$ → $(x-3)^2+1 \geq 1$，最小值 $=1$，此时 $x=3$
  3. (2) $x^2-4x-5=0$ → $x^2-4x+4 = 9$ → $(x-2)^2 = 9$
  4. $x-2 = \pm 3$ → $x_1=5$，$x_2=-1$

✅ 检验最小值：$x=3$ 代入→$9-18+10=1$ ✓

✅ 检验 (2)：$5^2-4\times 5-5=25-20-5=0$ ✓，$(-1)^2-4\times(-1)-5=1+4-5=0$ ✓

⚠️ 常见错误：① 配方时加了 $(\dfrac{p}{2})^2$ 忘了减 ② 开方时只取正根忘记负根。

(1) 猜想线段 $BE$、$CF$、$EF$ 之间的数量关系；
(2) 证明你的猜想。
答案：(1) $BE = AF$，$CF = AE$，且 $EF^2 = BE^2 + CF^2$
(2) 证明：连接 $AD$，证 $\triangle BDE \cong \triangle ADF$(SAS)，得 $BE=AF$；同理 $\triangle CDF \cong \triangle ADE$，得 $CF=AE$。在 $\text{Rt}\triangle AEF$ 中 $EF^2 = AE^2 + AF^2 = CF^2 + BE^2$。

🔑 关键思路：等腰直角三角形→$AD=BD=CD$ 且 $AD \perp BC$→用全等三角形证明线段关系。

  1. $AB=AC$，$\angle BAC=90^\circ$ → $\angle B = \angle C = 45^\circ$
  2. $D$ 是斜边 $BC$ 中点 → $AD=BD=CD=BC/2$，且 $AD \perp BC$，$\angle BAD=\angle CAD=45^\circ$
  3. 建系或几何证：由 $DE \perp DF$ → $\angle EDF = 90^\circ$，可推出 $\angle BDE = \angle ADF$
  4. 在 $\triangle BDE$ 和 $\triangle ADF$ 中：$BD=AD$，$\angle B = \angle DAF = 45^\circ$，$\angle BDE = \angle ADF$ → $\triangle BDE \cong \triangle ADF$(ASA)
  5. $BE = AF$；同理可证 $CF = AE$
  6. 在 $\text{Rt}\triangle AEF$ 中：$EF^2 = AE^2 + AF^2 = CF^2 + BE^2$

🔄 坐标法验证：设 $A(0,0)$，$B(1,0)$，$C(0,1)$，$D(1/2,1/2)$。设 $E(e,0)$，由 $DE \perp DF$ 解出 $F(0,1-e)$。则 $BE=|1-e|$，$AF=|1-e|$ → $BE=AF$；$CF=e$，$AE=e$ → $CF=AE$。

⚠️ 常见错误：① 不知道 $D$ 到三个顶点等距 ② 全等判定找错对应关系 ③ 坐标系建错导致关系混乱。
