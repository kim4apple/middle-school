7年级模拟卷1★☆☆
$-2$ 的相反数是（ ）
A. $2$   B. $-2$   C. $\dfrac{1}{2}$   D. $-\dfrac{1}{2}$
答案：A

🔑 关键思路：相反数的定义——只有符号不同的两个数互为相反数。$-2$ 的相反数就是 $2$。

  1. $-2$ 的相反数 $= -(-2) = 2$

⚠️ 常见错误：混淆"相反数"与"倒数"——倒数是 $\dfrac{1}{a}$，相反数是 $-a$。

7年级模拟卷1★☆☆
数据 $2, 3, 5, 5, 4$ 的众数是（ ）
A. $2$   B. $3$   C. $4$   D. $5$
答案：D

🔑 关键思路：众数是一组数据中出现次数最多的数。$5$ 出现 $2$ 次，其他各 $1$ 次。

  1. $2$ 出现 $1$ 次，$3$ 出现 $1$ 次，$5$ 出现 $2$ 次，$4$ 出现 $1$ 次
  2. 出现次数最多的是 $5$ → 众数为 $5$

8年级模拟卷1★★☆
一个正多边形的内角和为 $540^\circ$，则它的边数是（ ）
A. $4$   B. $5$   C. $6$   D. $7$
答案：B

🔑 关键思路：$n$ 边形内角和公式 $(n-2)\times 180^\circ = 540^\circ$，解出 $n$。

  1. $(n-2)\times 180^\circ = 540^\circ$
  2. $n-2 = \dfrac{540}{180} = 3$
  3. $n = 5$ → 正五边形

⚠️ 常见错误：混淆内角和与外角和（外角和恒为 $360^\circ$）。

点 $P(2, -3)$ 关于 $x$ 轴的对称点的坐标是（ ）
A. $(-2, -3)$   B. $(2, 3)$   C. $(-2, 3)$   D. $(-3, 2)$
答案：B

🔑 关键思路：关于 $x$ 轴对称：横坐标不变，纵坐标取相反数。$(2, -3) \rightarrow (2, 3)$。

8年级模拟卷1★★☆
等腰三角形顶角为 $80^\circ$，则底角的度数为（ ）
A. $80^\circ$   B. $50^\circ$   C. $40^\circ$   D. $100^\circ$
答案：B

🔑 关键思路：等腰三角形两底角相等。三角形内角和 $180^\circ$，底角 $= \dfrac{180^\circ - 80^\circ}{2} = 50^\circ$。

  1. 设底角为 $x$，则 $x + x + 80^\circ = 180^\circ$
  2. $2x = 100^\circ$ → $x = 50^\circ$

⚠️ 常见错误：直接用 $180^\circ - 80^\circ = 100^\circ$ 当作底角，忘了除以 $2$。

⚠️ 常见错误：只看 $b=1>0$ 就判断为增大，忽略了 $k$ 的符号才是单调性的决定因素。

8年级模拟卷1★★★
把 $x^2 - 4x - 12$ 分解因式，正确的是（ ）
A. $(x-6)(x+2)$   B. $(x+6)(x-2)$   C. $(x-3)(x+4)$   D. $(x-4)(x-3)$
答案：A

🔑 关键思路：十字相乘法：找两数 $m$, $n$ 使 $m+n=-4$, $mn=-12$。$m=-6$, $n=2$ 满足条件。

  1. 设 $x^2 - 4x - 12 = (x+m)(x+n)$
  2. 展开得 $x^2 + (m+n)x + mn$
  3. 所以 $\begin{cases} m+n = -4 \\ mn = -12 \end{cases}$
  4. 猜根：$m=-6$, $n=2$ → $(-6)+2=-4$ ✓，$(-6)\times 2=-12$ ✓
  5. $(x-6)(x+2)$

⚠️ 常见错误：符号搞反——写成 $(x+6)(x-2)$ 则 $m+n=4$，不符合。

💡 验证：展开 $(x-6)(x+2)=x^2+2x-6x-12=x^2-4x-12$ ✓

8年级模拟卷1★★★
在平行四边形 $ABCD$ 中，$AB=5$，$BC=3$，则对角线 $AC$ 的取值范围是（ ）
A. $2 < AC < 8$   B. $1 < AC < 9$   C. $2 < AC < 6$   D. $3 < AC < 8$
答案：A

🔑 关键思路：在 $\triangle ABC$ 中，$AB=5$, $BC=3$。由三角形两边之和大于第三边，两边之差小于第三边：$|5-3| < AC < 5+3$。

  1. 把 $AC$ 看作 $\triangle ABC$ 的第三边
  2. 三角形不等式：$|AB - BC| < AC < AB + BC$
  3. $|5-3| = 2$，$5+3 = 8$
  4. $2 < AC < 8$

⚠️ 常见错误：① 忘记三角形边长关系直接用平行四边形性质 ② 不等号方向写反。

8年级模拟卷1★★★
一次函数 $y=kx+b$ 的图像经过 $(1, 3)$ 和 $(2, 5)$，则 $k=$ ______，$b=$ ______
答案：$k=2$，$b=1$

🔑 关键思路：待定系数法——代入两点坐标，列方程组求 $k$ 和 $b$。

  1. 代入 $(1,3)$：$k+b=3$ ①
  2. 代入 $(2,5)$：$2k+b=5$ ②
  3. ②−①：$(2k+b)-(k+b)=5-3$ → $k=2$
  4. $k=2$ 代入①：$2+b=3$ → $b=1$

⚠️ 常见错误：代入坐标时 $x$ 和 $y$ 位置搞反。

8年级模拟卷1★★★
直角三角形的两边长分别为 $6$ 和 $8$，则第三边长为 ______
答案：$10$ 或 $2\sqrt{7}$

🔑 关键思路：分类讨论——$8$ 可能是斜边也可能是直角边。

  1. 若 $6$ 和 $8$ 都是直角边：斜边 $= \sqrt{6^2+8^2} = \sqrt{100}=10$
  2. 若 $8$ 是斜边，$6$ 是直角边：另一直角边 $= \sqrt{8^2-6^2} = \sqrt{64-36} = \sqrt{28}=2\sqrt{7}$

⚠️ 常见错误：① 不分类讨论，只给出 $10$ 一个答案 ② 误以为 "两边长" 就是 "两直角边长"。

计算：$(-3)^2 \times 2 + \sqrt{9} \div (-\dfrac{1}{3}) - |-4|$
答案：$5$

🔑 关键思路：按照乘方→乘除→加减的顺序逐步计算，注意符号。

  1. $(-3)^2 = 9$，$9 \times 2 = 18$
  2. $\sqrt{9} = 3$，$3 \div (-\dfrac{1}{3}) = 3 \times (-3) = -9$
  3. $|-4| = 4$
  4. 原式 $= 18 + (-9) - 4 = 5$

✅ 检验：$18-9-4=5$ ✓

⚠️ 常见错误：① $(-3)^2$ 算成 $-9$ ② 除以 $-\dfrac{1}{3}$ 时符号处理错误。

8年级模拟卷1★★☆
先化简，再求值：$(\dfrac{x}{x-1} - \dfrac{1}{x+1}) \div \dfrac{x^2+1}{x^2-1}$，其中 $x=2$
答案：$1$

🔑 关键思路：括号内先通分，除法变乘法，因式分解后约分。本题化简结果恰好与 $x$ 无关（$x\neq\pm 1$）。

  1. 括号内通分：$\dfrac{x(x+1)-(x-1)}{(x-1)(x+1)} = \dfrac{x^2+x-x+1}{(x-1)(x+1)} = \dfrac{x^2+1}{(x-1)(x+1)}$
  2. 除法变乘法：$= \dfrac{x^2+1}{(x-1)(x+1)} \times \dfrac{x^2-1}{x^2+1}$
  3. $x^2-1 = (x+1)(x-1)$，约分后 $= \dfrac{(x+1)(x-1)}{(x-1)(x+1)} = 1$
  4. $x=2$ 时分母均不为零，代入得原式 $=1$

⚠️ 常见错误：① 通分时分子忘乘 ② 除法变乘法忘取倒数 ③ 没发现化简后恒为 $1$。

8年级模拟卷1★★☆
已知：等腰 $\triangle ABC$ 中 $AB=AC$，$D$ 是 $BC$ 中点。求证：$AD \perp BC$。
答案：证明见解析（SSS全等→邻补角相等→直角）

🔑 关键思路：利用等腰三角形"三线合一"——底边上的中线也是底边上的高。用全等三角形证明。

  1. $AB=AC$（已知），$BD=CD$（$D$ 是中点），$AD=AD$（公共边）
  2. $\triangle ABD \cong \triangle ACD$（SSS）
  3. $\angle ADB = \angle ADC$（全等三角形对应角相等）
  4. $\angle ADB + \angle ADC = 180^\circ$（邻补角）
  5. $\angle ADB = \angle ADC = 90^\circ$ → $AD \perp BC$

⚠️ 常见错误：① 漏掉"邻补角和为 $180^\circ$"这一步 ② 直接用"三线合一"定理而不证明（考试中规范解答需要写出推理过程）。

8年级模拟卷1★★★
一次函数 $y=kx+b$ 的图像经过 $(1,3)$ 和 $(2,5)$，与 $x$ 轴交于 $A$，与 $y$ 轴交于 $B$，求 $\triangle AOB$ 的面积。
答案：$\dfrac{1}{4}$

🔑 关键思路：先求函数解析式，再求与坐标轴的交点坐标，最后用面积公式 $S=\dfrac{1}{2}|OA| \cdot |OB|$。

  1. 待定系数法求解析式：$\begin{cases} k+b=3 \\ 2k+b=5 \end{cases}$ → $k=2$, $b=1$ → $y=2x+1$
  2. $y=0$ 时：$2x+1=0$ → $x=-\dfrac{1}{2}$ → $A(-\dfrac{1}{2}, 0)$，$OA = \dfrac{1}{2}$
  3. $x=0$ 时：$y=1$ → $B(0, 1)$，$OB = 1$
  4. $S_{\triangle AOB} = \dfrac{1}{2} \times OA \times OB = \dfrac{1}{2} \times \dfrac{1}{2} \times 1 = \dfrac{1}{4}$

⚠️ 常见错误：① $OA$ 的长度忘记取绝对值 ② 面积公式忘乘 $\dfrac{1}{2}$。

7年级模拟卷1★★☆
给出一组数据：$2, 3, 3, 4, 5, 5, 5, 6$。求：(1) 平均数；(2) 中位数；(3) 众数；(4) 方差。
答案：(1) $\dfrac{33}{8}=4.125$   (2) $4.5$   (3) $5$   (4) $\dfrac{103}{64}$

🔑 关键思路：先排序（数据已排），平均数为总和除以个数，中位数取中间两数均值（偶数个），众数取频次最高值，方差用公式 $s^2=\dfrac{1}{n}\sum(x_i-\bar{x})^2$。

  1. 平均数 $\bar{x} = \dfrac{2+3+3+4+5+5+5+6}{8} = \dfrac{33}{8}$
  2. $8$ 个数据（偶数），中位数 $=$ 第 $4$、$5$ 个数的均值 $= \dfrac{4+5}{2}=4.5$
  3. $5$ 出现 $3$ 次（最多）→ 众数 $=5$
  4. 方差计算：

$(2-\frac{33}{8})^2 = (-\frac{17}{8})^2 = \frac{289}{64}$
$(3-\frac{33}{8})^2 = (-\frac{9}{8})^2 = \frac{81}{64} \times 2 = \frac{162}{64}$
$(4-\frac{33}{8})^2 = (-\frac{1}{8})^2 = \frac{1}{64}$
$(5-\frac{33}{8})^2 = (\frac{7}{8})^2 = \frac{49}{64} \times 3 = \frac{147}{64}$
$(6-\frac{33}{8})^2 = (\frac{15}{8})^2 = \frac{225}{64}$
$\sum = \frac{289+162+1+147+225}{64} = \frac{824}{64} = \frac{103}{8}$
$s^2 = \dfrac{1}{8} \times \dfrac{103}{8} = \dfrac{103}{64}$

⚠️ 常见错误：① 不排序直接取中间数 ② 方差公式分母用 $n-1$（初中用 $n$）。

如图，矩形 $ABCD$ 中，$AB=6$，$BC=8$，点 $P$ 在 $BC$ 边上运动（与 $B$、$C$ 不重合）。
(1) 设 $BP=x$，用 $x$ 的代数式表示 $AP^2$；
(2) $x$ 为何值时 $AP$ 最小？最小值是多少？
答案：(1) $AP^2 = x^2 + 36$   (2) $x=0$ 时 $AP$ 最小，最小值为 $6$

🔑 关键思路：建立坐标系（$B$ 为原点，$BC$ 在 $x$ 轴）→ 写出各点坐标 → 用两点距离公式表达 $AP^2$ → 二次函数求最值。

  1. 建系：$B(0,0)$，$C(8,0)$，$A(0,6)$，则 $P(x, 0)$（$0 < x < 8$）
  2. $AP^2 = (x-0)^2 + (0-6)^2 = x^2 + 36$
  3. (2) $AP^2 = x^2 + 36 \geq 36$（当 $x=0$ 时取等号）
  4. $x=0$ 对应 $P$ 与 $B$ 重合，此时 $AP_{min} = \sqrt{36} = 6$

🔑 几何解释：$A$ 到 $BC$ 的最短距离就是 $A$ 到直线 $BC$ 的垂线段长度，即 $AB=6$。当 $P$ 与 $B$ 重合时恰好达到。

⚠️ 常见错误：① 坐标系建错导致 $AP^2$ 表达式写反 ② 未考虑 $x \geq 0$ 的限制 ③ 求最值时忘记开方。

8年级模拟卷1★★★
【阅读材料】分拆法：$\dfrac{2}{x^2-1} = \dfrac{A}{x-1} + \dfrac{B}{x+1}$，求 $A$、$B$ 的值。
【应用】利用上述结论，解不等式：$\dfrac{1}{x-1} - \dfrac{1}{x+1} \leq 0$。
答案：$A=1$，$B=-1$；不等式解集：$-1 < x < 1$

🔑 关键思路：通分后对比系数求 $A$、$B$，再利用分拆结果解不等式。

  1. $\dfrac{2}{(x-1)(x+1)} = \dfrac{A(x+1)+B(x-1)}{(x-1)(x+1)}$
  2. 分子：$2 = A(x+1)+B(x-1) = (A+B)x + (A-B)$
  3. 对比系数：$\begin{cases} A+B = 0 \\ A-B = 2 \end{cases}$ → $A=1$，$B=-1$
  4. 应用：$\dfrac{1}{x-1} - \dfrac{1}{x+1} = \dfrac{(x+1)-(x-1)}{(x-1)(x+1)} = \dfrac{2}{x^2-1} \leq 0$
  5. 分子 $2>0$，所以 $x^2-1 < 0$ → $(x+1)(x-1) < 0$
  6. 解得 $-1 < x < 1$（注意 $x \neq \pm 1$）

⚠️ 常见错误：① 通分后忘对比系数 ② 解 $(x+1)(x-1) < 0$ 时方向写反 ③ 忘排除 $x=\pm 1$。
