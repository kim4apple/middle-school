第10章 · 酸和碱

九年级下册 | 酸的通性 · 碱的通性 · 中和反应 | 核心：H⁺ 和 OH⁻ 的化学

★★★★☆ 重难点 📊 中考高频 酸碱盐的开篇 · 离子观建立的转折点
🧪 实验：酸碱指示剂的变色
1取三支试管，分别加入稀盐酸（酸性）、蒸馏水（中性）、NaOH 溶液（碱性）各 2 mL。
2向每支试管中滴加 2-3 滴紫色石蕊溶液——酸中变红色，中性不变色（紫色），碱中变蓝色。口诀："酸红碱蓝中为紫"。
3另取三支试管重复步骤 1，滴加 2-3 滴无色酚酞溶液——酸中无色，中性无色，碱中变红色。口诀："酸无碱红中无"。
4注意：酚酞遇碱变红，遇酸不变色（并非所有酸都不变色——"酚酞遇酸不变色"是结论，不是性质判断）。

📋 结论：石蕊和酚酞是酸碱指示剂，能根据溶液酸碱性的不同呈现不同颜色。

🧪 实验：浓硫酸的稀释——"酸入水"
1在烧杯中加入一定量的水（先在烧杯中）。
2用玻璃棒引流，将浓硫酸沿烧杯内壁缓慢注入水中，并用玻璃棒不断搅拌。
3用手触摸烧杯外壁——发烫（浓硫酸溶于水放大量热）。
4严禁将水倒入浓硫酸！水的密度比浓硫酸小，倒入后会浮在硫酸表面，剧烈放热会导致液体飞溅，非常危险。

🚨 安全铁律：酸入水、沿内壁、慢倒入、不断搅。绝对禁止"水入酸"！

🧠 中和反应的本质 · H⁺ + OH⁻ → H₂O

🖱️ 旋转查看。酸溶液中的 H⁺ 和碱溶液中的 OH⁻ 结合生成水分子。

H⁺ + OH⁻ → H₂O
🖱️ 旋转/缩放 · 切换暗色请刷新

中和反应的实质：H⁺ + OH⁻ → H₂O。
酸（如 HCl）在水中电离出 H⁺ 和 Cl⁻；碱（如 NaOH）在水中电离出 Na⁺ 和 OH⁻。H⁺ 和 OH⁻ 结合成水分子，Na⁺ 和 Cl⁻ 留在溶液中——蒸干后得到 NaCl 晶体。
关键理解：酸的共同性质来自 H⁺，碱的共同性质来自 OH⁻。

📝 本节必背方程式（4个）
1$HCl + NaOH \rightarrow NaCl + H_2O$ 中和反应·核心
2$H_2SO_4 + 2NaOH \rightarrow Na_2SO_4 + 2H_2O$
3$Ca(OH)_2 + CO_2 \rightarrow CaCO_3\downarrow + H_2O$ 检验CO₂
4$2HCl + Ca(OH)_2 \rightarrow CaCl_2 + 2H_2O$ 中和酸性土壤
🔬 本节实验中考考点
  • 指示剂变色：石蕊酸红碱蓝中紫；酚酞酸无碱红中无——酚酞用来检验碱性（不能区分酸和中性）
  • 浓硫酸稀释：酸入水！沿内壁！不断搅！（水入酸会沸腾飞溅）
  • 中和反应实验：NaOH+酚酞（红色）→逐滴加HCl→红色变无色→证明中和反应发生了（无明显现象的反应需要借助指示剂）
  • pH测定：用玻璃棒蘸取待测液→滴在pH试纸上→半分钟内与标准比色卡对比（不能将试纸直接浸入待测液）
🧩 易混对比：酸的通性 vs 碱的通性
🔹 酸（H⁺）使石蕊变红 · 与活泼金属→H₂ · 与金属氧化物→盐+水 · 与碱中和 · 与盐反应
🔸 碱（OH⁻）使石蕊变蓝 · 与某些非金属氧化物→盐+水 · 与酸中和 · 与盐反应
📋 中考真题示例
（2023 · 南京）下列物质中，能由金属与稀盐酸直接反应生成的是______
A. FeCl₃   B. CuCl₂   C. AlCl₃   D. AgCl
答案：C Fe+HCl→FeCl₂（不是FeCl₃！）；Cu和Ag在H后，不与稀盐酸反应

📖 核心概念

常见酸

盐酸（HCl）：无色、有刺激性气味、易挥发（敞口冒白雾——挥发出的 HCl 与水蒸气结合成盐酸小液滴）。胃酸主要成分。硫酸（H₂SO₄）：无色黏稠液体、不挥发、有强腐蚀性和吸水性（浓硫酸作干燥剂）。硝酸（HNO₃）：有强腐蚀性和强氧化性。

常见碱

氢氧化钠（NaOH）：白色固体、俗称烧碱/火碱/苛性钠。在空气中易吸收水分而潮解（可作干燥剂），也吸收 CO₂ 变质（$2NaOH + CO_2 \rightarrow Na_2CO_3 + H_2O$）→ 必须密封保存。氢氧化钙 Ca(OH)₂：俗称熟石灰/消石灰，微溶于水，其水溶液叫石灰水。

pH——酸碱度的定量表示

pH 范围通常为 0~14。pH < 7：酸性（pH 越小酸性越强）；pH = 7：中性；pH ＞ 7：碱性（pH 越大碱性越强）。pH 试纸使用方法：用干燥洁净的玻璃棒蘸取待测液滴到 pH 试纸上，与标准比色卡比较读数。不能将试纸直接伸入溶液！

中和反应

定义：酸与碱作用生成盐和水的反应。通式：酸 + 碱 → 盐 + 水。属于复分解反应。应用：① 改良酸性土壤——用熟石灰（Ca(OH)₂）；② 处理工业废水——酸性废水加碱中和，碱性废水加酸中和；③ 医药——含 Al(OH)₃ 或 NaHCO₃ 的药物中和过多胃酸。

酸溶液 ≠ 酸性溶液

酸溶液：溶质是"酸"的溶液（如盐酸、稀硫酸），一定显酸性。酸性溶液：溶质不一定是酸，但溶液显酸性。如 NaHSO₄ 溶液显酸性，但 NaHSO₄ 是盐而不是酸。同样，碱溶液 ≠ 碱性溶液（如 Na₂CO₃ 溶液显碱性，但 Na₂CO₃ 是盐而非碱）。

🧪 中和滴定操作要点

① 仪器：酸式滴定管（装酸）、碱式滴定管（装碱）、锥形瓶、铁架台。

② 操作：从滴定管向锥形瓶中逐滴加入酸（或碱），边滴边摇匀。

③ 终点判断：以酚酞为例——溶液由红色刚好变为无色（且半分钟内不变回红色），即为滴定终点。

④ 重复：至少做2次平行实验，取平均值减少误差。

#化学方程式反应类型考点
1$HCl + NaOH \rightarrow NaCl + H_2O$中和反应
（复分解）		最经典中和反应；无明显现象（需加指示剂判断终点）
2$H_2SO_4 + 2NaOH \rightarrow Na_2SO_4 + 2H_2O$中和反应注意配平：2NaOH（因为 H₂SO₄ 提供 2 个 H⁺）
3$CO_2 + 2NaOH \rightarrow Na_2CO_3 + H_2O$非中和反应！NaOH 吸收 CO₂ 变质，常用 NaOH 溶液吸收/检验 CO₂

5$Zn + H_2SO_4 \rightarrow ZnSO_4 + H_2\uparrow$置换反应实验室制 H₂；酸与活泼金属反应

🔢 计算专题：中和反应中溶质质量分数的计算

核心思路：利用中和反应方程式进行化学计量计算，结合溶质质量分数公式。

【例题】用 20 g 20% 的 NaOH 溶液与一定量稀盐酸恰好完全反应，求：(1) 参加反应的 HCl 的质量；(2) 若稀盐酸的溶质质量分数为 10%，求所用稀盐酸的质量。

解：NaOH 质量 = $20 \times 20\% = 4\ g$
$HCl + NaOH \rightarrow NaCl + H_2O$
36.5 → 40
$x \rightarrow 4$
$x = \dfrac{4 \times 36.5}{40} = 3.65\ g$（HCl 质量）
稀盐酸质量 = $\dfrac{3.65}{10\%} = 36.5\ g$
答：参加反应的 HCl 为 3.65 g，需 10% 稀盐酸 36.5 g。

🚨 安全与常见错误

🔴 实验安全

浓硫酸稀释：酸入水、沿内壁、慢倒入、不断搅。严禁水入酸！
浓硫酸腐蚀性：若不慎溅到皮肤上——先用干布拭去，再用大量水冲洗，最后涂 3%~5% NaHCO₃ 溶液。
NaOH 固体保存：必须密封——因为它既潮解（吸收空气中水分），又吸收 CO₂ 变质。

⚠️ 最高频易错

❌ pH ＜ 7 是酸性，pH ＞ 7 是碱性——不是反比关系！不能说"pH 越小酸性越弱"。
❌ 酸溶液（溶质是酸）一定显酸性，但酸性溶液不一定是酸溶液（如 NaHSO₄ 是盐）。
❌ 中和反应一定生成盐和水，但生成盐和水的反应不一定是中和反应（如 $CO_2 + 2NaOH \rightarrow Na_2CO_3 + H_2O$ 不是中和反应——CO₂ 不是酸，只是酸性氧化物）。
❌ pH 试纸不能用水润湿——会稀释待测液，导致 pH 测量不准。

🔬 实验探究设计：用 pH 试纸测定生活中常见物质的酸碱性

设计思路：收集生活中常见液体，用 pH 试纸逐一测定，与标准比色卡对比，记录 pH 值和酸碱性。

物质预期 pH 范围酸碱性原因
食醋（白醋）2~3酸性含醋酸 CH₃COOH
肥皂水9~10碱性含碱性物质
自来水约 6.5~7.5近中性经过处理的自来水
柠檬汁2~3酸性含柠檬酸
苏打水8~9弱碱性含 NaHCO₃

操作要点：pH 试纸不可润湿，不可伸入待测液中；用干净玻璃棒蘸取，与比色卡在自然光下对比。

🔬 实验探究：氢氧化钠是否变质？是否完全变质？

变质原理：NaOH 与空气中 CO₂ 反应：2NaOH + CO₂ → Na₂CO₃ + H₂O。检验 NaOH 是否变质 = 检验是否存在 Na₂CO₃。

方案一（检验是否变质）：取少量样品溶于水，滴加足量稀盐酸——有气泡产生（CO₂），说明已变质（Na₂CO₃ 存在）。

方案二（检验是否完全变质）：① 取少量样品溶于水，加入过量 CaCl₂ 溶液（除去所有 Na₂CO₃），过滤；② 向滤液中滴加无色酚酞——若变红，说明仍有 NaOH（部分变质）；若不变红，说明 NaOH 已完全变质。

关键理解：不能直接用酚酞检验！因为 Na₂CO₃ 溶液也呈碱性（会使酚酞变红）。必须先用 CaCl₂/BaCl₂ 除去 CO₃²⁻，再检验 OH⁻。

🥉 基础关★☆☆
请分别说出盐酸（HCl）和硫酸（H₂SO₄）的物理性质和化学性质。
📝 查看答案
解析：盐酸（HCl）：物理性质——无色、有刺激性气味液体，易挥发（敞口冒白雾，挥发出的 HCl 与水蒸气结合成盐酸小液滴）。化学性质——① 使石蕊变红；② 与活泼金属反应生成 H₂；③ 与金属氧化物反应（如 Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O）；④ 与碱中和；⑤ 与盐反应（如 AgNO₃ + HCl → AgCl↓ + HNO₃）。
硫酸（H₂SO₄）：物理性质——无色黏稠油状液体，不挥发，有吸水性（可作干燥剂）。化学性质——① 使石蕊变红；② 与活泼金属反应；③ 与金属氧化物反应（除锈）；④ 与碱中和；⑤ 与盐反应（如 BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl）。
🏆 得分点：分别答出盐酸的挥发性和硫酸的吸水性，以及各自的化学通性。
🥉 基础关★☆☆
请分别说出 NaOH 和 Ca(OH)₂ 的物理性质和化学性质。
📝 查看答案
解析：NaOH（氢氧化钠/烧碱/火碱/苛性钠）：物理性质——白色固体，易溶于水（放热），在空气中易潮解（吸收水分）。化学性质——① 使石蕊变蓝、酚酞变红；② 与 CO₂ 反应变质（2NaOH + CO₂ → Na₂CO₃ + H₂O）；③ 与酸中和；④ 与盐反应（如 CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄）。
Ca(OH)₂（熟石灰/消石灰）：物理性质——白色粉末，微溶于水，水溶液叫石灰水。化学性质——① 使石蕊变蓝；② 与 CO₂ 反应（CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O——检验 CO₂）；③ 与酸中和（改良酸性土壤）；④ 与盐反应。
🏆 得分点：答出 NaOH 的潮解性和 Ca(OH)₂ 的微溶性，以及各自的化学性质。
🥉 基础关★☆☆
pH 试纸的正确使用方法是什么？使用时有哪些注意事项？
📝 查看答案
解析：pH 试纸使用方法：① 用干燥洁净的玻璃棒蘸取待测液；② 将待测液滴在pH 试纸上；③ 半分钟内与标准比色卡对比，读出 pH 值。注意事项：① 不能将 pH 试纸直接伸入待测液中（会污染试剂）；② 不能用水润湿 pH 试纸（润湿会稀释待测液，导致酸性溶液 pH 偏大、碱性溶液 pH 偏小）；③ 读数要迅速（时间过长试纸颜色会变化）。

🥈 提升关★★☆
中和反应的实质是什么？请用离子方程式表示。
📝 查看答案
解析：中和反应的实质是酸溶液中的 H⁺ 与碱溶液中的 OH⁻ 结合生成水分子。离子方程式：H⁺ + OH⁻ → H₂O。例如 HCl + NaOH → NaCl + H₂O 的实质是：H⁺（来自 HCl 的电离）+ OH⁻（来自 NaOH 的电离）→ H₂O，而 Na⁺ 和 Cl⁻ 没有参与反应，只是"旁观离子"。
🏆 得分点：写出 H⁺ + OH⁻ → H₂O，并说明中和反应的实质是 H⁺ 和 OH⁻ 结合生成水。

石蕊和酚酞在酸、碱、中性溶液中分别呈现什么颜色？请用口诀记忆。
📝 查看答案
解析：紫色石蕊溶液：酸红、碱蓝、中为紫（酸性→红色，中性→紫色，碱性→蓝色）。无色酚酞溶液：酸无、碱红、中无（酸性→无色，中性→无色，碱性→红色）。注意：酚酞遇酸不变色，但"不变色"不等于"是中性"——酚酞在酸性和中性溶液中都是无色，所以酚酞只能检验碱性，不能区分酸性和中性。
🏆 得分点：正确说出石蕊和酚酞的变色规律，并理解酚酞的局限性。

用 pH 变化曲线描述向 NaOH 溶液中逐滴加入稀盐酸的过程。pH 如何变化？滴定终点如何判断？
📝 查看答案
解析：向 NaOH 溶液（碱性，pH ＞ 7）中逐滴加入稀盐酸（酸性，pH ＜ 7）：① 开始时溶液呈碱性，pH ＞ 7（如 pH ≈ 13）；② 随着盐酸的加入，H⁺ 中和 OH⁻，pH 逐渐下降；③ 当恰好完全中和时，溶液呈中性，pH = 7（滴定终点）；④ 继续加盐酸，溶液呈酸性，pH ＜ 7。pH 变化曲线在终点附近有突跃（从 pH ≈ 10 突降到 pH ≈ 4）。
滴定终点判断：在 NaOH 溶液中先滴加酚酞（溶液变红），逐滴加入盐酸至溶液刚好变为无色（且半分钟内不变回红色）→ 达到滴定终点。
🏆 得分点：能描述 pH 从大到小经过 7 的变化过程，并说明用酚酞指示终点的方法。

现有两瓶无色溶液，已知分别是稀盐酸和稀硫酸。请设计实验鉴别它们，写出方法、现象和化学方程式。
📝 查看答案
解析：鉴别方法——钡盐法：分别取少量两种溶液于试管中，各滴加几滴BaCl₂ 溶液（或 Ba(NO₃)₂ 溶液）。产生白色沉淀的是稀硫酸（生成 BaSO₄ 白色沉淀，不溶于稀硝酸），无现象的是稀盐酸。化学方程式：BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl。注意：不能用 AgNO₃ 鉴别——AgCl 和 Ag₂SO₄ 都是白色沉淀，无法区分。
🏆 得分点：用 BaCl₂ 或 Ba(NO₃)₂ 鉴别，写出方程式，说明 BaSO₄ 不溶于酸。

向一定量的 NaOH 溶液中逐滴加入稀盐酸至过量。请分析反应后溶液中溶质的成分（考虑过量问题）。
📝 查看答案
解析：中和反应：NaOH + HCl → NaCl + H₂O。根据加入盐酸的量不同，反应后溶质成分有三种情况：① 盐酸不足（NaOH 过量）——溶质为 NaCl 和 NaOH（溶液呈碱性，酚酞变红）；② 恰好完全反应——溶质只有 NaCl（溶液呈中性，pH = 7）；③ 盐酸过量——溶质为 NaCl 和 HCl（溶液呈酸性，石蕊变红）。
检验方法：取反应后溶液，加入紫色石蕊溶液——变蓝说明 NaOH 过量，变紫说明恰好中和，变红说明 HCl 过量。也可用 pH 试纸测定。
🏆 得分点：能分三种情况讨论溶质成分，并设计检验方案。
