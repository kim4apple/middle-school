第6章 · 碳和碳的氧化物

九年级上册 | 同素异形体 · CO₂制取 | 金刚石→石墨→C₆₀→CO₂→CO全链路

🧪 实验一：CO₂的实验室制取——最经典的初中化学制备实验
1反应原理：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑。药品：石灰石（或大理石）和稀盐酸。
⚠️ 不用稀硫酸——CaSO₄微溶于水，包裹在石灰石表面阻止反应。
⚠️ 不用浓盐酸——浓盐酸挥发HCl气体，使收集的CO₂不纯。
⚠️ 不用Na₂CO₃——反应速率太快，不易控制。
2发生装置：固液不加热型——与H₂O₂制O₂相同（长颈漏斗/分液漏斗 + 锥形瓶 + 双孔橡皮塞）。
3收集方法：只能向上排空气法（CO₂密度比空气大）。不用排水法——CO₂能溶于水并与水反应（CO₂+H₂O→H₂CO₃）。
4检验方法：将生成的气体通入澄清石灰水——石灰水变浑浊（CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O）。
验满方法：将燃着的木条放在集气瓶口——木条熄灭，则已满（CO₂不能燃烧也不支持燃烧）。

📋 结论：用澄清石灰水检验CO₂是最经典、最可靠的化学鉴定方法——白色沉淀CaCO₃是确证CO₂的唯一特征现象。

💎 金刚石：每个碳原子与周围 4 个碳原子形成共价键，正四面体空间网状结构——自然界最硬的物质（用于切割玻璃、钻头）。

📖 核心概念

碳单质：金刚石/石墨/C₆₀

三种同素异形体——由同种元素组成的不同单质。金刚石（正四面体网状）→硬度最大；石墨（层状）→质软导电；C₆₀（球状分子）→分子晶体。木炭/活性炭是无定形碳（石墨微晶和杂质），活性炭因疏松多孔而有吸附性。

CO₂的物理性质

通常状况下为无色无味气体，密度比空气大（可用向上排空气法收集），能溶于水（不能用排水法收集）。固体CO₂——干冰，在常压下直接升华（从固体→气体），不经过液态，升华时吸收大量热，用于人工降雨和冷藏。

CO₂的化学性质

①不可燃也不助燃（用作灭火剂——但不等于不能支持任何燃烧，如Mg可在CO₂中燃烧：2Mg+CO₂→2MgO+C）。②与水反应：CO₂+H₂O→H₂CO₃（碳酸使紫色石蕊溶液变红）。③与碱反应：CO₂+Ca(OH)₂→CaCO₃↓+H₂O（检验CO₂）。④CO₂可使澄清石灰水变浑浊——这是检验CO₂的特征反应。

CO（一氧化碳）的毒性

CO具有毒性的化学原理：CO与血液中血红蛋白的结合能力比O₂大200~300倍！CO与血红蛋白结合后形成碳氧血红蛋白，使血红蛋白丧失了输送氧气的能力→人体组织缺氧→窒息死亡。煤气中毒就是CO中毒。冬季使用煤炉取暖必须注意通风。

CO的化学性质

①可燃性：2CO+O₂→2CO₂（蓝色火焰，燃烧产物只有CO₂）。②还原性：CO+CuO→Cu+CO₂（还原氧化铜，黑色粉末变红色）。③CO有毒——实验中使用CO必须在通风橱中进行，尾气处理（点燃或收集）防止污染。

温室效应

大气中CO₂含量升高导致全球变暖。CO₂能允许太阳短波辐射透过，但阻止地球表面向宇宙空间辐射长波热量（类似温室大棚的玻璃）。甲烷CH₄、氟氯烃、臭氧O₃等也是温室气体。缓解措施：减少化石燃料使用、植树造林、开发清洁能源。

⚗️ 化学方程式

① CO₂制取（本章核心）：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

② CO₂检验（特征反应）：CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O（石灰水变浑浊）

③ 碳充分燃烧：C + O₂ → CO₂（化合反应 + 氧化反应）

④ 碳不充分燃烧：2C + O₂ → 2CO（是CO中毒的重要原因）

⑤ CO燃烧：2CO + O₂ → 2CO₂（蓝色火焰，检验CO产物的方法——通入澄清石灰水）

⑥ CO还原CuO：CO + CuO → Cu + CO₂（CO的还原性，尾气须处理）

⑦ CO₂与水反应：CO₂ + H₂O → H₂CO₃（化合反应，碳酸不稳定：H₂CO₃ → H₂O + CO₂↑）

🔢 含杂质石灰石制取CO₂的计算

核心公式：纯净物质量 = 不纯物质量 × 纯度（质量分数）

例题1：某石灰石样品含80% CaCO₃。取12.5g该样品与足量稀盐酸反应，能生成多少克CO₂？

解：m(CaCO₃纯) = 12.5g × 80% = 10g

设生成CO₂的质量为x。

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

100                                44

10g                               x

100/44 = 10g/x → x = 10g × 44 / 100 = 4.4g

答：能生成4.4g CO₂。

例题2：含CaCO₃ 90%的石灰石500t，高温煅烧（CaCO₃ → CaO + CO₂↑），可制得生石灰（CaO）多少吨？

解：m(CaCO₃纯) = 500t × 90% = 450t

设生成CaO的质量为x。

CaCO₃ → CaO + CO₂↑

100          56

450t         x

100/56 = 450t/x → x = 450t × 56 / 100 = 252t

答：可制得约252t生石灰。

🚨 常见错误辨析

🔴 CO₂灭火的认知陷阱

❌ "CO₂不能支持任何物质的燃烧"——错！镁条可以在CO₂中燃烧：2Mg + CO₂ → 2MgO + C（有黑色颗粒产生）。所以镁引起的火灾不能用CO₂灭火器！
❌ "CO₂能灭火是因为它不能燃烧也不支持燃烧"——这只是原因之一。另一个原因是CO₂密度比空气大，能覆盖在可燃物表面隔绝空气（O₂）。

⚠️ CO的"杀人机制"

CO中毒不是腐蚀性中毒，而是化学性窒息——CO与血红蛋白结合力远超O₂，使血红蛋白失去运氧能力。症状：头痛→恶心→昏迷→死亡。急救方法：迅速移至通风处，吸入新鲜空气或纯氧。冬季煤炉取暖必须保持通风——煤气中毒往往在睡梦中发生，特别危险。

🎯 中考真题精选（碳和碳的氧化物 · 近年典型）
九上★☆☆
金刚石和石墨物理性质差异很大的原因是（ ）
A. 原子排列方式不同   B. 碳原子的结构不同
C. 组成元素不同   D. 碳原子的质量不同
答案：A
🔑 关键思路：金刚石和石墨都是碳单质（同素异形体），碳原子本身完全相同。性质差异的唯一原因是碳原子排列方式不同——金刚石正四面体网状 vs 石墨层状。"结构决定性质"是此类题的通用答案。

九上★★☆
实验室制取CO₂，下列做法错误的是（ ）
A. 用大理石和稀盐酸作反应物   B. 用排水法收集CO₂
C. 用向上排空气法收集CO₂   D. 将燃着的木条放在集气瓶口验满
答案：B
🔑 关键思路：CO₂能溶于水并与水反应（CO₂+H₂O→H₂CO₃），所以不能用排水法收集。密度比空气大→用向上排空气法。检验用澄清石灰水，验满用燃着的木条放瓶口。
💡 显示/隐藏解析
九上★★☆
除去CO中混有的少量CO₂，可以采用的方法是（ ）
A. 点燃   B. 通过澄清石灰水   C. 通过水   D. 通过灼热的CuO
答案：B
🔑 关键思路：除杂原则：不引入新杂质、不与保留物反应。CO₂+Ca(OH)₂→CaCO₃↓+H₂O——CO₂被吸收，CO不与Ca(OH)₂反应，通过澄清石灰水后只剩CO。A点燃会同时消耗CO，C水只能吸收少量CO₂，D灼热CuO会把CO也反应掉（CO+CuO→Cu+CO₂）。
💡 显示/隐藏解析
九上★★★
某气体通过澄清石灰水时石灰水变浑浊，该气体可能是（ ）
A. 一定是CO₂   B. 一定含CO₂   C. 一定不含CO₂   D. 无法判断
答案：B
🔑 关键思路：使澄清石灰水变浑浊是CO₂的特征反应——只要石灰水变浑浊，一定说明有CO₂存在。但该气体"可能"是纯CO₂，也可能是含CO₂的混合气体（如CO₂和CO的混合物）。所以答案是"一定含CO₂"——不能确定是不是纯CO₂。
💡 显示/隐藏解析

🔬 实验探究设计：证明鸡蛋壳/水垢的主要成分是CaCO₃

问题：厨房里的鸡蛋壳、水壶里的水垢，它们的主要成分是什么？如何用化学方法证明？

① 方案A：加酸产生气体——取少量鸡蛋壳碎片/水垢放入试管，滴加稀盐酸——观察到产生大量气泡（CO₂）。将生成的气体通入澄清石灰水——石灰水变浑浊。结论：鸡蛋壳/水垢中含有碳酸盐（CO₃²⁻），主要成分是CaCO₃。

② 方案B：高温煅烧——将鸡蛋壳/水垢放在火上充分灼烧（模拟 CaCO₃ → CaO + CO₂↑），冷却后加水——固体发热、变成粉末（CaO + H₂O → Ca(OH)₂）。取上层清液滴入酚酞——溶液变红（Ca(OH)₂呈碱性）。

📋 结论：加酸产生能使石灰水变浑浊的气体，是检验碳酸盐（含CO₃²⁻）的通用方法。化学方程式：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑。
