# 设计模板审核意见

> 审核人：国家特级教师
> 定位：零基础 → 优等生
> 审核对象：`docs/design-template.md`

---

## 一、总体评价

模板在**结构完整度**和**技术实现**上达到了优秀水平，CSS 设计系统、模块模板、SVG 规范、JS 功能清单都很扎实。但要达到"从零基础到优等生"的育人目标，在**认知科学**、**学习策略**、**训练体系**三个维度还有明显缺口。

以下是逐项审核意见，按"缺什么 → 为什么缺 → 怎么补"组织。

---

## 二、严重缺失（必须补）

### 1. 缺少"学习路径水位线"设计

**现状**：有"四层递进知识树"和"40天计划"，但只是章节顺序和天数的罗列。

**问题**：零基础学生不知道"我该学到什么程度才算够"。同一章的内容，初学者和冲刺优等生的要求完全不同。

**建议增加模块**：

```
┌─ 🚩 本章学习水位线
│   青铜（基础）→ 白银（掌握）→ 黄金（精通）
│   青铜：能背公式、做对基础题（60%）
│   白银：能解释原理、做对变式题（80%）
│   黄金：能讲给别人听、做对中考压轴题（95%+）
```

每章的概念卡、公式卡、技巧卡旁标注水位标记 🥉🥈🥇，让学生清楚自己当前在哪一层。

### 2. 缺少"知识图谱 + 先行组织者"

**现状**：每章开头是"学习目标"（3条知识点罗列）和"前置知识链接"（一句话提及）。

**问题**：零基础学生看到"掌握一元一次方程的解法"这种目标，大脑是关闭的——他不知道这个方程长什么样、为什么要学、学了有什么用。

**建议**：每章开头增加一个 **先行组织者框**，用大白话+生活场景告诉学生"本章要解决什么问题"：

```html
<div class="advance-organizer">
  <strong>🔍 生活中的问题：</strong>
  你去商店买3支笔和2个本子花了26元，同学买同样的1支笔和3个本子花了18元。
  怎么算出笔和本子各多少钱？——这就是本章要解决的问题。
</div>
```

### 3. 缺少"错误模型库"

**现状**：附录有"高频易错陷阱总览"，但只是静态列表。

**问题**：优等生和普通学生的最大区别不是做对多少题，而是**知道自己错在哪里**。静态列表学生看完就忘。

**建议**：每张技巧卡增加一个 **常见错误诊断** 子模块，用"错误× → 诊断 → 正确✓"三段式：

```
┌─ ❌ 常见错误诊断
│  ❌ 错误写法：$x^2 + y^2 = (x+y)^2$（初学最易犯的错误）
│  🔍 错因分析：把平方和当成了和的平方，忘记了 $(x+y)^2 = x^2 + 2xy + y^2$
│  ✅ 记忆锚：平方和 = 无交叉项，和的平方 = 有交叉项 $2xy$
```

把学生在每个知识点上**最可能犯的3个错误**提前写出来，胜过后来自我纠错10遍。

### 4. 缺少"间隔重复"的结构化设计

**现状**：附录A4中提到"艾宾浩斯间隔复习"，但没有任何实际结构支撑。

**问题**：只说"要复习"等于没说。学生需要明确的**什么时候复习什么**的指令。

**建议**：在每章末尾的自查清单后，增加一个 **复习时间表**：

```html
<div class="review-schedule">
  <strong>📅 复习计划</strong>
  <span class="review-dot" data-day="1">1天后</span>
  <span class="review-dot" data-day="3">3天后</span>
  <span class="review-dot" data-day="7">1周后</span>
  <span class="review-dot" data-day="30">1月后</span>
</div>
```

对应复习内容：1天后做课后习题、3天后重做错题、7天后讲给别人听、30天后做真题。

## 三、重要优化（建议补）

### 5. 缺少"变式训练梯度"

**现状**：`technique-card` 有 `.variant` 模块，但只有1个变式。

**问题**：优等生和普通生的分水岭是**变式能力**——换个问法就卡住 = 没真懂。

**建议**：每张技巧卡至少设计 3 个变式，按难度梯度排列：

```
┌─ 变式训练
│  🌱 基础变式：改数字（计算 $2x+3=7$ → $3x-5=10$）
│  🌿 综合变式：改形式（整数系数 → 分数系数 $\\frac{x}{2} + 1 = 3$）
│  🌳 拓展变式：改情境（从解方程到应用题）
```

每道变式学生完成后可对照解析自查。

### 6. 缺少"出声思考"训练模块

**现状**：费曼挑战是"讲给别人听"，但没有结构化的引导。

**问题**：零基础学生面对"讲一遍"的指令不知道从哪说起，也缺乏自我纠错的能力。

**建议**：每章费曼挑战改为 **结构化的出声思考模板**：

```html
<div class="feynman-template">
  <strong>🗣 费曼挑战（结构化版）</strong>
  <ol>
    <li>用一句话说清本章核心是什么</li>
    <li>举一个生活例子</li>
    <li>写出你最怕的一道题，边做边说出每一步的思考</li>
    <li>如果卡住了，回头看哪一节？</li>
  </ol>
</div>
```

### 7. 缺少"认知负荷管理"

**现状**：每章有10个模块（学习目标→清单），全部展开时信息量大。

**问题**：零基础学生一次性面对10个模块会产生认知过载。研究表明，工作记忆容量约4±1个组块。

**建议**：增加一个 **学习模式按钮**：

```
┌─ 🎯 选择学习模式
│  📖 浏览模式（默认收起所有details，只看标题）
│  📝 学习模式（只展开「概念+公式+技巧」）
│  🏃 冲刺模式（只展开「技巧+真题+清单」）
│  🖨️ 打印模式（展开全部）
```

每个模式控制哪些 `<details>` 默认 `open`。

### 8. 缺少"跨章连接"的可视化

**现状**：`.knowledge-link` 用一句话提及前置知识。

**问题**：学生学完一章时看不到它和下一章的关系，学完就忘。

**建议**：在每章末尾增加一个 **下一站预告**：

```html
<div class="next-stop">
  <strong>➡️ 下一章预告</strong>
  <p>本章学会了 $v=s/t$ ——描述运动快慢。下一章我们将研究声音这个具体的运动——声音怎么产生、怎么传播、为什么有高有低。</p>
</div>
```

同时在知识导图中用箭头标注章节间依赖关系。

## 四、细节完善

### 9. 自查清单 → 自查评分

**现状**：checkbox 只有 ✓/空白。

**建议**改为三级评分：

```html
<label>
  <select class="self-score">
    <option value="0">🔴 完全不会</option>
    <option value="1">🟡 有点印象</option>
    <option value="2">🟢 熟练掌握</option>
  </select>
  能利用三边关系判断能否构成三角形
</label>
```

红色 = 需要回看，黄色 = 需要练习，绿色 = 过关。

### 10. 中考真题 → 增加难度标签

**现状**：`<span class="q-tag">2023北京</span>` 只有年份城市。

**建议**增加难度和题型：

```html
<span class="q-tag">2023北京</span>
<span class="q-diff easy">★☆☆ 基础</span>
<span class="q-type">选择</span>
```

难度分级：★☆☆ 基础、★★☆ 中等、★★★ 压轴

### 11. 耗时估计

**现状**：没有建议的每个模块学习耗时。

**建议**：每个模块标题旁标注建议耗时：

```
📖 核心概念（15分钟）
💡 解题技巧（20分钟）
🎯 中考真题（25分钟）
```

让学生能合理规划每次学习的时间预算。

### 12. 进度追踪可视化

**现状**：没有全书的进度指示。

**建议**：在侧边栏每章链接旁加进度条：

```
第3章 · 一元一次方程  ████████░░ 80%
第4章 · 几何图形初步  ██░░░░░░░░ 20%
```

进度数据存在 `localStorage`，每次 checklist 打勾自动更新。

## 五、审核结论

### 现有模板评分（10分制）

| 维度 | 评分 | 说明 |
|------|------|------|
| 技术实现 | 9/10 | CSS设计系统完善，SVG/KaTeX/JS功能全面 |
| 知识覆盖 | 8/10 | 模块结构完整，但缺先行组织者 |
| 教学法科学 | 5/10 | 缺少间隔重复、变式梯度、认知负荷管理 |
| 自学友好度 | 6/10 | 缺少水位线、学习模式、进度追踪 |
| 零基础→优等生 | 4/10 | 没有分层的区分设计，所有学生用同一套内容 |

### 最优先补的3项

1. **学习水位线**（🥉🥈🥇） — 让不同水平的学生各取所需
2. **错误诊断模型**（❌→🔍→✅） — 预防比纠错更高效
3. **间隔重复时间表** — 解决"学完就忘"的核心痛点

### 建议实施方案

将现有10个模块扩展为13个——在 ④思维导图 后插入 ⑤先行组织者，在 ⑧解题技巧 中增加 错误诊断 和 变式梯度，在 ⑩自查清单 后增加 间隔复习时间表 和 下一站预告。

*审核完毕。模板的骨架非常扎实，补上以上教学法模块后，才能真正做到"零基础进、优等生出"。*
