第1章 · 有理数

🎯 学习目标

理解正负数的意义；掌握有理数的四则运算及乘方；会用科学记数法表示数；理解绝对值的几何意义

🧭 数形结合（一）：数轴是最原始的"坐标系"。在数轴上，数→点，大小关系→左右位置，运算→移动。画出数轴是解决有理数题的"第一武器"。本章到第7章平面直角坐标系，再到八年级一次函数，一脉相承。
🧠 思维导图
有理数
  • 基本概念 正负数 · 数轴（原点、正方向、单位长度）
  • 相反数 $a + (-a) = 0$ · 绝对值 $\left|a\right| = \begin{cases} a & a \geq 0 \\ -a & a < 0 \end{cases}$
  • 分类 有理数 = 整数 ∪ 分数，可化为有限小数或无限循环小数
  • 运算
    • 加减 同号相加取同号，异号相加取绝对值大的符号
    • 乘除 同号得正，异号得负；除以一个数 = 乘以它的倒数
    • 乘方 $a^n$：$a$为底数，$n$为指数；负数的偶次幂为正，奇次幂为负
    • 混合运算 先乘方 → 再乘除 → 后加减，同级从左到右，有括号先算括号
  • 科学记数法 $a \times 10^n$，其中 $1 \leq |a| < 10$，$n$ 为整数
  • 近似数 精确度 · 有效数字 · 四舍五入
📖 核心概念
正数与负数

大于 $0$ 的数为正数（$+$ 可省略），小于 $0$ 的数为负数。$0$ 既不是正数也不是负数，是正负数的分界。

数轴

规定了原点、正方向和单位长度的直线。数轴上右边的数总比左边大。

相反数

只有符号不同的两个数互为相反数。$a$ 的相反数是 $-a$。$0$ 的相反数是 $0$。在数轴上关于原点对称。

绝对值

数轴上表示数 $a$ 的点到原点的距离，记作 $|a|$。$|a| \geq 0$ 恒成立。

倒数

乘积为 $1$ 的两个数互为倒数。$a$ 的倒数是 $\frac{1}{a}\ (a \neq 0)$。$0$ 没有倒数。

科学记数法

把一个数写成 $a \times 10^n$ 的形式，其中 $1 \leq |a| < 10$。方便表示很大的数或很小的数。

近似数与精确度

近似数：与准确数接近的数。精确度：近似数与准确数的接近程度。四舍五入法是最常用的取近似数方法。精确到哪一位，就看它的下一位数字，大于等于5进1，小于5舍去。

📐 公式速查
加法法则

同号：$|a|+|b|$，取相同符号

异号：$||a|-|b||$，取绝对值较大的符号

$a + 0 = a$

减法法则

$$a - b = a + (-b)$$

减去一个数，等于加上它的相反数

乘法法则

同号得正：$(+a) \times (+b) = ab$，$(-a) \times (-b) = ab$

异号得负：$(+a) \times (-b) = -ab$

$a \times 0 = 0$

除法法则

$$a \div b = a \times \frac{1}{b}\ (b \neq 0)$$

除以一个非零数，等于乘以它的倒数

乘方法则

正数的任何次幂为正；负数的奇次幂为负、偶次幂为正

$(-1)^n = \begin{cases} 1 & n \text{为偶数} \\ -1 & n \text{为奇数} \end{cases}$

运算顺序

第一级：乘方（先算） → 第二级：乘除 → 第三级：加减

同级从左到右，有括号先算括号

科学记数法

$$a \times 10^n \quad (1 \leq |a| < 10,\ n\text{为整数})$$

整数位数 = $n+1$；从小数点后第 $k$ 位开始非零：$n=-k$

绝对值

$$|a| = \begin{cases} a & a \geq 0 \\ -a & a < 0 \end{cases}$$

$|a| \geq 0$ 恒成立；$|a| = 0 \iff a = 0$

💡 解题技巧

1 数轴比较大小

利用数轴直观比较：右边的数总大于左边的数。画数轴，标出各点位置。

例题：比较 $-3$，$0$，$2$，$-1.5$ 的大小
解：在数轴上从左到右排列：$-3 < -1.5 < 0 < 2$

⚠️ 易错：负数比较时，绝对值大的反而小，如 $-5 < -3$（不要写成 $-5 > -3$）

2 绝对值化简

先判断符号确定去绝对值的结果，再运算。牢记 $|a| \geq 0$。

例题：化简 $|x-3| + |x+1|$，当 $-1 < x < 3$
解：$x-3<0$，$|x-3|=3-x$；$x+1>0$，$|x+1|=x+1$。原式$=(3-x)+(x+1)=4$

⚠️ 易错：绝对值总是非负的，忘记判断符号直接去掉绝对值符号

3 分配律巧算

遇到 $a(b+c)$ 或带分数除法，先化为乘法再用分配律简化计算。

例题：计算 $(-\frac{3}{4}) \times (8 - 1\frac{1}{3} - 0.04)$
解：$= -\frac{3}{4} \times 8 + \frac{3}{4} \times \frac{4}{3} + \frac{3}{4} \times 0.04 = -6 + 1 + 0.03 = -4.97$

⚠️ 易错：负号分配时要变号，$-(a+b)=-a-b$

4 辨别 $(-a)^n$ 与 $-a^n$

括号决定底数范围：$(-a)^n$ 的底数是 $-a$（含负号），$-a^n$ 的底数是 $a$（不含负号）。先确定底数，再计算符号。

例题：计算 $(-2)^2$ 和 $-2^2$
解：$(-2)^2$ 底数是 $-2$，$(-2)^2 = (-2)\times(-2) = 4$；$-2^2$ 底数是 $2$，$-2^2 = -(2\times2) = -4$。两者差一个负号！

⚠️ 易错：$-2^2$ 不是 $(-2)\times(-2)$！负号在指数运算之后才起作用

5 科学记数法 $a$ 的范围

$a \times 10^n$ 中必须满足 $1 \leq |a| < 10$。整数位数$-$1 = $n$（$n$为整数）。

例题：将 $1200000$ 用科学记数法表示。
解：$1200000 = 1.2 \times 10^6$（不是 $12 \times 10^5$ 或 $0.12 \times 10^7$！）

⚠️ 易错：$120$ 写成 $12 \times 10^1$（$12 \geq 10$，不符合要求）；或写成 $0.12 \times 10^3$（$0.12 < 1$，不符合要求）

第2章 · 整式的加减

🎯 学习目标

识别单项式和多项式的系数与次数；熟练合并同类项与去括号；能进行整式的加减运算并化简求值

🔗 前置知识：有理数运算（第1章）。本章整式的系数运算依赖有理数法则，熟练后可直接迁移。
🧭 整体思想：把 $a^2-ab$ 看成一个"整体"来合并，不要拆开再拼回去。整体代入是化简求值的核心技巧——"能整体就不拆散"，这种思维在方程组消元（第8章）中还会用到。
🧠 思维导图
整式的加减
  • 单项式 数与字母的乘积
    系数：数字因数 · 次数：所有字母指数之和
  • 多项式 几个单项式的和
    项（含常数项） · 次数：最高次项次数
  • 整式 单项式 + 多项式
  • 同类项 所含字母相同 + 相同字母指数相同
  • 合并同类项 系数相加，字母部分不变
  • 去括号 括号前"+"不变号 · 括号前"-"全变号
📖 核心概念
单项式

由数与字母的乘积组成的代数式。单独一个数或字母也是单项式。系数是数字因数，次数是所有字母的指数之和。

多项式

几个单项式的和。每个单项式叫项，不含字母的项叫常数项。次数为最高次项的次数。

整式

单项式和多项式统称整式。注意区分整式与分式（分母含字母为分式）。

同类项

所含字母相同且相同字母的指数也相同的项。常数项与常数项也是同类项。同类项与系数无关。

📐 公式速查
合并同类项法则

系数相加，字母部分（字母及其指数）保持不变

例：$3x^2 + 5x^2 = (3+5)x^2 = 8x^2$

去括号法则

括号前是 $+$：$$+(a + b - c) = a + b - c$$

括号前是 $-$：$$-(a + b - c) = -a - b + c$$

整式加减步骤

① 去括号（注意变号） → ② 找同类项 → ③ 合并同类项

代数式求值：先进化简，再代入求值！

💡 解题技巧

1 先找同类项再合并

做题时先划出同类项（相同字母+相同指数），再合并系数。避免遗漏。

例题：合并 $3x^2y - 2xy^2 + 5x^2y + xy^2$
解：$3x^2y$ 与 $5x^2y$ 同类 → $8x^2y$；$-2xy^2$ 与 $xy^2$ 同类 → $-xy^2$。结果为 $8x^2y - xy^2$

⚠️ 易错：$x^2y$ 与 $xy^2$ 不是同类项！指数位置不同

2 先化简再求值

遇到代数式求值，一定先化简（去括号+合并），再代入数值，这样计算量最小。

例题：求值 $2(a^2-ab)-3(a^2-ab)$，其中 $a=3,b=-2$
解：原式$=(2-3)(a^2-ab) = -(a^2-ab) = -a^2+ab$；代入得 $-9+(-6)=-15$

⚠️ 易错：不化简直接代入，数值复杂容易算错

⚠️ 易错：去括号漏乘：$2(x+3)=2x+3$ 是错的！分配律要乘遍每一项 $2x+6$

🎯 中考真题精选（整式 · 近年典型）
2022北京
化简：$3a^2 - 2a + 4a^2 - 7a$
答案：$7a^2 - 9a$。合并同类项：$3a^2+4a^2=7a^2$，$-2a-7a=-9a$。
2023杭州
先化简，再求值：$(x+2)(x-2) - x(x-1)$，其中 $x = 3$
答案：1。化简：$=x^2-4-x^2+x = x-4$。代入 $x=3$：$3-4=-1$。⚠️ 中考标准步骤：先化简再代入！
2022广州
已知 $x + y = 3$，求整式 $2x + 2y + 1$ 的值。
答案：7。整体代入：$2x+2y+1 = 2(x+y)+1 = 2×3+1 = 7$。能整体就不拆散！
2023重庆
化简：$(2a - b) - (a + 2b)$
答案：$a - 3b$。去括号：$=2a-b-a-2b = a-3b$。括号前是"-"，去掉括号后每一项都要变号。

第3章 · 一元一次方程

🎯 学习目标

掌握等式的基本性质；熟练解一元一次方程；能用方程解决行程、工程、利润等实际问题

🔗 前置知识：整式化简（第2章）· 等式的性质（本章核心）。后续关联：方程组（第8章）、不等式（第9章）的解法前四步与本章完全相同，仅最后一步有差异。
🧭 方程思想：方程是七年级最重要的数学工具。核心思维：设未知数→用未知数表示已知量→找等量关系→列方程→求解。这种"用方程描述现实"的能力，是第3章、第8章乃至整个中学数学的灵魂。

📋 应用题通用解题模板（7步法）

  1. 审 — 读题，划出关键词和数据
  2. 找 — 找到等量关系（谁和谁相等/谁的几倍是谁/谁比谁多多少……）
  3. 设 — 设未知数（一般设"求什么"或设"关键量"）
  4. 列 — 用未知数表示所有相关量，根据等量关系列出方程
  5. 解 — 解方程（移项→合并→系数化1）
  6. 验 — 代入检验（方程是否成立 + 答案是否符合实际）
  7. 答 — 写出完整答句
🧠 思维导图
一元一次方程 $ax + b = 0\ (a \neq 0)$
  • 基本概念 方程 · 方程的解 · 解方程 · 一元一次方程
  • 等式性质 加减：$a=b \implies a\pm c=b\pm c$ · 乘除：$a=b \implies ac=bc\ (c\neq0)$
  • 解法步骤 去分母 → 去括号 → 移项 → 合并同类项 → 系数化为1
  • 应用题
    • 行程问题 $s=vt$（相遇、追及、顺逆流）
    • 工程问题 工作量 = 工作效率 × 时间，常设总量为 1
    • 利润问题 利润 = 售价 − 进价；利润率 = 利润 ÷ 进价 × 100%
📖 核心概念
方程与解

含有未知数的等式叫方程。使方程左右两边相等的未知数的值叫方程的解。求方程解的过程叫解方程。

一元一次方程

只含一个未知数，未知数的次数都是1，等号两边都是整式。标准形式：$ax+b=0\ (a\neq0)$。

等式的性质

性质1：等式两边加（或减）同一个数（或式子），结果仍相等。

性质2：等式两边乘同一个数，或除以同一个不为 $0$ 的数，结果仍相等。

移项

把方程一边的某项变号后移到另一边，即移项要变号。实质是利用等式性质1。

📐 公式速查
等式性质

$a = b \implies a \pm c = b \pm c$

$a = b \implies ac = bc$  ·  $a = b \implies \frac{a}{c} = \frac{b}{c}\ (c \neq 0)$

解方程五步法

① 去分母（找最小公倍数乘各项）

② 去括号（注意变号）

③ 移项（移项要变号！）

④ 合并同类项

⑤ 系数化为 $1$（两边同除以系数）

行程问题

路程 = 速度 × 时间：$s = vt$

相遇：$s_A + s_B = S_{\text{总}}$  ·  追及：$s_A - s_B = S_{\text{距离}}$

顺流：$v = v_{\text{静}} + v_{\text{水}}$  ·  逆流：$v = v_{\text{静}} - v_{\text{水}}$

工程与利润

工作量 = 工作效率 × 时间（设总量为 $1$）

利润 = 售价 − 进价

利润率 = $\frac{\text{利润}}{\text{进价}} \times 100\%$

利率与利息

利息 = 本金 × 利率 × 时间

本息和 = 本金 + 利息 = 本金 × $(1 + \text{利率} \times \text{时间})$

💡 解题技巧

1 移项要变号

解方程的核心纪律：把某一项从等号一边移到另一边，必须改变符号。

例题：解 $3x + 5 = 2x - 1$
解：移项 $3x - 2x = -1 - 5$，得 $x = -6$

⚠️ 易错：$5$ 从左边移到右边变 $-5$，忘记变号是最高频错误

💡 原理：移项的本质是利用等式的性质1——等式两边同时减去（或加上）同一个数，等式仍成立。将 $2x$ 从右边移到左边，等价于两边同时减 $2x$，所以右边剩下 $-1-2x$，左边变成 $3x-2x$。变号不是魔法，是等式性质的结果。

📝 变式：解方程 $5x - 7 = 3x + 1$
解：移项 $5x-3x=1+7$，$2x=8$，$x=4$。
验证：左边 $5×4-7=13$，右边 $3×4+1=13$，✓

📋 中考怎么考：中考基础题（选择/填空），通常作为解方程的中间步骤。1分2分的题，错在移项变号太可惜。

2 去分母找最小公倍数

含分母的方程，先找出所有分母的最小公倍数，再每项都乘这个数。不要漏乘没有分母的项！

例题：解 $\frac{x-1}{2} = \frac{x+2}{3}$
解：两边乘6，$3(x-1) = 2(x+2)$，$3x-3=2x+4$，$x=7$

⚠️ 易错：漏乘整数项或常数项，如 $\frac{x}{2}+1=3$ 乘2时忘记乘 $1$

💡 原理：去分母的本质是利用等式的性质2——等式两边乘以同一个不为零的数，等式仍成立。乘最小公倍数保证去分母后系数为整数且最小，减少计算量。注意：没有分母的项也必须乘！

📝 变式（真题型）：解 $\frac{2x-1}{3} - \frac{x+1}{6} = 1$
解：两边乘6：$2(2x-1)-(x+1)=6$，$4x-2-x-1=6$，$3x-3=6$，$3x=9$，$x=3$

📋 中考怎么考：中考必考解答题第一题（6-8分），去分母漏乘是这里最常扣分的点。

3 行程问题画线段图

相遇/追及问题，先画线段图理清各段路程关系，再列方程。

例题：甲从A出发速度5km/h，乙从B出发速度3km/h，AB相距24km，相向而行几小时相遇？
解：设 $x$ 小时后相遇，$5x + 3x = 24$，$x = 3$

⚠️ 易错：忽略单位统一（如速度是 km/h，时间给的分钟，先换算）

💡 原理：行程问题的核心关系式是 $s=vt$。相遇问题中两人路程之和等于总路程；追及问题中两人路程之差等于初始距离。画线段图是将"文字"转化为"数学关系"的桥梁——各段标上未知数，等量关系一目了然。

📝 变式（追及问题）：甲从A地出发速度6km/h，1小时后乙从A地出发速度10km/h追甲，几小时后追上？
解：设乙出发 $x$ 小时后追上。甲已走 $6×1=6$km，甲路程 $6+6x$，乙路程 $10x$。$10x = 6+6x$，$4x=6$，$x=1.5$

4 配套问题与方案选择

配套问题：找"几比几搭配"→列乘积等式。方案选择：分别算出两种方案的费用/数量表达式，用方程找平衡点。

例题：一张桌子配4把椅子。现有木料可做20张桌子或60把椅子，如何分配使桌椅恰好配套？
解：设做桌子 $x$ 张，则做椅子 $4x$ 把。$\frac{x}{20} + \frac{4x}{60} = 1$，$\frac{3x+4x}{60}=1$，$x=\frac{60}{7}$，取整数讨论。

⚠️ 易错：配套关系不是"数量相等"，而是"按比例对应"——1张桌配4把椅，椅子 = 4×桌子

💡 原理：配套问题的本质是"两个量的比例固定"。找到比例关系后，用一个量表示另一个量，等式：两种产品消耗总量=总资源。方案选择则利用不等式比较两种方案的成本/收益。

📝 变式（方案选择）：某地有两种出租车：起步价8元/3km，超出每千米2元。方案B：每千米2.5元无起步价。问路程多少千米时两种方案费用相同？超过多少千米选A更划算？
解：设 $x$ km。A费用：$x\leq3$ 时8元，$x>3$ 时 $8+2(x-3)=2x+2$。B费用：$2.5x$。当 $x>3$ 时，$2x+2=2.5x$，$x=4$km。$x<4$ 时 B 更划算，$x>4$ 时 A 更划算。

📖 核心概念
直线·射线·线段

直线：没有端点，向两端无限延伸，两点确定一条直线。
射线：一个端点，向一端无限延伸。
线段：两个端点，可度量长度。

线段中点

若点 $M$ 是线段 $AB$ 的中点，则 $AM = MB = \frac{1}{2}AB$。这是七年级最重要的几何关系之一。

角的概念

有公共端点的两条射线组成的图形。公共端点叫角的顶点，两条射线叫角的边。

余角与补角

余角：两角之和为 $90°$。$\angle\alpha + \angle\beta = 90°$。
补角：两角之和为 $180°$。$\angle\alpha + \angle\beta = 180°$。
同角（等角）的余角相等，同角（等角）的补角相等。

三视图

从正面看→主视图，从上面看→俯视图，从左面看→左视图。三视图位置：主视图左上，俯视图在主视图正下方，左视图在主视图正右方。长对正、高平齐、宽相等。

正方体展开图

正方体表面展开图共11种，分四类：1-4-1型（6种）、2-3-1型（3种）、2-2-2型（1种）、3-3型（1种）。口诀：一线不过四，田凹不能有。对立面不会相邻。

角平分线

从一个角的顶点出发，把这个角分成两个相等角的射线。若 $OC$ 是 $\angle AOB$ 的平分线，则 $\angle AOC = \angle COB = \frac{1}{2}\angle AOB$。角平分线是几何中的重要概念，与线段中点对称对应。

📐 公式速查
度分秒换算

$$1° = 60' \quad 1' = 60'' \quad 1° = 3600''$$

度化分：乘60；分化秒：乘60；秒化度：除以3600

线段

中点：$AM = MB = \frac{1}{2}AB$

两点之间线段最短（线段公理）

余角与补角

若 $\angle\alpha + \angle\beta = 90°$，则 $\angle\alpha$ 与 $\angle\beta$ 互余

若 $\angle\alpha + \angle\beta = 180°$，则 $\angle\alpha$ 与 $\angle\beta$ 互补

等角的余角相等，等角的补角相等

角平分线

若 $OC$ 是 $\angle AOB$ 的平分线，则：

$$\angle AOC = \angle COB = \frac{1}{2}\angle AOB$$

💡 解题技巧

1 三视图与展开图判断

三视图："长对正"（主俯长相等）、"高平齐"（主左高相等）、"宽相等"（俯左宽相等）。
展开图：判断能否折成正方体——相邻面最多4个，不能有"田""凹"字形。

例题：下图能否折成正方体？（一个"田"字形排列的4个正方形）

⚠️ 易错："1-4-1"型中，中间4个一排，上下各一个不能都在同一侧

2 度分秒的加减运算

加法：度+度、分+分、秒+秒。满 $60$ 进位（$60''=1'$，$60'=1°$）。
减法：不够减时向前一位借 $1$ 当 $60$。

例题：计算 $28°35'42'' + 36°48'25''$
解：$42''+25''=67''=1'7''$，$35'+48'+1'=84'=1°24'$，$28°+36°+1°=65°$。结果为 $65°24'7''$。

⚠️ 易错：忘掉进位，直接写 $64°83'67''$ 不化简；减法借位时忘记 $1°=60'$

3 分类讨论线段问题

当点在线段上的位置不确定时，要分类讨论（在线段上、在线段的延长线上等）。

例题：线段 $AB=10$，点 $C$ 在直线 $AB$ 上，$BC=4$，求 $AC$。
解：① $C$ 在 $B$ 右侧：$AC=AB+BC=14$；② $C$ 在 $B$ 左侧（$AB$ 上）：$AC=AB-BC=6$。$AC=6$ 或 $14$

⚠️ 易错："在直线AB上"≠"在线段AB上"；审题注意"直线"、"线段"、"射线"的区别

4 钟表角度问题

钟表一圈 $360°$，12 大格，每格 $30°$。分针每分钟走 $6°$，时针每分钟走 $0.5°$。

例题：3点整，时针与分针夹角为多少？
解：3点整分针在12，时针在3，差3大格。$3 \times 30° = 90°$

⚠️ 易错：计算某一时刻夹角时，时针不是恰好指在刻度上，需加偏移量（n点m分：时针角度=$30n+0.5m$）

📝 变式（含偏移量）：3:15时，时针与分针的夹角是多少？
解：分针在3处（15分→90°位置）。时针：$30×3 + 0.5×15 = 90+7.5 = 97.5°$。夹角 = $|97.5°-90°| = 7.5°$（注意：不只3点整才简单，用偏移公式 $30n+0.5m$ 算任意时刻）

📖 核心概念
对顶角与邻补角

对顶角：两直线相交，一个角的两边是另一个角两边的反向延长线。对顶角相等。
邻补角：有一条公共边，另一边互为反向延长线。邻补角互补。

垂线

两直线相交成 $90°$ 时，两直线互相垂直。过一点有且只有一条直线与已知直线垂直。连接直线外一点与直线上各点的所有线段中，垂线段最短。

三线八角

两条直线被第三条直线所截，形成8个角。同位角（F型）、内错角（Z型）、同旁内角（U型）。识别时看位置关系，不要死记图形。

判定 vs 性质

判定（由角推线）：已知角的关系 → 判断两直线平行。
性质（由线推角）：已知两直线平行 → 得出角的关系。两者互逆，不可混淆！

平移

把一个图形整体沿某一直线方向移动。性质：①平移不改变图形的形状和大小；②连接各组对应点的线段平行（或在同一直线上）且相等。平移由方向和距离决定。

命题

命题：判断一件事情的语句。由题设和结论两部分组成，通常写成"如果……那么……"的形式。"如果"后是题设，"那么"后是结论。

真命题与假命题

真命题：正确的命题（题设成立时，结论一定成立）。经过推理证实的真命题叫定理。
假命题：错误的命题。判断假命题只需举一个反例（满足题设但不满足结论）。

📐 公式速查
相交线结论

对顶角：$\angle 1 = \angle 3$，$\angle 2 = \angle 4$

邻补角：$\angle 1 + \angle 2 = 180°$

平行线判定（已知角→判平行）

同位角相等 $\implies l_1 \parallel l_2$

内错角相等 $\implies l_1 \parallel l_2$

同旁内角互补 $\implies l_1 \parallel l_2$

平行线性质（已知平行→得角）

$l_1 \parallel l_2 \implies$ 同位角相等

$l_1 \parallel l_2 \implies$ 内错角相等

$l_1 \parallel l_2 \implies$ 同旁内角互补

平行公理与距离

平行公理：过直线外一点有且只有一条直线与已知直线平行

点到直线的距离 = 垂线段的长度

两平行线间的距离处处相等

平移性质

平移前后：对应线段平行且相等；对应角相等

对应点所连线段平行（或在同一直线上）且相等

平移作图：定方向 → 定距离 → 画对应点 → 连线

命题相关公式

命题改写："对顶角相等" → "如果两个角是对顶角，那么这两个角相等"

判断真假：若题设成立但结论不成立，则命题为假命题

举反例：只需一个反例即可证明假命题

💡 解题技巧

1 三线八角快速识别

看两个角的两边分别由截线和被截线构成。先找截线（公共边所在直线），再看两边位置关系确定是同位角、内错角还是同旁内角。

口诀：同位"F"型、内错"Z"型、同旁内"U"型（辅助记忆，实际位置可能变形）

⚠️ 易错：两条被截线不平行时，同位角不一定相等！判定定理的前提是"被第三条直线所截"

💡 原理：三线八角的本质是两条直线被第三条直线所截形成的8个角。三条线中有一条是"截线"（公共边所在直线），另外两条是"被截线"。先确定截线，再看两个角的边分别属于哪两条被截线，位置关系自然就明确了。不要死记"F""Z""U"——有时图形旋转后就认不出来了。

📝 变式：两条平行线被第三条直线所截，已知 $\angle 1 = 65°$，求其余7个角的度数。
解：利用对顶角相等和邻补角互补写出同截点4个角，再利用平行性质（同位角相等/内错角相等）映射到另一截点。$\angle 1 = 65°$，则 $\angle 3 = 65°$（对顶角），$\angle 2 = \angle 4 = 115°$（邻补角），另一截点对应角同理。

2 命题的改写与判断

改写：找出题设和结论，用"如果……那么……"连接。判断真假：假设题设成立，验证结论是否必然成立。成立→真命题，不成立→假命题（给一个反例）。

例题1：将"同位角相等"改写为"如果……那么……"形式，并判断真假。
解：如果两个角是同位角，那么这两个角相等。这是假命题（两条直线不平行时同位角不相等）。

例题2：判断"相等的角是对顶角"的真假。
解：假命题。反例：两条平行线被第三条直线所截，同位角相等但不是对顶角。

⚠️ 易错：真命题的逆命题不一定是真命题。如"对顶角相等"为真，但逆命题"相等的角是对顶角"为假

3 "由角推线" vs "由线推角"

审题时分清方向：要求证明平行（用判定），还是已知平行求角度（用性质）。判定和性质是互逆命题。

判定：已知 $\angle 1 = \angle 2$，求证 $a \parallel b$。
性质：已知 $a \parallel b$，$\angle 1 = 70°$，求 $\angle 2$。

⚠️ 易错：把性质当判定用，或反过来

💡 原理：这是初一最重要的逻辑思维训练——充分条件和必要条件的区别。判定定理：已知角的条件 → 结论是平行（角是平行的充分条件）；性质定理：已知平行 → 结论是角的关系（平行是角相等的充分条件）。两者互为逆命题，真命题的逆命题不一定真！

📝 变式（真题型）：如图，已知 $\angle 1 = \angle 2$，判断 $AB$ 是否平行于 $CD$，并说明理由。
解：$\angle 1$ 和 $\angle 2$ 是同位角，同位角相等 → $AB \parallel CD$（用的是判定定理）。若已知 $AB \parallel CD$，求 $\angle 2=70°$ 时 $\angle 1$ 的度数，则用性质定理回答 $\angle 1 = 70°$。

📋 中考怎么考：中考几何证明题的核心（6-8分）。阅卷时"判定理由写错"直接0分——题目问为什么平行要写判定定理，问为什么角相等要写性质定理。

第6章 · 实数

🎯 学习目标

理解平方根、算术平方根、立方根的概念；能估算无理数；掌握实数的运算法则

🔗 前置知识：有理数运算（第1章）。注意：$\sqrt{a}$ 的结果总是 ≥ 0（算术平方根），这和第1章绝对值的非负性一脉相承。
🧭 数形结合（二）：$\sqrt{2}$ 有多大？在数轴上找到1和2之间，用夹逼法逐步逼近——这是"无限逼近"思想的雏形。$\sqrt{a^2} = |a|$ 本质是：算术平方根总是非负的（数轴上只能取非负距离）。
🧠 思维导图
实数
  • 平方根
    • 算术平方根 $\sqrt{a} \geq 0$（非负平方根）
    • 平方根 $\pm\sqrt{a}$（正数有两个，0只有一个，负数没有）
  • 立方根 $\sqrt[3]{a}$：每个实数有唯一立方根
  • 无理数 无限不循环小数（如 $\sqrt{2}, \pi$）
  • 实数分类 实数 = 有理数（整数+分数）+ 无理数
  • 实数运算 数轴可一一对应 · 运算法则同有理数
📖 核心概念
算术平方根

若 $x^2 = a$ 且 $x \geq 0$，则 $x = \sqrt{a}$（$a \geq 0$）。$\sqrt{0}=0$，负数没有算术平方根。$\sqrt{a^2} = |a|$。

平方根

若 $x^2 = a$，则 $x = \pm\sqrt{a}$。正数有两个互为相反数的平方根，$0$ 只有一个，负数没有平方根。

立方根

若 $x^3 = a$，则 $x = \sqrt[3]{a}$。每个实数都有唯一的立方根。$\sqrt[3]{-a} = -\sqrt[3]{a}$。

实数的分类

实数 $\begin{cases} \text{有理数} \begin{cases} \text{整数} \\ \text{分数} \end{cases} \\ \text{无理数（无限不循环小数）} \end{cases}$

📐 公式速查
平方根

$\sqrt{a} \geq 0\ (a \geq 0)$

$(\sqrt{a})^2 = a\ (a \geq 0)$

$\sqrt{a^2} = |a| = \begin{cases} a & a \geq 0 \\ -a & a < 0 \end{cases}$

立方根

$(\sqrt[3]{a})^3 = \sqrt[3]{a^3} = a$

$\sqrt[3]{-a} = -\sqrt[3]{a}$

负数有立方根，且为负数

根式运算

$\sqrt{a} \cdot \sqrt{b} = \sqrt{ab}\ (a,b \geq 0)$

$\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}\ (a \geq 0, b > 0)$

$\sqrt{a^2 b} = a\sqrt{b}\ (a \geq 0, b \geq 0)$

常见平方根值

$\sqrt{4}=2,\ \sqrt{9}=3,\ \sqrt{16}=4,\ \sqrt{25}=5,\ \sqrt{36}=6$

$\sqrt{2}\approx1.414,\ \sqrt{3}\approx1.732,\ \sqrt{5}\approx2.236$

💡 解题技巧

1 估算无理数（夹逼法）

找到比这个数小的最大完全平方数和比它大的最小完全平方数，夹在中间估计。

例题：估计 $\sqrt{50}$ 在哪两个整数之间。
解：$7^2=49<50$，$8^2=64>50$，$\therefore 7 < \sqrt{50} < 8$

⚠️ 易错：$\sqrt{50} \neq 25$，不要和 $50 \div 2$ 混淆

2 无理数比较大小

方法一：平方比较法（正数：平方大的原数大）；方法二：估值比较法（先估算大约值）。

例题：比较 $\sqrt{10}$ 和 $\pi$ 的大小。
解：$(\sqrt{10})^2=10$，$\pi^2\approx9.86$，$\therefore \sqrt{10}>\pi$

⚠️ 易错：直接比较 $\sqrt{10}\approx3.162$ vs $\pi\approx3.142$ 可能估不准，平方法更可靠

3 平方根 vs 算术平方根

平方根有两个：正数的平方根是 $\pm\sqrt{a}$，0的平方根是0。$\sqrt{16} = 4$，但16的平方根是 $\pm4$。区分技巧：看到 $\sqrt{\phantom{x}}$ 符号，结果 ≥ 0（算术平方根）；问"平方根"必须写 $\pm$。

例题：求 $16$ 的平方根和算术平方根。
解：平方根为 $\pm 4$，算术平方根为 $4$（即 $\sqrt{16} = 4$）。

⚠️ 易错：$\sqrt{16} = \pm 4$（错误！√符号只表示算术平方根，必须 ≥ 0）

4 $\sqrt{a^2} = |a|$ 防遗忘

$\sqrt{a^2}$ 不等于 $a$，而是等于 $|a|$。因为 $\sqrt{\phantom{x}}$ 的结果必须 ≥ 0。若 $a$ 为负，$|a| = -a$。

例题：化简 $\sqrt{(-5)^2}$
解：$\sqrt{(-5)^2} = |-5| = 5$（不是 $-5$！）

⚠️ 易错：$\sqrt{a^2} = a$ 不全面，必须是 $\sqrt{a^2} = |a|$，考试极高频

第7章 · 平面直角坐标系

🎯 学习目标

理解平面直角坐标系的概念；能根据坐标描点；掌握点的平移和对称坐标规律

🔗 前置知识：数轴（第1章）· 数的比较大小。后续关联：坐标系是函数（八年级）的基础，平移概念与第5章平移一节紧密相关。
🧭 数形结合（三）：坐标系将"数"和"形"完美统一——一个点对应一对数，一条线段对应一个距离公式，平移对应坐标加减。从数轴（一维）到坐标系（二维），是数学抽象能力的一次跃升。
🧠 思维导图
平面直角坐标系
  • 有序数对 $(a,b)$ 表示位置，顺序不能颠倒
  • 坐标 $P(x,y)$：$x$横坐标，$y$纵坐标
  • 象限
    • 第一象限 $(+,+)$
    • 第二象限 $(-,+)$
    • 第三象限 $(-,-)$
    • 第四象限 $(+,-)$
  • 坐标轴上的点 $x$轴上 $y=0$ · $y$轴上 $x=0$ · 原点 $(0,0)$ 不在任何象限
  • 平移 左减右加 $x$，上加下减 $y$
  • 点到坐标轴距离 到 $x$ 轴：$|y|$ · 到 $y$ 轴：$|x|$
📖 核心概念
有序数对

用两个数表示一个确定的位置，如 $(3,2)$ 表示第3列第2排。$(a,b)$ 与 $(b,a)$ 一般表示不同位置。

象限知识点

第一象限 $(+,+)$、第二象限 $(-,+)$、第三象限 $(-,-)$、第四象限 $(+,-)$。坐标轴上的点不属于任何象限。

坐标轴上的点

$x$ 轴上的点：纵坐标为 $0$，即 $(a,0)$。
$y$ 轴上的点：横坐标为 $0$，即 $(0,b)$。
原点：$(0,0)$。

对称点坐标

关于 $x$ 轴对称：$(a,b) \to (a,-b)$
关于 $y$ 轴对称：$(a,b) \to (-a,b)$
关于原点对称：$(a,b) \to (-a,-b)$

点到坐标轴的距离

$P(a,b)$ 到 $x$ 轴的距离 = $|b|$（纵坐标绝对值）；到 $y$ 轴的距离 = $|a|$（横坐标绝对值）。注意：到坐标轴的距离一定是非负的！考试极高频率。

📐 公式速查
点的平移

向左移 $a$ 个单位：$(x,y) \to (x-a,y)$

向右移 $a$ 个单位：$(x,y) \to (x+a,y)$

向上移 $b$ 个单位：$(x,y) \to (x,y+b)$

向下移 $b$ 个单位：$(x,y) \to (x,y-b)$

口诀：左减右加 x，上加下减 y

对称规律

关于 $x$ 轴对称：$y$ 变相反数

关于 $y$ 轴对称：$x$ 变相反数

关于原点对称：$x$、$y$ 都变相反数

坐标求线段长与面积

水平线段（$y_1=y_2$）：长度 = $|x_1 - x_2|$

竖直线段（$x_1=x_2$）：长度 = $|y_1 - y_2|$

三角形面积（割补法）：用长方形包围，减去外围三角形

点到坐标轴的距离

点 $P(x, y)$ 到 $x$ 轴的距离 = $|y|$

点 $P(x, y)$ 到 $y$ 轴的距离 = $|x|$

⚠️ 距离开绝对值！如 $P(3,-4)$ 到 $x$ 轴距离为 $|-4|=4$，不是 $-4$

💡 解题技巧

1 描点法画图

先列表（选取 $x$ 值算出对应的 $y$），再在坐标系中描点，最后连线。注意点取得越多越精确。

例题：描出 $A(-2,3)$、$B(4,0)$、$C(1,-2)$ 的位置。
解：分别在两条坐标轴上标出横纵坐标，过该点作垂线的交点即所求。

⚠️ 易错：$(x,y)$ 第一个数是横坐标，第二个是纵坐标，顺序不能搞反

2 坐标求面积（割补法）

不规则三角形面积：用长方形包围三角形，总面积减去周围几个小三角形面积。

例题：给三点坐标 $A(1,1),B(5,1),C(3,4)$，求 $\triangle ABC$ 面积。
解：$S = \frac{1}{2} \times |5-1| \times |4-1| = \frac{1}{2} \times 4 \times 3 = 6$（底在x轴平行方向）

⚠️ 易错：线段长度 = 坐标差的绝对值，别忘加绝对值符号

第8章 · 二元一次方程组

🎯 学习目标

理解二元一次方程（组）的概念；熟练运用代入法和加减法解方程组；能建立方程组解决实际问题

🔗 前置知识：一元一次方程解法（第3章）。消元的核心是将二元降为一元，与第3章解法完全一致。后续关联：三元方程组（九年级）同理逐步消元。
🧭 消元思想：二元→一元，化未知为已知，化复杂为简单——这是数学最核心的降维思维。选择消元方法的标准：哪个未知数容易表示 → 代入法；哪个未知数系数已有关系 → 加减法。选择比计算更重要。

📋 方程组应用题通用模板

  1. 设两个元 — 设两个未知数，写清楚各代表什么
  2. 找两个等量关系 — 两个条件→两个方程（缺一不可）
  3. 选方法 — 观察系数：有±1→代入法；系数相同/相反→加减法
  4. 消元求解 — 化二元为一元，解出后再回代求另一个
  5. 规范书写 — 解必须写成 $\begin{cases} x=m \\ y=n \end{cases}$ 形式
  6. 检验 — 代入两个原方程都成立
🧠 思维导图
二元一次方程组
  • 二元一次方程 含两个未知数，未知数次数为1的整式方程 $ax+by=c$
  • 二元一次方程组 两个二元一次方程联立 $\begin{cases} ax+by=c \\ dx+ey=f \end{cases}$
  • 方程组的解 同时满足两个方程的 $x,y$ 值，写成 $\begin{cases} x=m \\ y=n \end{cases}$
  • 解法
    • 代入消元法 将一个未知数用另一个表示，代入另一方程
    • 加减消元法 乘适当倍数使某未知数系数相等或相反，相加减消元
  • 应用 鸡兔同笼 · 配套问题 · 数字问题 · 方案选择
📖 核心概念
二元一次方程

含有两个未知数，且含有未知数的项的次数都是1的整式方程。标准形式：$ax + by = c$（$a$、$b$不全为$0$）。一个二元一次方程有无数组解。

二元一次方程组

由两个二元一次方程联立组成。两个方程中同一未知数表示同一个量。一般有唯一解。

消元思想

将二元转化为一元是解方程组的核心思想。两种方法：代入消元法（用一个未知数表示另一个然后代入）、加减消元法（通过加减消除一个未知数）。

三元一次方程组*

含三个未知数的方程组，解法思路相同——逐步消元，化为二元→一元求解。人教版七年级下册阅读材料，中考常涉及。

📐 公式速查
代入消元法步骤

① 从一个方程中解出一个未知数（用含另一个的式子表示）

② 代入另一个方程，消去一个未知数 → 得一元一次方程

③ 解出这个未知数，再回代求另一个

④ 写出解 $\begin{cases} x=m \\ y=n \end{cases}$

加减消元法步骤

① 观察系数，看哪个未知数系数绝对值相等或成倍数

② 必要时方程两边同乘一个数，使某未知数系数相等或相反

③ 相加（系数相反）或相减（系数相同）消去一个未知数

④ 解一元一次方程，回代求另一个

常用应用题公式

鸡兔同笼：头数 $x+y=n$，腿数 $2x+4y=m$

配套问题：产品A的某比例 = 产品B的某比例

数字问题：两位数 $=10a+b$（$a$为十位，$b$为个位）

💡 解题技巧

1 选择消元方法

代入法适用：某个方程中一个未知数的系数为1或-1时最方便。
加减法适用：两个方程中某未知数系数相等、相反或成倍数关系时。

例：$\begin{cases} x = 2y + 1 \\ 3x + y = 10 \end{cases}$ 用代入法；$\begin{cases} 2x + 3y = 7 \\ 4x - 3y = 5 \end{cases}$ 用加减法（$3y$与$-3y$）

⚠️ 易错：选错方法导致计算量大增，先观察系数再决定

💡 原理：消元的核心思想是"化二元为一元"。代入法——用含一个未知数的式子替换另一个方程中的该未知数，消去一个未知数；加减法——利用等式性质使某一未知数的系数相等或相反，然后加减消去。选择标准：哪个未知数容易用一个表示（系数为±1）→代入法；哪个未知数系数已相等/相反/成倍数→加减法。

📝 变式（真题型）：解 $\begin{cases} 3x - 2y = 10 \\ x + y = 5 \end{cases}$
分析：第二个方程 $y=5-x$ 代入第一个，或第二个×2加第一个消 $y$。代入更快：$3x-2(5-x)=10$，$3x-10+2x=10$，$5x=20$，$x=4$，代入得 $y=1$

2 鸡兔同笼问题

设未知量时选好变量（如设鸡x只，兔y只），利用"头数"和"腿数"两个等量关系列方程组。

例题：鸡兔同笼，共35头，94条腿。求鸡兔各几只。
解：设鸡 $x$ 只，兔 $y$ 只。$\begin{cases} x+y=35 \\ 2x+4y=94 \end{cases}$，解得 $\begin{cases} x=23 \\ y=12 \end{cases}$

⚠️ 易错：兔4条腿、鸡2条腿，别写反了

💡 原理：鸡兔同笼是二元一次方程组最经典的应用。两个等量关系：①头数（鸡+兔=总数）；②腿数（2×鸡+4×兔=总腿数）。设两个未知数后，两个条件正好对应两个方程。如果只有一个条件，方程有无数组解——这就是为什么要两个方程才能确定两个未知数。

1 乘除负数必须变号

解不等式跟解方程几乎一样，但只要乘或除以负数，不等号方向就必须改变。养成检查最后一步系数符号的习惯。

例题：解 $-3x \geq 12$

⚠️ 易错：$-3x \geq 12 \implies x \geq -4$（忘记变号，这是最高频错误！）

💡 原理：为什么乘除负数要变号？以 $-3 > -5$ 为例，两边除以 $-3$（负数）得 $1 < \frac{5}{3}$。如果不变号就变成 $1 > \frac{5}{3}$，这显然是错的。从数轴上看：乘除负数相当于把数轴"翻转"（乘-1就是关于原点对称），原本右边的大数翻转到左边，变成小数。

2 "至少/至多"翻译技巧

至少：$\geq$  ·  至多/不超过：$\leq$  ·  超过/大于：$>$  ·  不足/小于：$<$

例题：某班至少42人及格，总人数50人，求及格率。

⚠️ 易错："至少"用 $\geq$ 还是 $>$？至少包含这个数，用 $\geq$

💡 原理：应用题翻译的难点是把"语文"转化为"数学"。口诀：至少→≥（不少于），至多→≤（不超过），超过→>，不足→<。注意"至少"包含该数（用≥），"超过"不包含该数（用>）。例："至少80分"意思是 ≥80，"超过80分"意思是 >80。

抽样调查：抽取一部分对象调查，根据样本估计总体。关键在于样本要有代表性和足够大。

总体·个体·样本

总体：要考察的全体对象。
个体：总体中每一个考察对象。
样本：从总体中抽取的一部分个体。
样本容量：样本中个体的数目（没有单位）。

条形图：比较数量多少。
折线图：展示变化趋势。
扇形图：展示百分比构成。
直方图：展示连续数据的分布情况。

频数分布直方图

组距：每组两个端点之间的距离。
频数：落在各小组内的数据个数。
频率：频数与数据总数的比值。各小长方形面积表示频数。

📐 公式速查
扇形圆心角

$$\text{圆心角} = \text{百分比} \times 360°$$

或：$\text{圆心角} = \frac{\text{该部分数量}}{\text{总数}} \times 360°$

频率

$$\text{频率} = \frac{\text{频数}}{\text{数据总数}}$$

所有频率之和 = $1$（100%）

组数与组距

组数 ≈ $\frac{\text{最大值} - \text{最小值}}{\text{组距}}$（向上取整）

组数一般取 5~12 组为宜

统计概念区分

总体：要考察的全体对象

个体：总体中的每一个考察对象

样本：从总体中抽取的一部分个体

样本容量：样本中个体的数目（无单位）

⚠️ 高频易错陷阱总览
🔴 符号错误

$(-2)^2=4$ 别写成 $-4$（底数是$-2$）
$-2^2=-4$ 别写成 $4$（底数是$2$）
移项不変号（$x+5=3 \to x=3+5$）
去括号不変号（$-(a+b) \to -a+b$）
解不等式乘除负数不変号

🔴 漏乘漏项

去分母漏乘沒有分母的項
$2(x+3)$ 写成 $2x+3$（漏乘$2\times3$）
多项式缺项忘补0占位
合并同类项忘系数为1時不写

🔴 概念混淆

$\sqrt{16}=4$ ≠ $\pm4$（符号表示算术平方根）
$\sqrt{a^2}=|a|$ ≠ $a$
$x^2y$与$xy^2$不是同类项
判定vs性质混淆（由角推线 vs 由线推角）
频数（个数）vs 频率（比值）混淆

🔴 审题不清

"在直线AB上"≠"在线段AB上"（需分类讨论）
"平方根"vs"算术平方根"
"非负"包括0
坐标$(x,y)$顺序不能反
"至少"=≥，"超过"=>，"不超过"=≤

🔴 格式不规范

方程组解写成 $x=1,y=2$ 而非 $\begin{cases}x=1\\y=2\end{cases}$
度分秒不化簡（$64°83'$ → $65°23'$）
几何证明不写"∵∴"
应用题只写算式不写"答："
