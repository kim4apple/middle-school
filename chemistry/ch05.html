第5章 · 化学方程式

九年级上册 | 质量守恒 · 配平方法 | 初中化学最重要的计算工具

🧪 实验一：红磷燃烧验证质量守恒定律
1取一个锥形瓶，底部铺一层细沙（防止红磷燃烧产生的高温使锥形瓶炸裂），放入少量红磷。
2在锥形瓶口套一个气球（缓冲压力，使实验在密闭体系中进行），用天平称量整个装置的总质量 m₁。
3将锥形瓶放在石棉网上加热（或用玻璃棒在酒精灯上灼烧后接触红磷），引燃红磷——红磷燃烧产生大量白烟（P₂O₅固体），气球先膨胀后变瘪。
4冷却至室温后再次称量——总质量 m₂。比较：m₁ = m₂——反应前后总质量不变！

📋 结论：参加化学反应的各物质的质量总和，等于反应后生成的各物质的质量总和——这就是质量守恒定律。

🧪 实验二：铁钉与硫酸铜溶液反应验证质量守恒定律
1在烧杯中加入适量硫酸铜蓝色溶液，将几根打磨光亮的铁钉放入烧杯中。
2用天平称量反应前烧杯+铁钉+溶液的总质量 m₁。
3静置一段时间，观察现象——铁钉表面析出红色固体（铜），蓝色溶液颜色变浅（变为浅绿色FeSO₄溶液）。
4再次称量总质量 m₂——m₁ = m₂（此实验无需密闭装置，因为没有气体参与或生成）。

📋 化学方程式：Fe + CuSO₄ → FeSO₄ + Cu ——这是典型的置换反应，也是验证质量守恒定律的经典实验。

🧠 化学变化的微观本质——质量守恒的微观解释

为什么化学反应前后物质的总质量不变？从微观角度看：化学反应是分子被破坏→原子重新组合的过程。

在化学反应中，反应前后：

微粒"六个不变""两个一定变""一个可能变"
宏观①元素种类不变
②物质总质量不变		①物质种类一定改变（生成了新物质）
②分子种类一定改变（分子被破坏、生成新分子）		分子个数可能改变（如2H₂+O₂→2H₂O中3分子→2分子）
微观①原子种类不变
②原子数目不变
③原子质量不变

质量守恒定律是一切化学反应都必须遵守的普遍规律，是配平化学方程式的根本依据。

📖 核心概念

质量守恒定律

参加化学反应的各物质的质量总和，等于反应后生成的各物质的质量总和。注意：①适用范围——一切化学反应（物理变化不适用）；②"参加反应"——不包括剩余的反应物；③"质量总和"——包括气体的质量（有气体参与时必须密闭装置）。

化学方程式的意义

以 2H₂ + O₂ → 2H₂O 为例：①质的方面：表示反应物（H₂和O₂）、生成物（H₂O）和反应条件（点燃）；②量的方面：表示各物质间的微粒个数比（2:1:2）和质量比（4:32:36=1:8:9）。

配平原则

在化学式前面添加适当化学计量数，使反应前后各元素原子的种类相同、个数相等。遵循质量守恒定律。注意：不能改动化学式中的角标（如H₂O不能改成HO或H₂O₂），只能调整前边的系数。

化学反应类型（四大基本类型）

①化合反应：A+B+… → X（多变一）
②分解反应：X → A+B+…（一变多）
③置换反应：A+BC → AC+B（单换单）
④复分解反应：AB+CD → AD+CB（互交换分，需生成沉淀/气体/水）

化学方程式的读法

以 C + O₂ → CO₂ 为例：①宏观读法：碳和氧气在点燃条件下反应生成二氧化碳；②微观读法：每1个碳原子和1个氧分子反应生成1个二氧化碳分子；③质量读法：每12份质量的碳和32份质量的氧气反应生成44份质量的二氧化碳。

⚗️ 配平方法详解 + 配平练习

方法一：观察法（最小公倍数法）

以配平 KClO₃ → KCl + O₂↑ 为例：左边O原子3个，右边O原子2个。3和2的最小公倍数是6。在O₂前配3（右O=6），在KClO₃前配2（左O=6）→ 2KClO₃ → KCl + 3O₂↑。再观察：左K=2，右K=1 → 在KCl前配2 → 2KClO₃ → 2KCl + 3O₂↑。

方法二：待定系数法（万能法）

以配平 C₂H₂ + O₂ → CO₂ + H₂O 为例：设 C₂H₂ 系数=1 → 左C=2, H=2。则 CO₂ 前配2（右C=2），H₂O 前配1（右H=2）。右O总数=2×2+1=5 → O₂ 前配5/2。通分×2 → 2C₂H₂ + 5O₂ → 4CO₂ + 2H₂O。

📝 配平练习（6个方程式）：

① C + O₂ → CO₂  （已平：1:1:1）

② P + O₂ → P₂O₅  （配平：4P + 5O₂ → 2P₂O₅）

③ KClO₃ → KCl + O₂  （配平：2KClO₃ → 2KCl + 3O₂↑）

④ H₂O₂ → H₂O + O₂  （配平：2H₂O₂ → 2H₂O + O₂↑）

⑤ Fe + O₂ → Fe₃O₄  （配平：3Fe + 2O₂ → Fe₃O₄）

⑥ C₂H₄ + O₂ → CO₂ + H₂O  （配平：C₂H₄ + 3O₂ → 2CO₂ + 2H₂O）

🎬 配平演示：H₂ + O₂ → H₂O

H₂ + O₂ → H₂O

🔢 化学方程式计算——规范格式（三步法）

规范格式：设未知量 → 写出化学方程式 → 找出已知量与未知量的质量关系 → 列比例式 → 求解 → 答

例题1（基础）：6g碳在足量氧气中完全燃烧，能生成多少克二氧化碳？

解：设能生成CO₂的质量为x。

C + O₂ → CO₂

12              44

6g             x

12/44 = 6g/x → x = 6g × 44 / 12 = 22g

答：能生成22g二氧化碳。

例题2（升华）：实验室用加热高锰酸钾制取氧气。充分加热31.6g高锰酸钾，能制得多少克氧气？

解：设能制得O₂的质量为x。

2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑

2×158=316                    32

31.6g                       x

316/32 = 31.6g/x → x = 31.6g × 32 / 316 = 3.2g

答：能制得3.2g氧气。

🚨 常见错误辨析

🔴 化学方程式书写误区

❌ 忘记标反应条件（如"点燃""加热""高温""通电""催化剂"）。
❌ 气体↑：只有反应物中没有气体而生成物中有气体时才标↑（如H₂O₂→H₂O+O₂↑）。反应物中已有气体则不需要标（如C+O₂→CO₂——O₂已是气体，不标↑）。
❌ 沉淀↓：只有反应在溶液中进行，生成不溶物时才标↓。
❌ 配平前就做计算——必须先配平！化学计算必须基于正确的化学方程式！

⚠️ 计算格式误区

❌ 设未知量带单位——错！x不带单位，最后答中写"x g"。
❌ 相对分子质量算错——如KMnO₄的相对分子质量=39+55+16×4=158，漏算角标。
❌ 列的化学方程式下面，已知量的质量和所求量写在对应物质的正下方——写错位置会导致比例式错误。
❌ 忘了求化学计量数×相对分子质量——如2KMnO₄是2×158=316不是158。

🎯 中考真题精选（化学方程式 · 近年典型）
九上★☆☆
下列现象能用质量守恒定律解释的是（ ）
A. 10g水变成10g水蒸气   B. 铁钉生锈后质量增加
C. 湿衣服晾干后质量减小   D. 蜡烛燃烧越来越短
答案：B
🔑 关键思路：质量守恒定律只适用于化学变化。A和C是物理变化（水的蒸发），不适用。D蜡烛燃烧生成CO₂和H₂O逸散到空气中，所以蜡烛质量减小——但参加反应的"石蜡+氧气"的总质量 = 生成的"CO₂+H₂O"的总质量，仍然守恒，只是产物逸散了。B铁+氧气+水→铁锈，产物质量增加了氧和水的质量，符合守恒。
💡 显示/隐藏解析
九上★★☆
配平化学方程式：____C₂H₆ + ____O₂ → ____CO₂ + ____H₂O，系数分别为（ ）
A. 1, 7, 2, 3   B. 2, 7, 4, 6   C. 1, 3, 2, 3   D. 2, 5, 4, 3
答案：B
🔑 关键思路：设C₂H₆系数=1 → 左C=2, H=6 → 右CO₂前配2, H₂O前配3。右O总数=2×2+3=7 → O₂前配7/2。通分×2 → 得 2C₂H₆ + 7O₂ → 4CO₂ + 6H₂O。选B（验证：2,7,4,6）。
💡 显示/隐藏解析
九上★★☆
在反应 2A + 3B → 2C + 4D 中，已知6g A与足量B完全反应生成18g C和8g D，则参加反应的B的质量是（ ）
A. 12g   B. 16g   C. 20g   D. 26g
答案：C
🔑 关键思路：根据质量守恒定律：m(A) + m(B) = m(C) + m(D) → 6g + m(B) = 18g + 8g → m(B) = 20g。化学方程式中的系数2:3:2:4只提供质量比，但不影响总量守恒。
💡 显示/隐藏解析
九上★★★
某物质在氧气中完全燃烧，只生成CO₂和H₂O，则该物质中（ ）
A. 一定含有碳、氢、氧三种元素   B. 一定含有碳、氢元素，可能含有氧元素
C. 只含有碳、氢两种元素   D. 无法判断
答案：B
🔑 关键思路：产物中有CO₂→一定有C元素；产物中有H₂O→一定有H元素。产物中的O元素可能来自反应物中的O₂，也可能部分来自该物质——所以该物质中一定含C、H，可能含O。要判断是否含O：比较m(C)+m(H)与物质的总质量——相等则不含O，小于则含O。
💡 显示/隐藏解析

🔬 实验探究设计：有气体生成时验证质量守恒定律的注意事项

问题：敞口装置中做有气体生成的反应，测得的"反应后质量"总是小于"反应前质量"——这不违反质量守恒定律吗？

分析：不违反！如碳酸钠与盐酸反应：Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑——反应生成的CO₂逸散到空气中。敞口称量时，CO₂的质量没有被计入反应后的总质量中，所以测得质量减小。实际上，参加反应的所有物质（Na₂CO₃ + HCl）的质量总和 = 反应后所有产物（NaCl + H₂O + CO₂）的质量总和。

验证方法：将实验改为密闭装置进行——如利用锥形瓶+气球组合（气球收集生成的气体），反应前后总质量不变。

📋 结论：任何有气体参与或生成的反应，验证质量守恒定律时必须使用密闭装置。镁条在空气中燃烧后固体质量"增加"——是因为氧气参与了反应（Mg+O₂→MgO），如果用密闭装置称量则总质量不变。
