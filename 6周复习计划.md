## 三、物理 八年级 知识清单

### 八年级（上）

#### 第1章 机械运动
- [ ] 长度和时间的测量（刻度尺、停表）
- [ ] 运动的描述（参照物）
- [ ] 速度公式 v=s/t（单位换算：m/s↔km/h）
- [ ] 匀速直线运动、s-t 图像、v-t 图像
- [ ] 平均速度（≠速度平均值）

#### 第2章 声现象
- [ ] 声音的产生（振动）与传播（需要介质）
- [ ] 声速（空气中约 340m/s）
- [ ] 声音三特性：音调（频率）、响度（振幅）、音色
- [ ] 超声波与次声波
- [ ] 噪声的控制途径：声源→传播→人耳

#### 第3章 物态变化
- [ ] 温度计原理（液体热胀冷缩）
- [ ] 六种物态变化
  - 熔化（吸热）、凝固（放热）
  - 汽化（吸热：蒸发/沸腾）、液化（放热）
  - 升华（吸热）、凝华（放热）
- [ ] 晶体与非晶体（有无固定熔点）
- [ ] 沸点与气压的关系

#### 第4章 光现象
- [ ] 光的直线传播（小孔成像、日食月食）
- [ ] 光速（真空中 3×10⁸ m/s）
- [ ] 光的反射定律（三线共面、两角相等）
- [ ] 镜面反射与漫反射
- [ ] 平面镜成像特点（等大、等距、对称、虚像）
- [ ] 光的折射规律
- [ ] 光的色散（白光由七色光组成）

#### 第5章 透镜及其应用
- [ ] 凸透镜（会聚）与凹透镜（发散）
- [ ] 凸透镜成像规律（五幅图，必须背熟）：
  | 物距 u | 像距 v | 像的性质 | 应用 |
  |--------|--------|---------|------|
  | u>2f | f<v<2f | 倒立缩小实像 | 照相机 |
  | u=2f | v=2f | 倒立等大实像 | — |
  | f<u<2f | v>2f | 倒立放大实像 | 投影仪 |
  | u=f | — | 不成像 | — |
  | u<f | — | 正立放大虚像 | 放大镜 |

- [ ] 眼睛与眼镜（近视→凹透镜，远视→凸透镜）
- [ ] 显微镜与望远镜

#### 第6章 质量与密度
- [ ] 质量（天平使用：水平→归零→调平→左物右码）
- [ ] 密度公式 ρ=m/V（单位换算：g/cm³→kg/m³ ×10³）
- [ ] 量筒测体积
- [ ] 密度应用：鉴别物质、空心实心判断
- [ ] 测量液体密度和固体密度的实验步骤

### 八年级（下）

#### 第7章 力
- [ ] 力的作用效果（形变、运动状态改变）
- [ ] 力的三要素、力的示意图（带箭头的线段！）
- [ ] 弹力（弹簧测力计原理：在弹性限度内，伸长量与拉力成正比）
- [ ] 重力 G=mg（g=9.8N/kg≈10N/kg）
- [ ] 重力的方向：竖直向下
- [ ] 摩擦力（滑动摩擦/静摩擦/滚动摩擦）
- [ ] 影响滑动摩擦力的因素：压力大小、接触面粗糙程度

#### 第8章 运动和力
- [ ] 牛顿第一定律（惯性定律）
- [ ] 惯性（一切物体都有惯性，只与质量有关）
- [ ] 二力平衡条件：同体、等大、反向、共线
- [ ] 平衡力与相互作用力的区别

#### 第9章 压强
- [ ] 压强 p=F/S（单位：Pa=N/m²）
- [ ] 增大/减小压强的方法
- [ ] 液体压强 p=ρgh（同深度向各方向压强相等）
- [ ] 连通器原理
- [ ] 大气压强（托里拆利实验：标准大气压=760mmHg=1.013×10⁵Pa）
- [ ] 流体压强与流速的关系（机翼升力）

#### 第10章 浮力
- [ ] 浮力产生原因：上下表面压力差 F浮=F向上-F向下
- [ ] 阿基米德原理 F浮=G排=ρ液gV排
- [ ] 物体的浮沉条件：
  | 状态 | 条件 | F浮 vs G |
  |------|------|---------|
  | 上浮 | ρ物<ρ液 | F浮>G |
  | 下沉 | ρ物>ρ液 | F浮<G |
  | 悬浮 | ρ物=ρ液 | F浮=G |
  | 漂浮 | ρ物<ρ液 | F浮=G |
- [ ] 浮力计算四种方法：称重法、压力差法、阿基米德法、平衡法

#### 第11章 功和机械能
- [ ] 功 W=Fs（有力有距离且方向一致）
- [ ] 功率 P=W/t（单位：W）
- [ ] 动能（与质量、速度有关）
- [ ] 势能：重力势能（mgh）、弹性势能
- [ ] 机械能守恒

#### 第12章 简单机械
- [ ] 杠杆五要素：支点、动力、阻力、动力臂、阻力臂
- [ ] 杠杆平衡条件 F₁L₁=F₂L₂
- [ ] 三类杠杆：省力（L₁>L₂）、费力（L₁<L₂）、等臂（L₁=L₂）
- [ ] 定滑轮（不省力，改方向）、动滑轮（省一半力）、滑轮组
- [ ] 机械效率 η=W有/W总×100%（永远<100%）

# 第四部分：错题管理规范

### 错题本格式

```
| 日期 | 来源 | 原题 | 错误答案 | 正确答案 | 错因分类 | 正确思路 |
```

### 错因分类编码
- **A**：知识漏洞（公式记错、概念混淆）
- **B**：审题失误（看错条件、漏看关键词）
- **C**：计算失误（符号错了、抄错数）
- **D**：方法错误（思路不对、选错定理）
- **E**：时间不够（没来得及做/仓促）

### 错题处理流程
1. 当天错题→当天整理到错题本
2. 第二天→重做一遍（不看答案）
3. 周末→本周错题全部重做
4. 仍出错→标⭐，下周末继续重做，直到消除

### 每周错题统计
每周末对本周错题做一次分类统计：

| 科目 | A类 | B类 | C类 | D类 | E类 | 合计 |
|------|-----|-----|-----|-----|-----|------|
| 数学 | | | | | | |
| 英语 | | | | | | |
| 物理 | | | | | | |

- A 类和 C 类最多 → 基础不牢，下周增加课本回顾时间
- B 类最多 → 审题习惯问题，下周每题圈画关键词
- D 类最多 → 方法选择问题，下周多练"一题多解"对比
- E 类最多 → 速度不足，下周增加限时训练

# 第六部分：各科得分策略

## 物理（新学科，重点得分区）

### 考试策略
| 题型 | 做题策略 |
|------|---------|
| **选择题** | 排除法！不会的也要列公式排除 |
| **填空题** | 注意单位！写公式推导 |
| **作图题** | 用尺规！箭头/虚线/实线分清楚 |
| **实验题** | 先写原理公式，再套实验步骤 |
| **计算题** | 已知→求→解→答，一步不跳 |

### 快速提分突破口（按性价比排序）
1. **作图题**（送分题，练规范即可全拿）
2. **计算题**（格式规范+公式写对=至少拿一半分）
3. **实验题**（背住 7 个实验的步骤和原理，从日常积累）
4. **填空题**（注意单位换算和物理量符号）
5. **选择题**（排除法+概念辨析）

## 数学

### 考试策略
| 题型 | 做题策略 |
|------|---------|
| **选择+填空** | 30 分钟内完成。特殊值法、代入验证、排除法 |
| **计算/化简** | 步步写，不跳步，做完验算 |
| **证明题** | 条件→定理→结论，缺一不可 |
| **应用题** | 先找等量/不等量关系，再列式 |
| **压轴题第1问** | 必做！通常就是基础计算 |
| **统计题** | 送分题，概念清晰+计算仔细即可全拿 |

### 快速提分突破口
1. **计算零失误**（符号+括号+检验，这是底线）
2. **几何证明规范**（每一步写理由，阅卷按步给分）
3. **函数画图分析**（k/b 含义搞清楚，图像画对=至少对一半）
4. **选择填空速度**（30 分钟练熟，保证后面大题时间充足）
5. **统计不丢分**（平均数/中位数/众数/方差概念对比记）

## 英语

### 考试策略
| 题型 | 做题策略 |
|------|---------|
| **听力** | 发卷先看听力题！预读+预测 |
| **单选** | 看选项→判断考点→排除 |
| **完形** | 第一遍通读不填→第二遍上下文推→第三遍语法验证 |
| **阅读** | 先读题再读文→题干关键词定位→同义替换确认 |
| **作文** | 5 分钟审题+提纲→打草稿→工整抄写 |

### 快速提分突破口
1. **单词量**（七八年级 1500+ 词必须认识，每天 20 个是底线）
2. **听力积累**（隔天 1 套不可断，听力是慢功夫）
3. **时态判断**（时间状语是突破口：yesterday/now/already/since）
4. **完形上下文逻辑**（不是语法对就行，要看前后句意思）
5. **作文套模板**（5 种体裁各备一套框架，考试直接套，但每周至少要练 2 篇）
6. **语感培养**（每日 10 分钟课文朗读，潜移默化）
