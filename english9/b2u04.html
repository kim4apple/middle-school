Reduce, Reuse, Recycle

环保3R · 状语从句和宾语从句 · 环境保护

📖 话题：Environmental protection and the 3Rs💬 功能：Give suggestions for environmental protection; express causes and results📚 语法：状语从句 (时间/原因/条件/目的) · 宾语从句 (that/if/whether)📝 写作：环保倡议

🎯 我能做到

📖 课文精讲 · Green Action at School
场景：李明、张华和Sarah在讨论如何在学校减少浪费。
Li Ming:I think that our school produces too much waste every day.
Zhang Hua:You're right. When we use plastic bottles and throw them away, they end up in landfills.
Sarah:That's why we should follow the 3Rs: reduce, reuse, and recycle.
Li Ming:I wonder whether we can recycle paper in the classroom.
Zhang Hua:Yes! We can reuse paper by writing on both sides before recycling it.
Sarah:I believe that if everyone makes small changes, we can make a big difference.
Li Ming:So that we can protect our planet, we should start using reusable water bottles.
📖 查看翻译

李明：我觉得我们学校每天产生太多垃圾了。

张华：你说得对。当我们使用塑料瓶然后扔掉，它们最终会进入垃圾填埋场。

Sarah：这就是为什么我们应该遵循3R原则：减少、再利用和回收。

李明：我想知道我们能否在教室回收纸张。

张华：可以！我们可以在回收之前双面使用纸张来重复利用它。

Sarah：我相信如果每个人做出小的改变，我们就能带来大的不同。

李明：为了保护我们的地球，我们应该开始使用可重复使用的水瓶。

🔑 重点讲解：

I think that... 表示「我认为……」 — I think that our school produces too much waste.

When... 表示「当……时」 — When we use plastic bottles, they end up in landfills.

I wonder whether... 表示「我想知道是否……」 — I wonder whether we can recycle paper.

If... 表示「如果……」 — If everyone makes small changes, we can make a big difference.

So that... 表示「为了/以便」 — So that we can protect our planet, we should use reusable bottles.

📚 语法聚焦 · 状语从句 & 宾语从句

一、状语从句 (Adverbial Clauses)

类别引导词用法例句
时间when / while / before / after表示时间关系When we throw away plastic, it pollutes the earth.
原因because / as / since说明原因或理由We should recycle because it saves resources.
条件if / unless表示条件（如果/除非）If everyone recycles, we can reduce waste.
目的so that表示目的（为了/以便）We save energy so that we can protect the planet.

二、宾语从句 (Object Clauses)

引导词用法例句
that陈述句作宾语，that可省略I think (that) recycling is important.
if / whether一般疑问句作宾语，"是否"I wonder whether they recycle at home.
I'm not sure if this is recyclable.

宾语从句语序

原问句错误语序正确语序
Can we recycle this?I wonder can we recycle this?I wonder whether we can recycle this.
Is it eco-friendly?I'm not sure is it eco-friendly.I'm not sure if it is eco-friendly.
Does this help?Do you know does this help?Do you know if this helps?

状语从句的要点：从句主语与主句主语不一致时，各自保留主语；从句位置可在主句前（加逗号）或主句后。

宾语从句必须用陈述句语序：引导词 + 主语 + 谓语 + 其他。

if 和 whether 都表示"是否"，whether 更正式，可与 or not 连用：I don't know whether he recycles or not.

If you want to help the environment, start by reducing waste. 如果你想帮助环境，从减少垃圾开始。

📖 阅读专项 · The Power of Three

Have you ever wondered what happens to your trash after you throw it away? Every day, people around the world produce millions of tons of waste. Much of it ends up in landfills or pollutes the ocean. But there is a simple way to make a difference — the 3Rs: Reduce, Reuse, and Recycle.

Reduce means using less. When you buy only what you need, you create less waste. For example, instead of buying bottled water, you can carry a reusable bottle. This simple change can reduce plastic waste significantly. Because we use fewer resources when we reduce, we also save energy and protect nature.

Reuse means using things again instead of throwing them away. Glass jars can be used for storage. Old clothes can be turned into cleaning rags. If we think creatively, we can find new uses for old items. Reusing saves resources and reduces the amount of waste we produce.

Recycle means turning waste into new products. Paper, glass, metal, and some plastics can all be recycled. When you put a plastic bottle in the recycling bin, it can be made into something new. So that we can save energy and reduce pollution, we should recycle as much as possible.

The 3Rs are powerful because they work together. When we reduce, we create less waste. When we reuse, we give items a second life. When we recycle, we turn waste into resources. Remember: every small action counts. Start practicing the 3Rs today, and together we can protect our beautiful planet!

✍️ 写作进阶 · 环保倡议
📋 写作题目
假设你是学校环保社团的成员，写一份倡议书，号召同学们践行3R原则来保护环境。请使用至少两个状语从句和两个宾语从句。
开头：提出问题（环境污染/资源浪费的现状）
主体1：解释3R原则（Reduce / Reuse / Recycle）
主体2：提出具体行动建议
结尾：号召大家行动起来
Dear classmates, I believe that our school can become greener. Every day we produce a lot of waste, and much of it goes to landfills. That's why we need the 3Rs. First, reduce what you use. When you buy fewer things, you create less waste. Second, reuse items if possible. For example, you can use old jars for storage instead of throwing them away. Third, recycle as much as you can. I'm sure that if everyone recycles paper, glass, and metal, we can make a big difference. So that we can protect the Earth, let's start practicing the 3Rs today!
亲爱的同学们，我相信我们的学校可以变得更绿色。每天我们产生大量垃圾，其中很多进入了垃圾填埋场。这就是为什么我们需要3R原则。首先，减少你的使用量。当你少买东西时，产生的垃圾就更少。第二，如有可能重复使用物品。例如，你可以用旧罐子储存东西，而不是扔掉它们。第三，尽可能多地回收。我相信如果每个人都回收纸张、玻璃和金属，我们就能带来很大的改变。为了保护地球，让我们从今天开始实践3R原则！
