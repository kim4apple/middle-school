第6章 质量与密度

📖 学习目标

① 知道质量的概念、单位及单位换算 ② 会正确使用托盘天平测量质量 ③ 理解密度的概念、公式和单位 ④ 掌握密度公式 \(\rho = m/V\) 的应用与计算 ⑤ 学会用天平和量筒测量物质的密度 ⑥ 能运用密度知识鉴别物质、判断空心实心

🔗 知识衔接 小学学过"物体含有物质的多少叫质量"，但那时没有严格定义。本章将给出质量的严格定义，并引入"单位体积的质量——密度"这一全新概念。密度是物质的一种特性，它将质量和体积联系起来，为我们认识物质世界提供了新的视角。上一章我们学习了透镜成像，本章将体会"比值定义法"在物理学中的广泛应用。
💡 思维方法：比值定义法 物理学中常把两个物理量的比值定义为一个新物理量，以便揭示物质的内在属性。本章的密度 \(\rho = m/V\) 就是用物体的质量与体积的比值来定义密度——它反映了物质在"单位体积"内所含质量的多少，是物质本身的特性，与物体的质量和体积无关。
🧠 思维导图
  • 质量与密度
    • 质量 \(m\)
      • 定义 物体所含物质的多少
      • 属性 不随形状、状态、位置、温度改变
      • 单位 kg（主）、g、mg、t
      • 测量 托盘天平（左物右码）、电子秤
    • 密度 \(\rho\)
      • 定义 单位体积的质量 \(\rho = m/V\)
      • 特性 物质固有属性，与 m、V 无关
      • 单位 kg/m³（主）、g/cm³
      • 换算 1 g/cm³ = 1000 kg/m³
    • 密度应用
      • 鉴别物质 测 \(\rho\) 查密度表
      • 空心实心判断 比较 \(\rho\) 或 m 或 V
      • 计算质量 \(m = \rho V\)（不便直接称量）
      • 计算体积 \(V = m/\rho\)（不规则物体）
    • 测量密度
      • 规则固体 天平 + 刻度尺（计算体积）
      • 不规则固体 天平 + 量筒（排水法）
      • 液体 天平 + 量筒（差值法）
📦 核心概念
🔍 先行组织者：棉花和铁哪个重？通常人们说铁重——但一卡车的棉花比一个铁钉重多了！所以不能说"铁比棉花重"，应该说"铁的密度比棉花大"。密度=质量÷体积（ρ=m/V），它描述的是单位体积里包含多少物质。水的密度是 1g/cm³（最方便的记忆点）。密度是鉴别物质的重要手段——纯金和镀金的密度不同，一测便知。
⚖️ 质量

物体所含物质的多少叫质量，用 \(m\) 表示。
质量是物体的属性——不随形状、状态、位置和温度的改变而变化。
一杯水凝固成冰，质量不变；从地球到月球，质量也不变。

🥉 青铜级
⚖️ 质量单位

国际单位：千克（kg）
换算：\(1\,\text{t} = 10^3\,\text{kg}\)，\(1\,\text{kg} = 10^3\,\text{g}\)，\(1\,\text{g} = 10^3\,\text{mg}\)
常见质量：一个鸡蛋约 50 g，中学生约 50 kg，一头大象约 5 t

🥉 青铜级
⚖️ 托盘天平

结构：横梁、指针、分度盘、托盘、平衡螺母、游码、标尺
使用步骤：放→调→测→读
① 放水平 ② 游码归零 ③ 调平衡螺母使指针指中 ④ 左物右码 ⑤ 砝码质量+游码读数

🥈 白银级
📊 密度

单位体积内所含物质的多少——密度 \(\rho\)
\(\rho = \dfrac{m}{V}\)（质量与体积的比值）
密度是物质的特性，与质量、体积无关——同种物质密度相同，不同物质密度一般不同。

🥇 黄金级
📊 密度单位换算

国际单位：\(\text{kg/m}^3\)   常用单位：\(\text{g/cm}^3\)
\(1\,\text{g/cm}^3 = 1000\,\text{kg/m}^3\)
水的密度：\(\rho_{\text{水}} = 1.0 \times 10^3\,\text{kg/m}^3 = 1\,\text{g/cm}^3\)

🥈 白银级
🧪 排水法测体积

不规则固体放入量筒中，利用排水法测体积：
\(V = V_2 - V_1\)（放入后的读数减去放入前的读数）
注意量筒读数时视线应与凹液面最低处相平。

🥈 白银级
🕳️ 空心与实心

判断方法（三种）：
① 比较密度：\(\rho_{\text{算}} < \rho_{\text{实}}\) → 空心
② 比较质量：\(m_{\text{算}} > m_{\text{实}}\) → 空心
③ 比较体积：\(V_{\text{算}} > V_{\text{实}}\) → 空心

🥇 黄金级
🔬 鉴别物质

测出待测物的密度 \(\rho_{\text{测}}\)，查阅密度表，找到与之最接近的标准物质。
例如：测得某金属块 \(\rho = 2.7\,\text{g/cm}^3\) → 可能是铝（铝的密度 \(2.7\,\text{g/cm}^3\)）

🥈 白银级
📐 公式速查
密度公式   \(\displaystyle \rho = \frac{m}{V}\)     \(m = \rho V\)     \(V = \frac{m}{\rho}\) 🥇 黄金级
单位换算   \(1\,\text{g/cm}^3 = 10^3\,\text{kg/m}^3\)
水的密度 \(\rho_{\text{水}} = 1.0 \times 10^3\,\text{kg/m}^3 = 1\,\text{g/cm}^3\)
铁的密度 \(\rho_{\text{铁}} = 7.9 \times 10^3\,\text{kg/m}^3 = 7.9\,\text{g/cm}^3\) 🥈 白银级
常用密度（单位 \(\times 10^3\,\text{kg/m}^3\)）
水 1.0   酒精 0.8   水银 13.6   空气（标准）0.00129
铁 7.9   铜 8.9   铝 2.7   金 19.3   银 10.5   冰 0.9 🥉 青铜级
质量单位换算   \(1\,\text{kg} = 10^3\,\text{g} = 10^6\,\text{mg}\)   \(1\,\text{t} = 10^3\,\text{kg}\) 🥉 青铜级
🔬 实验探究

🧪 测量物质的密度

1 准备器材：托盘天平（含砝码）、量筒、待测固体（小石块）、待测液体（盐水）、细线、烧杯
2 测固体密度：① 用天平测出石块质量 \(m\) ② 量筒中倒入适量水，读数 \(V_1\) ③ 用细线系住石块，缓慢浸入水中，读数 \(V_2\) ④ \(V = V_2 - V_1\)，\(\rho = m/V\)
3 测液体密度（差值法）：① 用天平测出烧杯和盐水的总质量 \(m_1\) ② 将部分盐水倒入量筒，读数 \(V\) ③ 测出剩余烧杯和盐水质量 \(m_2\) ④ \(m = m_1 - m_2\)，\(\rho = m/V\)
📌 实验原理：\(\rho = m/V\)——用天平测质量，用量筒（或刻度尺）测体积，通过比值求得密度。固体用排水法测体积，液体用差值法减小误差。
⚠️ 注意事项：① 天平使用前要调平（游码归零、调平衡螺母） ② 量筒读数视线与凹液面最低处相平 ③ 排水法测固体体积时固体要完全浸没 ④ 测液体密度时，先测总质量再倒出部分测剩余质量（避免倒不干净带来的误差）
✏️ 解题技巧

1 天平读数与误差分析

方法：砝码总质量 + 游码左侧对应刻度值
右盘砝码 50g + 20g + 10g = 80g，游码在标尺上对准 2.4g 刻度，则物体质量 \(m = 80 + 2.4 = 82.4\,\text{g}\)。
⚠️ 常见错误：① 游码读成右侧值 ② 忘了游码归零就开始调平衡 ③ 物码放反（左码右物）时正确读数 = 砝码 - 游码
❌ 错误：物码放反时，仍用"砝码质量 + 游码读数"计算物体质量
🔍 诊断：天平平衡条件为 左盘质量 = 右盘质量 + 游码读数。正常（左物右码）时：m物 = m砝 + m游；若放反（左码右物）：m砝 = m物 + m游 → m物 = m砝 - m游。不是简单的相加！
💡 原理：天平是等臂杠杆，左盘质量 = 右盘质量 + 游码示数
🌱 基础变式：托盘天平调平后，误将物体放在右盘、砝码（50 g + 20 g = 70 g）放在左盘，游码读数为 3.2 g。求物体的实际质量。——m物 = m砝 - m游 = 70 - 3.2 = 66.8 g。
📌 中考常考：给出天平读数和砝码组合求物体质量，或判断物码放反时的正确质量

2 密度计算——比例法

方法：同种物质 \(\rho\) 不变 → \(m_1/m_2 = V_1/V_2\)
已知铝的密度 \(2.7\,\text{g/cm}^3\)，一块铝质量为 54 g，体积为 20 cm³。另一块同种铝质量为 81 g，求其体积。
\(\because \rho\) 相同 \(\therefore m_1/m_2 = V_1/V_2\)
\(V_2 = V_1 \times m_2/m_1 = 20 \times 81/54 = 30\,\text{cm}^3\)
⚠️ 注意单位统一——通常体积用 cm³，质量用 g，密度单位就是 g/cm³。若用 kg/m³，体积需用 m³。
❌ 错误：质量 54 g、体积 20 cm³，计算密度时用 ρ = 54/20 = 2.7，单位直接写成 kg/m³
🔍 诊断：g/cm³ 和 kg/m³ 相差 1000 倍！54 g / 20 cm³ = 2.7 g/cm³ = 2.7 × 10³ kg/m³。如果混淆两个单位，计算结果会差 1000 倍。解题时先统一单位：质量用 g→体积用 cm³→密度用 g/cm³。
💡 原理：密度是物质特性，同种物质密度相同，质量与体积成正比
🌱 基础变式：一块铝锭体积为 100 cm³，质量为 270 g；另一块同种铝体积为 30 cm³，求其质量。——m = 270 × (30/100) = 81 g（同种物质质量与体积成正比）。
📌 中考常考：已知部分数据，通过密度不变求另一物体的质量或体积

3 空心实心判断

方法：三种比较法——比较密度、比较质量、比较体积
一个铜球质量 178 g，体积 30 cm³，判断是否空心。（\(\rho_{\text{铜}} = 8.9\,\text{g/cm}^3\)）
法一——比密度：\(\rho_{\text{球}} = 178/30 \approx 5.93 < 8.9\) → 空心
法二——比质量：若实心则 \(m = 8.9 \times 30 = 267 > 178\) → 空心
法三——比体积：若实心则 \(V = 178/8.9 = 20 < 30\) → 空心，空心部分体积 \(V_{\text{空}} = 30 - 20 = 10\,\text{cm}^3\)
⚠️ 三种方法任选一种即可判断，法三还能求出空心部分的体积。最常见错误是未统一单位或忘记换算。
❌ 错误：直接用 m总/V总 算出"密度"，然后与标准密度比较判断空心
🔍 诊断：m总/V总 算出的是球的"平均密度"，ρ平均 < ρ标准确实说明球是空心的。但很多学生混淆了这个关系——法一（比密度）就是 ρ平均 vs ρ标准，ρ平均小则空心；法二（比质量）则需先假设实心算出理论质量；法三（比体积）先算实心体积再与外观体积比较。
💡 原理：\(\rho = m/V\) 的变形应用，通过实测值与理论值的比较判断
🌱 基础变式：一个铝球质量为 54 g，外观体积为 25 cm³。ρ(铝) = 2.7 g/cm³，判断是否空心。——V实 = 54/2.7 = 20 cm³ < 25 cm³，空心，V空 = 25 - 20 = 5 cm³。
📌 中考常考：给出总质量和外观体积，判断空心并计算空心部分体积或注水后的总质量

2.
一个质量为 158 g 的铁球，体积为 30 cm³。（\(\rho_{\text{铁}} = 7.9 \times 10^3\,\text{kg/m}^3\)）
① 判断该铁球是实心还是空心？
② 若空心，求空心部分的体积。
③ 若在空心部分注满水，求注水后球的总质量。
答案：① \(\rho_{\text{铁}} = 7.9\,\text{g/cm}^3\)（换算后）。\(\rho_{\text{球}} = 158/30 \approx 5.27 < 7.9\) → 空心。
② 实心部分体积 \(V_{\text{实}} = m/\rho = 158/7.9 = 20\,\text{cm}^3\)，\(V_{\text{空}} = 30 - 20 = 10\,\text{cm}^3\)。
③ \(m_{\text{水}} = \rho_{\text{水}} V_{\text{空}} = 1 \times 10 = 10\,\text{g}\)，总质量 \(158 + 10 = 168\,\text{g}\)。

3.
为鉴别一块金属片的材质，小华用天平和量筒测量其密度。测得金属片质量为 54 g，在量筒中放入 20 mL 水，金属片浸没后水面上升到 40 mL。请计算该金属片的密度，并判断它可能是什么金属。
答案：金属片体积 \(V = 40 - 20 = 20\,\text{cm}^3\)，密度 \(\rho = 54/20 = 2.7\,\text{g/cm}^3 = 2.7 \times 10^3\,\text{kg/m}^3\)。查密度表可知该密度与铝的密度（\(2.7 \times 10^3\,\text{kg/m}^3\)）吻合，该金属片可能是铝。

4.
某品牌调和油的瓶上标有"净含量 1.8 L"字样，测得瓶和油的总质量为 1.68 kg，瓶的质量为 0.12 kg。求：① 调和油的密度；② 若用该空瓶装水，最多能装多少 kg？
答案：① 油的质量 \(m_{\text{油}} = 1.68 - 0.12 = 1.56\,\text{kg}\)，体积 \(V = 1.8\,\text{L} = 1.8 \times 10^{-3}\,\text{m}^3\)，\(\rho_{\text{油}} = m/V = 1.56/(1.8 \times 10^{-3}) \approx 867\,\text{kg/m}^3\)。
② 瓶容积 \(V_{\text{瓶}} = 1.8\,\text{L} = 1.8 \times 10^{-3}\,\text{m}^3\)，\(m_{\text{水}} = \rho_{\text{水}} V = 1.0 \times 10^3 \times 1.8 \times 10^{-3} = 1.8\,\text{kg}\)。
