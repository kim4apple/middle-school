🔄 四、化学反应类型判断指南

🧩 化合反应
A + B + … → C（多变一）
例：$C + O_2 \xrightarrow{点燃} CO_2$
判断要点：反应物≥2种，生成物只有1种
💥 分解反应
A → B + C + …（一变多）
例：$2KMnO_4 \xrightarrow{\Delta} K_2MnO_4 + MnO_2 + O_2\uparrow$
判断要点：反应物只有1种，生成物≥2种
⚡ 置换反应
A + BC → AC + B（单质→单质）
例：$Fe + CuSO_4 \rightarrow FeSO_4 + Cu$
判断要点：一种单质 + 一种化合物 → 另一种单质 + 另一种化合物
🔁 复分解反应
AB + CD → AD + CB（交换成分）
例：$HCl + NaOH \rightarrow NaCl + H_2O$
判断要点：两种化合物交换成分，价态不变
🎯 常见误判
• CO + CuO → Cu + CO₂ —— 不是置换！反应物都是化合物，没有单质参与
• CH₄ + 2O₂ → CO₂ + 2H₂O —— 不属于四大基本类型（氧化反应）
• 中和反应（酸+碱→盐+水）—— 属于复分解反应的特殊形式
• 2CO + O₂ → 2CO₂ —— 是化合反应（两种→一种），尽管反应物都是化合物
• 复分解反应的条件：生成物中必须有 气体↑ 或 沉淀↓ 或 水，三者至少具备其一

🔩 七、金属活动性顺序完整表格

活动性由强 → 弱 中考常考范围
K Ca Na Mg Al → Zn Fe Sn Pb → (H) → Cu Hg Ag Pt Au
强 中等 氢（分界线） 弱
口诀： 嫁给那美女，身体细纤轻，统共一百斤
🔄 前置置换
排在前面的金属能把后面的金属从它的盐溶液中置换出来。
⚠️ K、Ca、Na 除外 —— 它们太活泼，先与水反应。
💨 与酸反应
排在 H 前面的金属能置换出酸中的氢（生成 H₂）。
排在 H 后面的金属不与稀盐酸 / 稀硫酸反应。

🧂 八、酸碱盐溶解性速查表

📝 溶解性口诀
钾钠铵硝皆可溶 —— K⁺ / Na⁺ / NH₄⁺ / NO₃⁻ 盐全溶
盐酸盐不溶银亚汞 —— AgCl / Hg₂Cl₂ 不溶，其余溶
硫酸盐不溶钡和铅 —— BaSO₄ / PbSO₄ 不溶，CaSO₄ / Ag₂SO₄ 微溶
碳酸盐只溶钾钠铵 —— K₂CO₃ / Na₂CO₃ / (NH₄)₂CO₃ 溶，MgCO₃ 微溶
碱类只溶钾钠铵钡 —— KOH / NaOH / NH₃·H₂O / Ba(OH)₂ 溶，Ca(OH)₂ 微溶
🥇 必记"不溶"沉淀
• 白色：AgCl、BaSO₄、CaCO₃、BaCO₃、Mg(OH)₂、Al(OH)₃
• 蓝色：Cu(OH)₂
• 红褐色：Fe(OH)₃
• AgCl和BaSO₄不溶于稀硝酸 —— 用于Cl⁻/SO₄²⁻检验
🎯 复分解应用
• 判断反应能否发生：看生成物是否有 气体↑ / 沉淀↓ / 水
• 例：NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄（生成蓝色沉淀）
• 例：NaCl + KNO₃ → 不反应（无气体/沉淀/水生成）
⚠️ 中考易错：Ca(OH)₂ 微溶于水 —— 澄清石灰水浓度很低，属于饱和溶液但并不浓。Ag₂SO₄ / CaSO₄ 微溶，离子检验时注意干扰。

🧮 九、溶液配制与稀释计算

📐 核心公式
ω = m质 / m液 × 100%
m液 = m质 + m剂
变形：m质 = m液 × ω  |  m液 = m质 ÷ ω
💧 稀释公式
m₁ω₁ = m₂ω₂
加水质量 = m₂ − m₁ (或 V₂ − V₁)
稀释前后溶质质量不变
🔬 配制步骤
① 计算 → m质 = m液 × ω   ② 称量（固体用天平，液体用量筒） → ③ 溶解（玻璃棒搅拌） → ④ 装瓶贴标签（写上溶液名称和浓度）
若为液体稀释：计算所需浓溶液体积 → 量取 → 加水稀释 → 装瓶
📌 浓缩/增浓
加溶质：m质' = m质 + Δm, m液' = m液 + Δm
蒸发溶剂：m剂' = m剂 − Δm, m液' = m液 − Δm
以上两种方法溶质质量均增大
📊 混合计算
m₁ω₁ + m₂ω₂ = (m₁ + m₂)ω
已知任意三个量可求第四个
⚠️ 中考易错：
• 溶质质量分数不是体积分数 —— 用体积×密度换算质量后再计算
• 溶解度不是饱和溶液浓度 —— 可自主换算：ω = S / (100+S) × 100%
• 气体溶质（如HCl气体）溶于水，计算时溶质质量 = HCl气体质量，不是HCl溶液质量

💎 十一、常见元素化合价总表

🥇 固定化合价
• +1 价：K、Na、Ag、H
• +2 价：Ca、Mg、Ba、Zn
• +3 价：Al
• +4 价：Si
• -1 价：F、Cl（在氯化物中）
• -2 价：O（除过氧化物H₂O₂中为-1）
🔄 常见变价
• Fe：+2（亚铁——浅绿色） / +3（铁——黄色）
• Cu：+1（亚铜） / +2（铜——蓝色）
• S：-2（H₂S）/ +4（SO₂）/ +6（H₂SO₄）
• N：-3（NH₃）/ +2（NO）/ +4（NO₂）/ +5（HNO₃）
• Cl：-1（HCl）/ +1（NaClO）/ +5（KClO₃）/ +7（KClO₄）
• Mn：+2（Mn²⁺）/ +4（MnO₂）/ +6（K₂MnO₄）/ +7（KMnO₄）
🎵 化合价口诀
一价钾钠银氢，二价钙镁钡锌，
三铝四硅五价磷，二三铁、二四碳，
二四六硫都齐全，铜汞二价最常见，
氧是-2 氯是-1，单质零价永不变。
⚠️ 中考易错：
• 化合价标在元素符号正上方（如 $\stackrel{+2}{Fe}$），离子电荷标在右上角（如 Fe²⁺），二者不同！
• 原子团是一个整体，化合价为整团所带电荷数（如 SO₄²⁻ 整体 -2，不是每个原子）
• 单质中元素化合价为 0（如 H₂、Fe、O₂ 中均为 0）

方法适用情况举例
过滤分离不溶性固体和液体粗盐提纯除去泥沙；分离MnO₂（催化剂）和H₂O₂反应后的溶液
蒸发结晶溶解度受温度影响不大的物质海水晒盐（NaCl）
降温结晶溶解度受温度影响较大的物质从KNO₃和NaCl混合溶液中分离KNO₃
蒸馏分离沸点不同的液态混合物制取蒸馏水；分离酒精和水

🔍 十三、常见物质鉴别流程图

🧪 鉴别的一般思路
Step 1 观察颜色/状态/气味 → Step 2 加水观察溶解性 → Step 3 加指示剂/酸/盐溶液 → Step 4 根据现象下结论
🧪 鉴别 O₂ / CO₂ / CO / H₂
O₂：带火星木条 → 复燃
CO₂：燃着木条 → 熄灭；通入石灰水 → 变浑浊
CO：点燃 → 蓝色火焰；燃烧产物使石灰水变浑浊
H₂：点燃 → 淡蓝色火焰；燃烧产物只有水（干冷烧杯壁有水珠）
注：CO 和 H₂ 都燃烧呈蓝色，靠检验燃烧产物区分
🧂 鉴别 酸 / 碱 / 盐
酸：石蕊→红 / pH<7 / 加Zn有气泡(H₂)
碱：石蕊→蓝 酚酞→红 / pH>7
盐：无统一特性，利用特定离子反应检验
• Cl⁻：AgNO₃ + 稀HNO₃ → 白色沉淀
• SO₄²⁻：BaCl₂ + 稀HNO₃ → 白色沉淀
• CO₃²⁻：加稀HCl → 气体通入石灰水 → 变浑
• NH₄⁺：加NaOH加热 → 湿润红色石蕊试纸→蓝
🌈 常见白色粉末鉴别
① 观察颜色：白色 → 可能 NaCl / Na₂CO₃ / CaCO₃ / NaOH / NH₄Cl
② 加水：不溶 → CaCO₃（白色沉淀）
③ 溶解放热 → NaOH（NaOH溶于水放热）
④ 加酚酞：变红 → Na₂CO₃（碱性盐）/ NaOH（碱）
⑤ 加稀HCl：气泡 → Na₂CO₃（CO₃²⁻）
⑥ 加NaOH加热 + 试纸：试纸变蓝 → NH₄Cl（铵盐）
🎯 检验"三兄弟"
鉴别盐酸、NaOH、NaCl：
取样 → 各滴入石蕊试液：
• 变红 → 盐酸（酸）
• 变蓝 → NaOH（碱）
• 不变色 → NaCl（中性盐）

鉴别Na₂CO₃、NaOH、NaCl：
取样 → 各滴入稀盐酸：
• 有气泡 → Na₂CO₃
• 无气泡 → 再滴酚酞：变红→NaOH，不变→NaCl

⚠️ 十四、高频易错陷阱总览

① 铁与酸/盐溶液 → 生成 Fe²⁺ 不是 Fe³⁺
铁与盐酸：$Fe + 2HCl \rightarrow FeCl_2 + H_2\uparrow$（浅绿色）
铁与硫酸铜：$Fe + CuSO_4 \rightarrow FeSO_4 + Cu$（浅绿色）
② 纯碱是盐不是碱
Na₂CO₃ 俗称纯碱，但化学分类属盐（金属离子+酸根离子）。水溶液显碱性 ≠ 它是碱。
③ 有机物 ≠ 含碳化合物
CO、CO₂、H₂CO₃、CaCO₃、Na₂CO₃、NaHCO₃ 虽含碳，但属于无机物。
④ 浓硫酸稀释：酸入水！
严禁水入酸。口诀：酸入水、沿内壁、慢倒入、不断搅拌。
⑤ 饱和溶液 ≠ 浓溶液
Ca(OH)₂ 饱和溶液很稀（溶解度小）。KNO₃ 不饱和溶液可以很浓（溶解度大）。
⑥ Ca(OH)₂ 溶解度随温度升高而减小
这是唯一的常考特例！大多数固体溶解度随温度升高而增大。
⑦ 中和反应一定生成盐和水，但生成盐和水的反应不一定是中和反应
$CO_2 + 2NaOH \rightarrow Na_2CO_3 + H_2O$ 不是中和反应（CO₂ 不是酸）。
⑧ 合金是混合物，硬度比纯金属大，熔点比纯金属低
生铁和钢都是铁合金（含碳量不同：生铁2%~4.3%，钢0.03%~2%）。
⑨ 铵态氮肥不能与碱性物质混用
$NH_4^+ + OH^- \rightarrow NH_3\uparrow + H_2O$，NH₃ 逸出 → 氮元素损失 → 肥效降低。
⑩ 铁生锈必须同时有水+氧气
铁锈（$Fe_2O_3 \cdot xH_2O$）疏松多孔，不能阻止内部铁继续生锈。
⑪ 溶解度"四要素"缺一不可
一定温度、100g溶剂、饱和状态、单位g。不能用"%"表示溶解度。
⑫ AgCl 和 BaSO₄ 不溶于稀硝酸
这是检验 Cl⁻ 和 SO₄²⁻ 时加稀硝酸的原因——排除其他沉淀（如 BaCO₃ 溶于酸会产生气泡）。

🧴 十五、CO₂与NaOH反应专题

💡 为什么是专题？ CO₂ + 2NaOH → Na₂CO₃ + H₂O 无明显现象，如何证明反应发生？——中考实验探究最爱考，没有之一。
🔬 方法一：气压变化法（最经典）
实验：收集一瓶CO₂，倒入NaOH溶液，迅速盖紧瓶盖，振荡。
现象：塑料瓶变瘪 / 气球膨胀 / U形管左高右低
原理：CO₂被NaOH吸收 → 瓶内气压↓ → 大气压压瘪瓶子
⚠️ 不足：CO₂也能溶于水，需 等量水做对照实验 证明是反应而非溶解
🔬 方法二：生成物检验法（最严谨）
实验：反应后的溶液中滴加 CaCl₂ 溶液（或 BaCl₂、稀 HCl）
现象：产生白色沉淀（CaCO₃）
方程式：Na₂CO₃ + CaCl₂ → CaCO₃↓ + 2NaCl
✅ 对照：NaOH + CaCl₂ → 不反应（无沉淀）—— 排除干扰
🔬 方法三：pH变化法
实验：向NaOH溶液中通入CO₂，用pH计监测
现象：pH由碱性（>7）逐渐降低
原理：NaOH（强碱）→ Na₂CO₃（碱性但弱于NaOH）→ 最终可能 H₂CO₃（酸性）
🔬 方法四：指示剂法
实验：向NaOH溶液（含酚酞）通入CO₂
现象：红色逐渐变浅至无色
❌ 但不足：CO₂也能使碱的红色褪去（酸碱中和），无法单独证明 CO₂ 与 NaOH 发生了非中和反应以外的作用
⚖️ CO₂与Ca(OH)₂ vs CO₂与NaOH
CO₂ + Ca(OH)₂
现象：石灰水变浑浊（白色沉淀）
产物：CaCO₃↓ + H₂O
应用：CO₂ 的检验（唯一方法）
CO₂ + NaOH
现象：无明显现象
产物：Na₂CO₃ + H₂O
应用：CO₂ 的吸收/除杂（NaOH溶液吸收CO₂能力更强）
⚠️ 中考必记：
• 没有明显现象 ≠ 没有发生反应
• 气压变化类实验 必须设对照（等量水代替NaOH）才能排除CO₂溶于水的干扰
• CO₂通入NaOH溶液后先无后有气泡：先 NaOH→Na₂CO₃，过量CO₂→Na₂CO₃+CO₂+H₂O→2NaHCO₃，此时加酸先无气体（NaOH+HCl），后有气体（NaHCO₃+HCl）—— 这是中考推断题"先后反应"逻辑

✍️ 十六、中考化学答题规范

1. 化学用语规范：

• 元素符号：首字母大写，第二个字母小写（如 Co 是钴，CO 是一氧化碳——大小写决定含义）

• 化学式：下标数字不可写成正常大小（H₂O 不是 H2O）

• 化学方程式：必须配平 + 注明条件（点燃/加热△/高温/催化剂）+ 标↑↓

• 条件标注：加热用"△"（不可省略），催化剂写上方，"点燃"≠ "燃烧"（"点燃"是条件，"燃烧"是现象描述）

2. 实验题规范：

• 现象描述 = 看到/听到/闻到的，不能出现生成物名称作为现象

• 正确：铁钉表面出现红色固体，溶液由蓝色逐渐变为浅绿色

• 错误：铁钉表面生成了铜，溶液变为硫酸亚铁溶液（这是结论不是现象）

3. 计算题规范：

• 必须有"解：设...为x" → 写方程式 → 找比例关系 → 列比例式 → 求解 → "答："

• 计算结果保留要求（通常保留到小数点后1位或整数）

• 单位必须标注且前后一致

4. 推断题规范：

• 结论要按题目要求书写（化学式、化学方程式还是名称？读清楚！）

• 常见"题眼"：颜色（蓝色=Cu²⁺，浅绿=Fe²⁺，黄色=Fe³⁺，红褐色沉淀=Fe(OH)₃）

5. 选择题规范：

• "正确"vs"不正确"——圈出"不"字！

• "一定"vs"可能"——过于绝对的说法常是错的

• 单选题——最符合题意的一项（可能有部分正确的干扰项）

🔗 十九、跨章节综合专题

🔄 专题一：碳的完整旅程（C → CO → CO₂ → CaCO₃）
涉及章节：ch06（碳三角）+ ch10（CO₂与碱反应）+ ch11（CaCO₃与酸反应）

转化链：
① C（不足O₂）→ CO   2C + O₂ → 2CO（ch06）
② CO → CO₂   2CO + O₂ → 2CO₂（ch06）
③ CO₂ → CaCO₃   CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O（ch06/ch10）
④ CaCO₃ → CO₂   CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑（ch06/ch11）
⑤ CO₂ → CO   CO₂ + C → 2CO（ch06·高温）

中考命题方向：推断题中"碳三角"是最常见的循环之一，通常以CO₂为起点双向转化。
🧪 专题二：物质检验综合流程
涉及章节：ch02（O₂/CO₂检验）+ ch08（金属检验）+ ch10（酸碱指示剂）+ ch11（离子检验）

常见气体检验：
O₂：带火星木条复燃  |  CO₂：澄清石灰水变浑浊  |  H₂：点燃淡蓝色火焰  |  CO：点燃蓝色火焰，产物使石灰水变浑浊

常见离子检验：
Cl⁻：AgNO₃ + 稀HNO₃ → 白色沉淀  |  SO₄²⁻：BaCl₂ + 稀HNO₃ → 白色沉淀
CO₃²⁻：稀HCl → 气体通石灰水 → 变浑浊  |  NH₄⁺：NaOH → NH₃↑→湿润红石蕊试纸变蓝

中考命题方向：综合实验题中"鉴别三种白色固体/无色溶液"——要逐一取样，根据现象差异区分。
⚖️ 专题三：无现象反应的证明思路
涉及章节：ch10（中和反应）+ ch11（CO₂与NaOH）

核心思路：无明显现象 ≠ 没反应。通过反应后体系的气压变化或指示剂变色或生成物检验来间接证明。

经典案例：CO₂与NaOH反应——塑料瓶变瘪实验（压强减小）
