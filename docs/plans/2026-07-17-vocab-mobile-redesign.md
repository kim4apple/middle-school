# 单词专项 · 移动端界面重设计

## 概述

对英语8/9年级 `appendix-vocab.html` 的 5 个学习模块进行移动端深度 UI/UX 优化。

## 设计方向

**方案 B — Mini App 全屏沉浸式**，底部 Tab 导航 + 左右滑动手势切换模块。

## 整体骨架

```
┌──────────────────────────┐
│  ← 返回    单词专项 · 八年级│ ← 顶部栏 48px
├──────────────────────────┤
│                          │
│    [当前模块内容区]        │ ← 可滚动/可滑动
│    填满剩余空间            │
│                          │
├──────────────────────────┤
│ 🏠  📋  🎯  📊          │ ← 底部 Tab Bar 56px
│ 主页  单词  学习  统计     │    活跃 tab + 指示线
└──────────────────────────┘
```

### 核心改动：5 Tab → 4 Tab

- 合并 🃏 自测卡 + ✍️ 听写 + 📝 练习 → 统一为「🎯 学习」
- 新增「🏠 主页」Dashboard
- 保留「📋 单词表」和「📊 统计」

## 页面设计

### 1. 🏠 主页

```
◀ 返回     单词专项 · 八年级
─────────────────────────────
  👋 下午好！今天已学 12 词

  🔥 连续学习 3 天

  📅 待复习 5 词 → 立即开始

  📊 总进度 156/424 (37%)

  ┌────────────────────────┐
  │  快速开始：             │
  │  [继续上次]  [随机10词] │
  └────────────────────────┘
```

### 2. 📋 单词表

- 卡片布局（手机）/ 表格布局（桌面）
- 信息层级：单词 17px 加粗 → 音标 12px 灰 → 释义 15px → 例句 13px
- 复选框在卡片右上角，绿色表示已掌握
- 搜索框在筛选栏下方
- 筛选：全部/未掌握/已掌握/错词

### 3. 🎯 学习

进入后选择模式：
```
┌────────────────────────┐
│  🃏 自测卡              │
│  ✍️ 听写                │
│  📝 选择题练习          │
└────────────────────────┘
```

#### 🃏 自测卡
- 正面：单词 36px + 音标 16px
- 翻面（点击/手势）：释义 24px + 词性 + 例句
- 三颗按钮固定于卡片底部：😊简单（绿）/🤔较难（黄）/😵忘了（红）
- 顶部进度条，左右滑动切换
- 自动朗读（可选开关）

#### ✍️ 听写
- 中文提示固定在输入框上方（键盘弹出时也不消失）
- 自动朗读 2 遍，可「再听一次」
- 答对：绿色动画 + 1.5s 自动继续
- 答错：显示正确拼写 + 用户输入对比 + 记忆技巧
- 键盘弹出时隐藏底部 Tab Bar
- 使用 `visualViewport` API 适配键盘高度

#### 📝 选择题练习
- 显示英文单词 → 选择正确中文释义（4 选项）
- 答完显示正确率
- 错题自动加入「错词」列表

### 4. 📊 统计

- 学习概览：今日学习数 / 连续天数 / 总进度
- 四卡片：总词 / 已掌握 / 错词 / 掌握率
- SRS 到期复习列表
- 里程碑成就系统

## 交互规范

### 手势
- 左右滑动切换 Tab（跟随手指 + 弹性动画）
- 自测卡左右滑动切换单词
- 下拉刷新（统计页）

### 动画
- Tab 切换：交叉淡入 + 指示线滑动
- 勾选完成：弹性缩放
- 翻卡：3D `rotateY` + `preserve-3d`
- 听写答对：绿色对勾弹跳
- 答错：输入框抖动

### 状态管理
- 筛选条件跨模块共享
- 单元选择跨模块共享
- 最后浏览位置自动保存

## 视觉规范

### 间距系统
`4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48` (8px grid)

### 字体层级
| 层级 | 用途 | 大小 | 字重 |
|------|------|------|------|
| H1 | 页面标题 | 22px | 800 |
| H2 | 模块标题 | 18px | 700 |
| H3 | 卡片标题 | 16px | 700 |
| body | 正文 | 14px | 400 |
| cap | 辅助文字 | 12px | 400 |

### 色彩语义
- 简单/已掌握：绿色 `#16a34a`
- 较难/待复习：黄色 `#ca8a04`
- 忘了/错词：红色 `#dc2626`
- 进度条：分段渐变（绿→黄→红）

## 平台差异

### iOS
- `env(safe-area-inset-bottom)` Tab Bar safe area
- `visualViewport` API 监听键盘
- 输入框 `font-size: 16px` 防自动缩放
- `prefers-reduced-motion` 关闭复杂动画

### Android
- 硬件返回键：返回首页
- `visualViewport` API + `scrollIntoView`
- 触感反馈
