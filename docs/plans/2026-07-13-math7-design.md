# 七年级数学完全总结 · 设计文档

**日期**: 2026-07-13
**目标用户**: 学生自学
**教材版本**: 人教版七年级数学（上册 + 下册）

---

## 一、项目概述

构建一个单文件 HTML 页面，完整涵盖七年级数学上下两册的全部知识点，包括思维导图、核心概念、公式速查和解题技巧四大模块，帮助学生快速掌握知识体系结构和关键解题思路。

## 二、技术方案

| 项目 | 方案 |
|------|------|
| **交付物** | 单文件 `math7.html`，CSS/JS 内嵌 |
| **公式渲染** | Katex 本地文件（`katex.min.css` + `katex.min.js` + `auto-render.min.js`），相对路径引用 |
| **离线支持** | 完全离线可用 |
| **暗色模式** | CSS 自定义属性 + JS 切换 `data-theme`，`localStorage` 持久化 |
| **折叠/展开** | 原生 `<details>` + `<summary>` |
| **导航高亮** | IntersectionObserver 自动高亮当前章节 |
| **响应式** | 移动端侧边栏收起到顶部汉堡菜单 |
| **打印** | `@media print` 隐藏导航，全宽展开 |

## 三、内容结构

共 11 章（第 0 章为总览），每章包含 4 个模块。

### 第 0 章 · 完整总结
- 全年知识点思维导图（总图）
- 核心公式一览表
- 解题技巧分类索引

### 第 1 章 · 有理数
正负数、数轴、相反数、绝对值、有理数加减乘除乘方、科学记数法

### 第 2 章 · 整式的加减
单项式/多项式、合并同类项、去括号法则、整式加减

### 第 3 章 · 一元一次方程
方程概念、等式的性质、解一元一次方程、实际应用（行程/工程/利润）

### 第 4 章 · 几何图形初步
立体与平面图形、直线射线线段、角的概念与度量、余角补角

### 第 5 章 · 相交线与平行线
对顶角/邻补角、垂线、平行线判定与性质、命题定理

### 第 6 章 · 实数
平方根/立方根、无理数、实数分类与运算

### 第 7 章 · 平面直角坐标系
坐标概念、点的平移、坐标方法应用

### 第 8 章 · 二元一次方程组
代入法/加减法、实际应用、三元一次方程组*

### 第 9 章 · 不等式与不等式组
不等式性质、解不等式、数轴表示、不等式组

### 第 10 章 · 数据的收集整理
全面调查/抽样调查、统计图表、频数分布直方图

## 五、视觉风格 · 现代极简

| 元素 | 设计 |
|------|------|
| **基调** | 干净留白，信息层级清晰 |
| **底色** | `#ffffff`，深色模式 `#0f172a` |
| **文字色** | `#1a1a2e`，深色模式 `#e2e8f0` |
| **强调色** | 靛蓝 `#4f46e5`，深色模式 `#818cf8` |
| **辅助底色** | `#f1f5f9` |
| **字体** | 全站 `-apple-system, sans-serif`，无衬线 |
| **卡片** | 微阴影、8px 圆角、hover 阴影加深 |
| **公式** | 浅灰底卡片 + 左侧靛蓝 3px 竖条 |
| **导航** | 侧边栏浅灰底，当前章节靛蓝高亮条 |
