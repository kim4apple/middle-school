第11章 · 三角形

🎯 学习目标

掌握三角形三边关系与内角和定理；理解三角形外角的性质；熟练运用多边形内角和与外角和公式

🔗 前置知识：七年级学过的线段、角、平行线的基本概念和性质。你已经知道角是由两条有公共端点的射线组成的图形，知道角的度量和分类（锐角<90°, 直角=90°, 钝角>90°）。这些是研究三角形的基础。
🧭 分类讨论：三角形按边分（等边/等腰/三边不等）和按角分（锐角/直角/钝角）。三边关系中"给定两边长求第三边范围"是典型的分类讨论+不等式综合题。本章的分类讨论能力直接为第12章全等三角形的多解问题铺路。

📖 核心概念
🔍 先行组织者：自行车架为什么是三角形的？（因为三角形最稳定——四边形会变形，三角形不会）。三角形是最基本的封闭多边形——任何多边形都可以分割成若干个三角形。本章学三个核心关系：三边关系（能不能构成）、内角关系（和=180°）、外角关系（=不相邻两内角和）。这三条是三角形世界的"宪法"。

按边分类
三角形的定义与分类

由不在同一直线上的三条线段首尾顺次相接组成的图形。按边分：不等边三角形（三边互不相等）、等腰三角形（两边相等）、等边三角形（三边相等）。按角分：锐角三角形（三个角都 < 90°）、直角三角形（一个角 = 90°）、钝角三角形（一个角 > 90°）。

🥉 青铜级
三角形三边关系

三角形两边的和大于第三边，两边的差小于第三边。即对于 $\triangle ABC$，有 $a + b > c$，$|a - b| < c$。给定两边长为 $a$ 和 $b$（$a \geq b$），第三边 $c$ 的取值范围：$a - b < c < a + b$。

🥈 白银级
三角形内角和定理

三角形三个内角的和等于 $180°$。即 $\angle A + \angle B + \angle C = 180°$。证明方法：过顶点作对边的平行线，利用平行线性质。

🥇 黄金级
三角形的外角

三角形的一边与另一边的延长线组成的角。性质：① 三角形的外角等于与它不相邻的两个内角之和；② 三角形的外角大于任何一个与它不相邻的内角。三角形的外角和 = $360°$。

🥈 白银级
多边形的内角和与外角和

n 边形内角和 = $(n - 2) \times 180°$。任意多边形的外角和永远等于 $360°$（与边数无关）。对角线总数 = $\frac{n(n-3)}{2}$ 条。

💡 解题技巧

⚡ 1 三边关系判断能否构成三角形

给三条线段长度，只需检查最短两边之和 > 最长边（必要条件也是充分条件）。不需要验三条不等式。

例题：判断长为 3cm, 4cm, 8cm 的三条线段能否构成三角形。
解：$3+4=7 < 8$，不能构成三角形。

⚠️ 易错：只需验一次（最短两边和 vs 最长边），不要用"两边差"去验

❌ 错误：判断 3, 4, 8 能否构成三角形时，只验了 $3+4=7<8$ 就写不能
🔍 诊断：这个例子验对了。但学生常见错误：判定 3, 4, 5 时只验了 $3+4>5$ 就认为可以——确实只需验一次（最短两边和 vs 最长边）即可，不需要三条不等式全部验证。

💡 原理：几何事实——两点之间线段最短。三边关系的本质：从A到C，直接走（边AC）比绕道B走（AB+BC）更短。所以 AC < AB+BC，同理可得另外两个不等式。

🌱 基础变式：判断长为 5cm, 5cm, 7cm 的三条线段能否构成三角形。
解：最短两边和 5+5=10 > 7，能构成。且是等腰三角形。

2 已知两边求第三边范围

给定 $a$ 和 $b$（设 $a > b$），第三边 $c$ 满足：$a-b < c < a+b$。注意是"<"不是"≤"。

例题：三角形两边分别为 5 和 8，求第三边的取值范围。
解：$8-5 < x < 8+5$，即 $3 < x < 13$。若第三边为整数，可能的值是 4,5,6,7,8,9,10,11,12。

⚠️ 易错：取值范围是 $a-b < x < a+b$，两边都是"<"不包含等号

❌ 错误：三角形两边为 5 和 8，学生写第三边范围为 $3 \le x \le 13$
🔍 诊断：三边关系是"大于"和"小于"，不是"大于等于"和"小于等于"。等号时三点共线，构不成三角形。把 $a-b < x < a+b$ 的 $<$ 记成 $\le$ 是高频错误。

💡 原理：$a-b < c < a+b$ 是两个不等式合并：$c < a+b$（两边之和>第三边）和 $a < b+c$ → $a-b < c$（两边之差<第三边）。给两边长求第三边范围，考的其实是两个不等式的交集。

🌿 综合变式：等腰三角形有两边长分别为 4 和 9，求周长。
解：需分类：①4为腰→底9，检验 4+4<9 不能构成；②9为腰→底4，检验 9+9>4 ✓。周长=9+9+4=22。注意：等腰三角形也要满足三边关系！

3 三角形内外角综合计算

两个关键关系搭桥：① 内角 + 外角 = 180°（邻补角）；② 外角 = 不相邻两内角之和。看到"外角"就想到把外角转化成内角。

例题：$\triangle ABC$ 中，$\angle B = 50°$，外角 $\angle ACD = 110°$，求 $\angle A$。
解：$\angle ACD = \angle A + \angle B$，$110° = \angle A + 50°$，$\angle A = 60°$。

⚠️ 易错：外角是三角形一边延长线与另一边组成的角，不是任意向外延伸的角

❌ 错误：外角 $\angle ACD = 110°$，学生直接用 $180°-110°=70°$ 求不相邻内角
🔍 诊断：混淆了"外角 = 不相邻两内角之和"与"邻补角 = 180°"。外角 = 不相邻两内角之和，$\angle ACD = \angle A + \angle B$，不是用 $180°$ 去减。

💡 原理：三角形的一个外角=与它不相邻的两个内角之和。原因：外角和内角互补（和=180°），而三角形内角和也是180°→外角=另外两内角之和。用两对"和为180°"的关系推导。

🌿 综合变式：$\triangle ABC$ 中，$\angle A = 80°$，外角 $\angle ACD = 130°$，求 $\angle B$。
解：$\angle ACD = \angle A + \angle B$，$130° = 80° + \angle B$，$\angle B = 50°$。$\angle ACB = 180° - 130° = 50°$，验证 $\angle A + \angle B + \angle C = 80° + 50° + 50° = 180°$ ✓。

2023北京
★☆☆
下列各组线段中，能组成三角形的是（ ）
A. 1, 2, 3   B. 2, 3, 5   C. 3, 4, 8   D. 4, 5, 6
答案：D

🔑 关键思路：判断三线段能否构成三角形，只需验证最短两边之和是否大于最长边——这是必要条件也是充分条件。

  1. 排序识别最长边：每组数从小到大排列，找出最长边。A: 1,2,3；B: 2,3,5；C: 3,4,8；D: 4,5,6。
  2. 验证最短两边之和：A: 1+2=3（等于最长边，不能构成）；B: 2+3=5（等于最长边，不能）；C: 3+4=7<8（小于最长边，不能）；D: 4+5=9>6 ✓。
2022广州
★★☆
一个 n 边形的内角和是 $900°$，则 $n =$ ____。
答案：7

🔑 关键思路：利用 n 边形内角和公式 $(n-2)\times180°$ 反解边数，注意公式中是 $(n-2)$ 而非 $n$。

  1. 列方程：$(n-2)\times180°=900°$。
  2. 两边除以 $180°$：$n-2=900°\div180°=5$。
  3. 解出 $n$：$n=5+2=7$。
  4. 验证：七边形内角和 $=(7-2)\times180°=5\times180°=900°$ ✓。

⚠️ 常见错误：忘记加回2，直接用 $900°\div180°=5$ 答 $n=5$。公式是 $(n-2)\times180°$，$n-2=5$ → $n=7$，不是 $n=5$。

2023杭州
★★★
如图，在 $\triangle ABC$ 中，$\angle A = 50°$，$\angle B = 60°$，则 $\angle ACD$ 的度数为____。
答案：$110°$

🔑 关键思路：三角形的一个外角等于与它不相邻的两个内角之和。图中 $\angle ACD$ 是延长 BC 至 D 形成的外角，不相邻的内角是 $\angle A$ 和 $\angle B$。

  1. 识别外角：图中 BC 延长至 D，$\angle ACD$ 是 $\triangle ABC$ 在顶点 C 处的外角。
  2. 确定不相邻内角：与 $\angle ACD$ 相邻的是 $\angle ACB$，不相邻的是 $\angle A=50°$ 和 $\angle B=60°$。
  3. 应用外角定理：$\angle ACD=\angle A+\angle B=50°+60°=110°$。
  4. 验证：由邻补角得 $\angle ACB=180°-110°=70°$，内角和 $50°+60°+70°=180°$ ✓。

🔄 替代方法：先用内角和求 $\angle ACB=180°-50°-60°=70°$，再用邻补角 $\angle ACD=180°-70°=110°$。外角定理更直接，一步到位。

⚠️ 常见错误1：混淆"外角=不相邻两内角之和"与"邻补角=180°"，用 $180°-110°$ 去求不相邻内角——但题目并未给出 $110°$。

⚠️ 常见错误2：外角定理漏加一个内角，写成 $\angle ACD=\angle A=50°$（只取了一个内角）。

✅ 检验：内角和验证——$\angle A+\angle B+(\angle ACD的补角)=50°+60°+70°=180°$ ✓。

第12章 · 全等三角形

🎯 学习目标

理解全等三角形的概念；掌握 SSS、SAS、ASA、AAS、HL 五种判定方法；能用全等三角形进行几何证明；掌握角平分线的性质与判定

🔗 前置知识：三角形的边角关系（第11章）。全等是八年级几何证明的核心，五种判定方法是后续所有几何证明（轴对称、平行四边形、勾股定理）的基础。
🧭 几何证明的思维方式：证明的本质是"从已知出发，通过定理链条，到达结论"。全等是第一个需要完整证明的章节——你需要写出每一步的推理依据。核心思维：找三个条件 → 确定判定方法 → 写出全等 → 推出对应边角相等。

📋 几何证明入门：从"会看"到"会写"

  1. 标图 — 在图中标注所有已知条件（相等的边用相同标记，相等的角用相同弧线）
  2. 列已知 — 写出题目明确告诉你的条件，格式：∵ 已知，……
  3. 找隐含条件 — 考查有没有"公共边、公共角、对顶角、中点→边相等、角平分线→角相等"
  4. 选判定方法 — 根据找到的三个条件，确定用 SSS/SAS/ASA/AAS/HL 中的哪一个
  5. 书写证明 — 格式：在△___和△___中，条件①…②…③…，∴△___≌△___(判定方法)。∴对应边/角相等。

📖 核心概念
🔍 先行组织者：两张复印出来的纸完全重合——这就是"全等"。全等三角形=形状和大小完全相同的两个三角形。证明全等不需要验6个条件（3边3角），只需要3个关键条件：SSS、SAS、ASA、AAS、HL（直角三角形专用）。注意：SSA不能判定（这是最常见的陷阱）。

全等三角形

能够完全重合的两个三角形。对应边相等、对应角相等。记作 $\triangle ABC \cong \triangle DEF$，注意对应顶点写在对应位置。

🥉 青铜级
SSS（边边边）

三边分别相等的两个三角形全等。最"无脑"的判定——给三边长，直接证。常用于已知"中点→两段相等"的情形。

🥈 白银级
SAS（边角边）

两边和它们的夹角分别相等的两个三角形全等。注意：必须是夹角——SSA（边边角）不能判定全等！这是八年级最高频陷阱。

🥈 白银级
ASA / AAS

ASA：两角和它们的夹边。 AAS：两角和其中一角的对边。三角形内角和为180°，知两角即知第三角 → ASA 和 AAS 本质等价，都可推导出全等。

🥈 白银级
HL（斜边直角边）

仅用于直角三角形：斜边和一条直角边分别相等的两个直角三角形全等。本质是 SSA 在直角三角形中的特例——因为直角确定了第三边。

🥈 白银级
角平分线的性质

角平分线上的点到角两边的距离相等。逆定理：角的内部到两边距离相等的点在角平分线上。这两个定理是全等三角形最经典的应用——用全等证距离相等。

🥇 黄金级
尺规作图——作一个角等于已知角

只用无刻度直尺和圆规完成作图。作一个角等于已知角是基本作图之一。方法：画射线→以已知角顶点为圆心画弧→在目标射线上截取等弧→用圆规量取弧对应的弦长→画等长弧→连线完成。原理：SSS全等。

🥉 青铜级
尺规作图—作线段

已知：线段 $a$。求作：线段 $AB$，使 $AB=a$。
作法：① 画射线 $AC$；② 以 $A$ 为圆心、$a$ 的长为半径画弧，交 $AC$ 于 $B$；③ 线段 $AB$ 即为所求。
原理：圆的半径处处相等——圆规的本质就是"等距离转移器"。

🥉 青铜级
尺规作图—角平分线

已知：$\angle AOB$。求作：射线 $OC$，使 $\angle AOC = \angle BOC$。
作法：① 以 $O$ 为圆心、任意长为半径画弧，交 $OA$ 于 $D$、$OB$ 于 $E$；② 分别以 $D$、$E$ 为圆心、相同半径画弧，两弧交于 $\angle AOB$ 内部的点 $C$；③ 作射线 $OC$，即所求角平分线。
原理：证 $\triangle ODC \cong \triangle OEC$(SSS)→$\angle DOC=\angle EOC$。

🥈 白银级
尺规作图—作垂线

情况1（点在直线上）：以该点为圆心画弧交直线于两点→分别以这两点为圆心画弧交于另一点→连线即垂直（原理：垂直平分线性质的逆用）。
情况2（点在直线外）：以该点为圆心画弧交直线于两点→分别以这两点为圆心画弧交于直线另一侧→连接已知点和交点即垂直。
中考：尺规作图近3年中考卷均出现（选择/填空3分），要求识别作图痕迹对应哪种操作。

🥇 黄金级
📐 判定方法速查
五种判定方法对比

SSS：三边 → 给三条线段长度

SAS：两边+夹角 → 注意"夹角"二字

ASA：两角+夹边 → 给两个角度+中间的那条边

AAS：两角+一角对边 → 知两角则知第三角，可转ASA

HL：斜边+一直角边 → 仅限 Rt△！

⚠️ SSA（边边角）不能判定全等！

🥇 黄金级
角平分线

性质：$P$ 在 $\angle AOB$ 平分线上 $\implies$ $P$ 到 $OA$ 和 $OB$ 的距离相等

判定：$P$ 到 $OA$ 和 $OB$ 距离相等 $\implies$ $P$ 在 $\angle AOB$ 平分线上

隐含条件速查

公共边 → 直接相等（不需要额外证）

公共角 → 直接相等

💡 解题技巧

1 全等判定方法选择决策树

第一步：看有没有直角？有→优先考虑 HL。
第二步：看你找到了什么条件？三个边 → SSS；两边和它们的夹角 → SAS；两角和一边 → ASA/AAS。
第三步：只找到两边一角但角不是夹角（SSA）→ 不能判定！重新找条件。

例题：已知 $AB=DE$，$\angle A=\angle D$，$AC=DF$，证 $\triangle ABC \cong \triangle DEF$。
解：在△ABC和△DEF中，AB=DE, ∠A=∠D, AC=DF。这是 SAS（两边+夹角）。

⚠️ 易错：题目给了"两边一角"不等于可以用 SAS！必须确认这个角是夹角（位于两条边之间）

❌ 错误：题目给了两边一角，学生直接用 SAS——但那个角不是夹角
🔍 诊断：两边一角 $\neq$ SAS！必须确认这个角是两边的夹角。如果角不是夹角，SSA 不能判定全等。考试中 80% 的全等判定错误都是 SSA 陷阱。

📋 中考怎么考：6-8分的几何证明题，判定方法选错直接0分。先画图再选方法！

🌱 基础变式：已知 $\triangle ABC$ 和 $\triangle DEF$ 中，$AB=DE=5$，$\angle B=\angle E=60°$，$BC=EF=7$。判定方法是什么？
解：两边和夹角对应相等 → SAS。

2 找隐含条件——全等证明的关键

题目明写的条件通常不够三个。你需要从图中发现隐含条件：公共边/公共角（最常考！）、对顶角、中点和角平分线带来的相等关系。标注图时用"/"和"//"区分不同组的相等。

例题：如图，$C$ 是 $AB$ 中点，$CD=CE$，$\angle ACD=\angle BCE$，证 $\triangle ACD \cong \triangle BCE$。
解：隐含条件——$C$ 是中点 → $AC=BC$。三个条件：$AC=BC$（隐含），$CD=CE$（已知），$\angle ACD=\angle BCE$（已知）。SAS！

⚠️ 易错：公共边的对应关系——在△ACD中写AC，在△BCE中写对应的BC，别写反

❌ 错误：有公共边时，学生在 $\triangle ABC$ 中写 $AB$，在 $\triangle BAD$ 中写 $BA$，导致对应关系写错
🔍 诊断：公共边是同一个线段，但在不同三角形中名称不同。关键是把对应顶点写在对应位置——$\triangle ABC \cong \triangle BAD$ 意味着 $AB \leftrightarrow BA$，$BC \leftrightarrow AD$，$AC \leftrightarrow BD$。

💡 原理：全等证明的本质是"信息匹配"——已知条件分布在图形各处，你需要把它们收集到两个三角形中。隐含条件就是那些"没写出来但显而易见"的信息。训练自己：每看到一个图形，自动扫描：有没有公共边？公共角？对顶角？中点？垂足？

🌿 综合变式：如图，$AB \parallel DE$，$BC \parallel EF$，$B$ 是 $CF$ 中点。证 $\triangle ABC \cong \triangle DEF$。
解：隐含条件：$B$ 是中点 → $BC=BF$；$AB \parallel DE$ → $\angle ABC = \angle DEF$（同位角）；$BC \parallel EF$ → $\angle ACB = \angle DFE$。ASA → $\triangle ABC \cong \triangle DEF$。

3 辅助线策略（全等篇）

常见辅助线：① 连接两点——最基础的操作，创造公共边或构造新三角形；② 倍长中线——把中线延长一倍，构造全等进行边角转化；③ 截长补短——在长线段上截取等于短线段，或补长短线段。

例题：△ABC中，AD是中线。求证 AB+AC > 2AD。
解：倍长AD至E，连CE。△ADB≌△EDC(SAS)→AB=CE。△ACE中，AC+CE>AE→AC+AB>2AD。

⚠️ 易错：倍长中线的核心是构造全等，关键步骤是证△ADB≌△EDC（用SAS：AD=DE，BD=DC，对顶角∠ADB=∠CDE）

❌ 错误：倍长中线后，学生直接写 $\triangle ADB \cong \triangle EDC$ 忘了加判定条件
🔍 诊断：倍长中线构造全等需要三个条件：$AD=ED$（作的）、$BD=DC$（中点）、$\angle ADB=\angle EDC$（对顶角）。缺少其中任何一个都证不了全等。全等判定不能靠"看起来像"。
🌳 拓展变式：$\triangle ABC$ 中，$AB > AC$，$AD$ 是中线。求证 $\angle BAD < \angle CAD$。
解：倍长 $AD$ 至 $E$，连 $CE$。$\triangle ADB \cong \triangle EDC$ → $AB=CE$，$\angle BAD = \angle CED$。在 $\triangle ACE$ 中，$CE > AC$ → $\angle CAE > \angle CEA$ → $\angle CAD > \angle BAD$。

4 ⚡ 全等证明标准化模板

看到什么条件 → 优先选什么判定：
① 给三组边相等（含中点/公共边）→ SSS
② 给两组边 + 它们的夹角相等 → SAS（必须确认是夹角！）
③ 给两组角 + 一组边相等 → ASA（边是夹边）或 AAS（边是非夹边）
④ 有直角 + 斜边 + 一直角边 → HL

书写模板：在△___和△___中，
条件①：...（已知/已证）
条件②：...（已知/已证）
条件③：...（已知/已证）
∴ △___ ≌ △___（判定方法）
∴ 对应边/角相等（写出需要的结论）

例题：已知点C是AB中点，AD=BE，∠A=∠B，证△ACD≌△BCE。
解：条件分析：①中点→AC=BC（隐含！标记🔍）②AD=BE（已知）③∠A=∠B（已知）→ 两组边+夹角？是SAS！
书写：在△ACD和△BCE中，AC=BC（中点定义），∠A=∠B（已知），AD=BE（已知）。∴△ACD≌△BCE(SAS)。

💡 步骤总结：标已知→找隐含→选判定→写证明。隐含条件三宝：公共边、公共角、中点。

💡 解题技巧

1 三线合一的"三合一"用法

等腰三角形中，知道一条线是中垂线/角平分线/中线/高中的任意一个，就自动拥有另外三个身份。三线合一是等腰三角形解题的万能钥匙——把一个条件变成三个条件。

例题：等腰△ABC中，AB=AC，AD⊥BC于D。求证：D是BC中点。
解：由等腰三角形三线合一，AD既是高又是中线，∴D是BC中点。三线合一直接得结论。

⚠️ 易错：三线合一只适用于等腰三角形！不能在任意三角形中乱用

❌ 错误：在任意三角形中，学生写"因为 AD⊥BC，所以 BD=CD"（用三线合一）
🔍 诊断：三线合一只适用于等腰三角形！在非等腰三角形中，高、中线、角平分线是三条不同的线。必须先证明或已知三角形等腰，才能用三线合一。

💡 原理：等腰三角形是轴对称图形，对称轴就是顶角平分线（也是底边上的中线和高）。翻折不改变形状，所以这条线同时具备了三种身份。"三线合一"本质上就是轴对称的必然结果。

📋 中考怎么考：等腰三角形题必用到三线合一。常作为几何证明题的中间步骤（3-4分），不会直接用但缺它证不下去。

🌱 基础变式：等腰 $\triangle ABC$ 中，$AB=AC=10$，底边 $BC=12$，$AD \perp BC$ 于 $D$。（1）求 $BD$；（2）求 $AD$。
解：（1）三线合一 → $BD=DC=6$。（2）Rt$\triangle$ 勾股：$AD=\sqrt{10^2-6^2}=8$。

2 辅助线：作对称点

遇到"在直线上找一点使距离和最小"→将军饮马。方法：① 作其中一个点关于直线的对称点；② 连接对称点和另一个点；③ 与直线的交点即所求。

例题：河岸l同侧有A、B两村，在l上建水泵站P，使PA+PB最小。
解：作A关于l的对称点A'，连A'B交l于P。根据垂直平分线性质PA=PA'，PA+PB=PA'+PB=A'B。

⚠️ 易错：对称点不能在另一侧选错位置——一定是作关于"动点所在直线"的对称

❌ 错误：作对称点时作反了方向——把 A 对着河作对称，变成了 A 远离河的方向
🔍 诊断：对称点要关于"动点所在的直线"作对称。河是直线 l，A 和 B 在河同侧 → 作 A 关于 l 的对称点 A'。对称方向弄反会导致 A' 跑到远侧，PA+PB 无法转化为直线。

💡 原理：将军饮马问题的数学本质——垂直平分线上的点到线段两端距离相等。作对称点后，PA=PA'，"折线"PA+PB变成了"直线"A'B。两点之间线段最短→A'B就是最小值。核心：对称转化法将折线转变为直线。

🌳 拓展变式：在 $\angle AOB$ 内部有一点 $P$，$OA$ 上找 $M$、$OB$ 上找 $N$，使 $\triangle PMN$ 周长最小。
解：两次反射——作P关于OA的对称点P₁，关于OB的对称点P₂，连P₁P₂与OA、OB的交点即M、N。周长最小值=P₁P₂。

3 ⚠️ 等腰三角形已知两边——必须分类讨论

步骤：① 分两类：腰长=第一个数 / 腰长=第二个数 → ② 分别用三边关系验证能否构成三角形 → ③ 舍去不能构成的情况 → ④ 计算周长。核心：等腰+三边关系联考题，90%需要分类+验证。

例题：等腰三角形有两边长分别为 3 和 7，求周长。
解：分类①：腰=3，底=7 → 两腰和 3+3=6<7，不能构成！舍去。
分类②：腰=7，底=3 → 验证 7+7=14>3 ✓，构成。周长=7+7+3=17。答案：17。

⚠️ 易错：学生通常不分类，直接 $3+3+7=13$ 或 $7+7+3=17$ 碰运气。另外注意：两个数相等时只有一种情况（等边三角形）。

❌ 错误：两边为 3 和 7 → 写周长 = 3+3+7=13（不存在这种情况，因为 3+3<7）
🔍 诊断：分类之后不要忘了用三边关系验证——最短两边之和必须大于最长边。这是中考高频陷阱——不验证直接算的一定丢分。
🌱 基础变式：等腰三角形两边为 5 和 5，求周长。
解：发现两边相等 → 是等边三角形（因为是等腰且两边相等意味着另两边也相等？不——等腰三角形有两边相等即可，这里两边=5意味着腰=5，底未知。分：(1)腰=5底=5→等边，周长=15；(2)腰=5底≠5→需底满足三边关系，底<10即可）→ 需更多条件确定底或周长为 $15$ 或 $5+5+底$。

如图，$\triangle ABC$ 中，$AB = AC$，$AD \perp BC$ 于 $D$。若 $BC = 6$，$AD = 4$，求 $\triangle ABC$ 的面积。
答案：12

🔑 关键思路：等腰三角形中 $AD \perp BC$ → 由三线合一知 $AD$ 既是高也是中线 → 直接用底×高÷2 求面积，无需单独求底边长度。

  1. 应用三线合一：$AB=AC$（等腰），$AD \perp BC$（已知 AD 是高）→ 由三线合一知 $AD$ 同时也是底边 $BC$ 上的中线。
  2. 确认底和高：已知 $BC=6$（底），$AD=4$（高），无需计算 $BD$ 或 $DC$。
  3. 代入面积公式：$S_{\triangle ABC} = \frac{1}{2} \times BC \times AD = \frac{1}{2} \times 6 \times 4 = 12$。
  4. 验证：由勾股定理，$BD=BC/2=3$，$AB=\sqrt{BD^2+AD^2}=\sqrt{9+16}=5$。面积也可用 $\frac{1}{2} \times 6 \times 4 = 12$ ✓。

🔄 替代方法：不求面积而求底边一半→ $BD=BC/2=3$（三线合一），用勾股定理求腰长 $AB=5$，再代入 $S=\frac{1}{2} \times BC \times AD = 12$。结果相同，但直接代入公式更快。

⚠️ 常见错误1：忘记等腰三角形 $AD \perp BC$ 蕴含 $BD=DC$（三线合一），误以为需要单独求 $BD$ 再算面积。

⚠️ 常见错误2：用 $S=\frac{1}{2} \times AB \times AC$ 当成直角三角形面积公式。等腰三角形不是直角三角形，面积必须用底×高÷2。

✅ 检验：用勾股定理反推：$BD=3$，$AB=5$，$S=\frac{1}{2} \times 6 \times 4 = 12$，与底×高÷2 结果一致 ✓。

第14章 · 整式乘法与因式分解

🎯 学习目标

掌握幂的运算性质；熟练运用乘法公式进行整式乘法；能综合运用提公因式法、公式法和十字相乘法进行因式分解

🔗 前置知识：整式加减（第2章）· 一元一次方程（第3章）。后续关联：因式分解是解一元二次方程（九年级）的工具，分式化简（第15章）也大量依赖本章。
🧭 互逆思维：整式乘法和因式分解是互逆的两种运算——就像加法和减法、乘法和除法。学完乘法公式后，反过来看：$a^2-b^2$ 就是 $(a+b)(a-b)$ 的"未展开形式"。掌握"展开"和"还原"的双向能力，是八下解一元二次方程的根基。

📋 因式分解决策树（遇到因式分解题，按此顺序操作）

  1. 先提公因式 — 所有项都有共同因子？先提出来！不提就白费工夫
  2. 看项数 — 两项→平方差公式 $a^2-b^2=(a+b)(a-b)$；三项→完全平方 $a^2\pm 2ab+b^2=(a\pm b)^2$ 或十字相乘；四项及以上→分组分解
  3. 选方法 — 按项数选对应方法，计算验证是否正确
  4. 检查：分解到底了吗？ — 每个因式还能不能再分解？直到每个因式都是"不可再分"为止！

📖 核心概念
🔍 先行组织者：整式乘法和因式分解是一对互逆运算——就像加法和减法、乘法和除法一样。整式乘法=展开（$(x+1)(x+2) → x^2+3x+2$），因式分解=压缩回去（$x^2+3x+2 → (x+1)(x+2)$）。核心公式只有两个：平方差（$a^2-b^2$）和完全平方（$(a±b)^2$）。十字相乘法是特殊技巧——不用公式也能拆。
幂的运算法则

同底数幂相乘：$a^m \cdot a^n = a^{m+n}$。
幂的乘方：$(a^m)^n = a^{mn}$。
积的乘方：$(ab)^n = a^n b^n$。
同底数幂相除：$a^m \div a^n = a^{m-n} (a \neq 0)$。特别注意：$a^0 = 1 (a \neq 0)$。

🥉 青铜级
整式乘法

单项式×单项式：系数×系数，同底数幂×同底数幂。
单项式×多项式：用单项式乘多项式的每一项。
多项式×多项式：用一个多项式的每一项乘另一个多项式的每一项，再合并同类项。

🥈 白银级
平方差公式

$(a+b)(a-b) = a^2 - b^2$。口诀：两数和乘两数差等于平方差。关键特征：两项符号相反（一加一减），且前后项的绝对值相同。$4x^2-9 = (2x+3)(2x-3)$。

🥈 白银级
完全平方公式

$(a+b)^2 = a^2 + 2ab + b^2$。
$(a-b)^2 = a^2 - 2ab + b^2$。
口诀：首平方、尾平方、两倍乘积放中央。正数取"+2ab"，负数取"-2ab"。

🥈 白银级
十字相乘法

对二次三项式 $x^2 + px + q$：找两个数 $m$、$n$，使 $m+n=p$，$mn=q$。则 $x^2+px+q = (x+m)(x+n)$。如 $x^2+5x+6 = (x+2)(x+3)$（因为2+3=5, 2×3=6）。

🥇 黄金级
因式分解的最终原则

① 有公因式必须先提取；② 分解到每个因式都不能再分解为止；③ 结果写成乘积形式。如 $x^4-1=(x^2+1)(x^2-1)=(x^2+1)(x+1)(x-1)$——$x^2-1$ 还能用平方差再分！

❌ 错误：$x^4-1 = (x^2+1)(x^2-1)$ 就停手了 — $x^2-1$ 还能用平方差再分！
🔍 诊断：因式分解必须分到每个因式都不能再分解为止。$x^2-1 = (x+1)(x-1)$，所以最终是 $(x^2+1)(x+1)(x-1)$。判断技巧：每个因式再问自己一遍——还能用什么方法继续分解？
🥇 黄金级
添括号法则（去括号的逆运算）

$$a + b + c = a + (b + c)$$ $$a - b - c = a - (b + c)$$ 添括号后，括号前是"+"，括号内各项不变号；括号前是"−"，括号内各项都要变号。添括号常用于分组分解法因式分解。

多项式÷单项式：用多项式的每一项分别除以单项式。
注意：$a^m \div a^n = a^{m-n}\ (a \neq 0,\ m \geq n)$。

💡 解题技巧

1 平方差公式的识别

形如 $□^2 - ○^2$ 的一切式子都能用平方差。关键是识别"谁是谁的平方"。$4x^2-9$ = $(2x)^2-3^2$ → $(2x+3)(2x-3)$。$x^2-7$ = $(x+\sqrt{7})(x-\sqrt{7})$。

例题：因式分解 $16x^4 - 81y^4$
解：$=(4x^2)^2-(9y^2)^2 = (4x^2+9y^2)(4x^2-9y^2) = (4x^2+9y^2)(2x+3y)(2x-3y)$。注意：$4x^2-9y^2$ 还能用平方差再分！

⚠️ 易错：$x^2+y^2$ 不能用平方差！（符号相同，不是 $a^2-b^2$ 形式）

❌ 错误：$x^2 + y^2$ 用平方差公式分解为 $(x+y)(x+y)$
🔍 诊断：平方差公式要求两项相减（一正一负）。$x^2+y^2$ 是相加，不能用平方差。同样，$-x^2+y^2 = y^2-x^2$ 可以提取负号后再用。

💡 原理：$(a+b)(a-b)=a^2-b^2$的推导——用分配律展开：$(a+b)(a-b)=a·a-a·b+b·a-b·b=a^2-b^2$（中间两项 $-ab+ba=0$ 互相抵消）。正反两用：乘法→展开用左边；因式分解→用右边。看到两个平方相减，条件反射写出两个括号。

📋 中考怎么考：因式分解通常2-4分，试卷前几题。看到"两项且系数是平方数"→立刻反应平方差。

🌱 基础变式：因式分解 $9x^2 - 25y^2$
解：$=(3x)^2-(5y)^2 = (3x+5y)(3x-5y)$。

2 完全平方公式的"凑中间项"

判断一个三项式是否为完全平方：看首尾两项是否都是平方项，再看中间项是否为"2×首×尾"。$9x^2+12x+4$ → $(3x)^2+2·3x·2+2^2 = (3x+2)^2$。

例题：因式分解 $x^2 - 6x + 9$
解：首=$x^2$，尾=$9=3^2$，中间=$-6x = -2·x·3$。→ $(x-3)^2$。

⚠️ 易错：$x^2+4x+4=(x+2)^2$ 不能写成 $(x+4)^2$！中间项必须是 2×首×尾，$4x=2·x·2$ ✓

❌ 错误：$x^2+4x+8$ 被分解为 $(x+4)^2$
🔍 诊断：混淆了 $4x = 2 \cdot x \cdot 2$ 和 $(x+4)^2 = x^2+8x+16$。完全平方公式要求中间项 = $2ab$。$x^2+4x+8$ 中 $8 \neq 2^2=4$，不是完全平方。
❌ 错误：$(x-3)^2 = x^2 + 6x + 9$（符号错误，应是 $x^2 - 6x + 9$）
🔍 诊断：$(a-b)^2 = a^2 -2ab + b^2$，中间项是减号！记忆口诀："完全平方两数平方和，两倍乘积正中间，减号跟着两数差"。

💡 原理：$(a\pm b)^2 = (a\pm b)(a\pm b)$，用分配律展开得 $a^2 \pm 2ab + b^2$。"2ab"来自交叉项 $a·b$ 和 $b·a$ 的合并。首平方+尾平方的和，中间是2倍的首尾积。口诀记忆：首平方、尾平方、首尾两倍中间放。

🌿 综合变式：因式分解 $4x^2 + 20x + 25$
解：首=$(2x)^2$，尾=$5^2$，中间=$20x=2·2x·5$ ✓。→ $(2x+5)^2$。

⚡ 3 十字相乘法的"拆两头凑中间"

对 $x^2+px+q$，找两个数 a 和 b，使 a+b=p 且 ab=q。拆分关键：正q→两数同号（与p同号）；负q→两数异号（p的正负跟随绝对值大的）。

例题：因式分解 $x^2 + 5x + 6$；$x^2 - x - 12$
解1：q=6>0，同号与p=5同正→找a+b=5,ab=6→2+3=5→(x+2)(x+3)
解2：q=-12<0，异号→找a+b=-1,ab=-12→3+(-4)=-1→(x+3)(x-4)

⚠️ 易错：符号判断是关键——q为正，两数同号（都正或都负，看p）；q为负，两数异号

❌ 错误：$x^2 - 5x - 6$ 写成 $(x - 1)(x + 6)$ — $(-1)+6=5 \neq -5$
🔍 诊断：$q=-6$（异号），需找两数积为 -6，和为 -5。正确答案：$-6 + 1 = -5$ ✓ → 写作 $(x-6)(x+1)$。为避免争议，教材统一将绝对值较小的数写在前（$(x+1)(x-6)$ 也可接受，但 $(x-6)(x+1)$ 更规范）。

💡 原理：$(x+m)(x+n)=x^2+(m+n)x+mn$。反过来看，$x^2+px+q$ 要分解为 $(x+m)(x+n)$，必须满足 $m+n=p$ 且 $mn=q$。十字相乘法的"十字"就是交叉相乘再相加验证是否等于中间项——本质是在试验哪两个数满足这两个条件。

🌿 综合变式（含负系数）：因式分解 $x^2 - 3x - 10$
解：q=-10<0 → 两数异号，p=-3→负数的绝对值大。找 a+b=-3, ab=-10。a=2, b=-5 → $(x+2)(x-5)$。验证：$(x+2)(x-5)=x^2-3x-10$ ✓。

4 二次项系数不为1的十字相乘

对于 $ax^2+bx+c$（$a \neq 1$），需要拆两头、凑中间：① 把 $a$ 拆成两个因数 $a_1 \times a_2$；② 把 $c$ 拆成两个因数 $c_1 \times c_2$；③ 交叉相乘再相加，必须等于 $b$（即 $a_1c_2 + a_2c_1 = b$）。

例题：因式分解 $2x^2 + 7x + 3$
解：$a=2$ → 拆成 $2 \times 1$；$c=3$ → 拆成 $1 \times 3$ 或 $3 \times 1$。尝试：2×3+1×1=7 ✓ → $(2x+1)(x+3)$。验证展开：$(2x+1)(x+3)=2x^2+6x+x+3=2x^2+7x+3$ ✓。

⚠️ 易错：$a \neq 1$ 时需要分拆两次——先拆 $a$，再拆 $c$，交叉相乘验 $b$。别只做一个方向的拆分

❌ 错误：$2x^2+7x+3$ 写 $(2x+3)(x+1)$ — 交叉相乘 2×1+3×1=5 ≠ 7
🔍 诊断：不能只看 $a$ 和 $c$ 各自拆开就行，必须验证交叉相乘之和等于 $b$。错了就换一种拆分方式重新试。

💡 原理：$(a_1x+c_1)(a_2x+c_2) = a_1a_2x^2 + (a_1c_2+a_2c_1)x + c_1c_2$。所以要找 $a_1a_2=a$、$c_1c_2=c$、$a_1c_2+a_2c_1=b$。其中 $a$ 和 $c$ 都有多种拆分方式，需要逐一尝试。

🌱 基础变式：因式分解 $3x^2 - 5x - 2$
解：$a=3$ → 3×1；$c=-2$ → 试 -2×1：3×1+1×(-2)=1≠-5 → 换：3×(-2)+1×1=-5 ✓ → $(3x+1)(x-2)$。

🔑 关键思路：识别平方差公式结构——两项和乘以两项差 = 平方差。$(2x+1)(2x-1)$ 中 $a=2x$，$b=1$。

  1. 识别公式类型：$(2x+1)(2x-1)$ 是 $(a+b)(a-b)$ 形式，$a=2x$，$b=1$。
  2. 代入平方差公式：$(a+b)(a-b)=a^2-b^2=(2x)^2-1^2=4x^2-1$。

因式分解：$x^2 - 4 =$____。
答案：$(x+2)(x-2)$

🔑 关键思路：两项且是平方相减 → 平方差公式。$x^2-4=x^2-2^2$，识别 $a=x$，$b=2$。

  1. 识别类型：两项，$x^2$ 和 $4=2^2$ 都是完全平方，中间是减号 → 平方差公式。
  2. 代入逆公式：$a^2-b^2=(a+b)(a-b)$ → $x^2-2^2=(x+2)(x-2)$。
  3. 验证：展开 $(x+2)(x-2)=x^2-4$ ✓。

⚠️ 常见错误：$x^2-4$ 写成 $(x-4)(x+1)$——这是瞎拆。因式分解必须能展开还原验证。

2022杭州
★★☆
因式分解：$x^3 - 4x =$____。
答案：$x(x+2)(x-2)$

🔑 关键思路：因式分解第一步永远是先提公因式！$x^3$ 和 $4x$ 都有公因式 $x$，提出来后再用平方差分解剩余部分。

  1. 提取公因式：$x^3-4x=x(x^2-4)$。
  2. 识别剩余结构：$x^2-4$ 是平方差，$a=x$，$b=2$。
  3. 继续分解：$x(x^2-4)=x(x+2)(x-2)$。
  4. 检查分解到底：$x$（一次式，不可再分）；$(x+2)$（一次式，不可再分）；$(x-2)$（一次式，不可再分）→ 分解到底 ✓。

⚠️ 常见错误：跳过提公因式直接用平方差，写成 $(x\sqrt{x}+2\sqrt{x})(x\sqrt{x}-2\sqrt{x})$——既复杂又错误。先提公因式是铁律。

★★☆
因式分解：$x^2 + 6x + 9 =$____。
答案：$(x+3)^2$

🔑 关键思路：识别完全平方公式的特征——首尾都是平方项，中间项是 $2\times首\times尾$。$x^2$ 是首平方，$9=3^2$ 是尾平方，$6x=2\cdot x\cdot 3$ 匹配中间项。

  1. 检查首尾是否为平方：$x^2$（首平方✓），$9=3^2$（尾平方✓）。
  2. 验证中间项：$2\times首\times尾=2\cdot x\cdot 3=6x$ ✓，匹配题目中的 $+6x$。
  3. 确定符号：中间项为 $+6x$ → 用 $(a+b)^2$ 形式。
  4. 写出结果：$x^2+6x+9=(x+3)^2$。

⚠️ 常见错误：中间项验证失败时强行套公式。如 $x^2+6x+10$，$10 \neq 3^2$，不能写成 $(x+3)^2$。

★★★
因式分解：$a^4 - 1 =$____。
答案：$(a^2+1)(a+1)(a-1)$

🔑 关键思路：高次幂因式分解的核心是反复使用平方差公式，直到每个因式都不能再分解为止。$a^4$ 可视为 $(a^2)^2$，与 $1=1^2$ 构成平方差。

  1. 第一次平方差：$a^4-1=(a^2)^2-1^2=(a^2+1)(a^2-1)$。
  2. 检查能否继续分解：$a^2+1$ 在实数范围内不能继续分解（不是平方差，不能写成两平方相减）。$a^2-1$ 是平方差 → 继续。
  3. 第二次平方差：$a^2-1=a^2-1^2=(a+1)(a-1)$。
  4. 检查分解到底：$a^2+1$（实数范围不可再分✓），$a+1$（一次式✓），$a-1$（一次式✓）。
  5. 完整结果：$a^4-1=(a^2+1)(a+1)(a-1)$。

🔄 替代方法：也可先凑完全平方：$a^4-1=(a^4+2a^2+1)-2a^2=(a^2+1)^2-(\sqrt{2}a)^2$，但引入根号不简便，不如两次平方差直接。

⚠️ 常见错误1：第一次平方差后停手，写成 $(a^2+1)(a^2-1)$ 就交卷——$a^2-1$ 还能再分！中考"分解到底"检查是必考扣分点。

⚠️ 常见错误2：把 $a^2+1$ 继续用平方差写成 $(a+1)(a-1)$——但 $a^2+1$ 是平方和，不是平方差，在实数范围内不能再分解。

✅ 检验：展开验证：$(a^2+1)(a+1)(a-1)=(a^2+1)(a^2-1)=a^4-1$ ✓。

第15章 · 分式

🎯 学习目标

理解分式的概念和基本性质；掌握分式的四则运算；熟练解分式方程，能识别增根并正确检验

🔗 前置知识：因式分解（第14章）是分式通分和约分的工具。分式方程的解法与一元一次方程（第3章）相同，但增加了检验步骤。
🧭 逆向思维：分式 = 分数的"字母版"。所有运算规则（通分、约分、乘除）都与分数完全一致——只是把数字换成了整式。解分式方程时"去分母"与整数方程一致，但多了一步"检验增根"。把握"同"和"异"，一章顶两章。
🧠 思维导图
分式
  • 概念 $\frac{A}{B}$（$B$ 含字母，$B\neq0$）· 有意义的条件 · 值为0的条件
  • 基本性质 $\frac{A}{B} = \frac{A\times C}{B\times C} = \frac{A\div C}{B\div C}$（$C\neq0$）
  • 约分与通分 约分→分子分母同除公因式 · 通分→化为同分母
  • 四则运算 加减：先通分 · 乘除：$\frac{a}{b}\times\frac{c}{d}=\frac{ac}{bd}$ · $\frac{a}{b}\div\frac{c}{d}=\frac{a}{b}\times\frac{d}{c}$

    • 解法 去分母→整式方程→解→检验
    • 增根 使分母为0的解必须舍去！
📖 核心概念
🔍 先行组织者：分数是有理数用除法表示的形式，分式就是分数的代数版——分子分母变成了整式（含字母）。$\frac{1}{x-1}$ 这样的式子，当 x=1 时分母为零（无意义）。分式的运算规则和分数完全一样：约分、通分、乘除、加减。但解分式方程时多了一个"增根"概念——解出来不一定是真解！
分式的概念

形如 $\frac{A}{B}$（$A,B$为整式，$B$含字母）的式子。$B \neq 0$时分式有意义；$A=0$且$B \neq 0$时分式值为$0$。例：$\frac{1}{x-2}$ 中 $x \neq 2$。

🥉 青铜级
分式的基本性质

分式的分子分母乘（或除以）同一个不等于0的整式，分式的值不变。这是约分和通分的理论依据：$\frac{A}{B} = \frac{A \cdot M}{B \cdot M} (M \neq 0)$。

🥉 青铜级
分式的乘除

乘法：$\frac{a}{b} \cdot \frac{c}{d} = \frac{ac}{bd}$。
除法：$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \cdot \frac{d}{c} = \frac{ad}{bc}$。
法则同分数运算，但注意先因式分解再约分。

🥈 白银级
分式的加减

同分母：$\frac{a}{c} \pm \frac{b}{c} = \frac{a \pm b}{c}$。
异分母：先通分→找最简公分母→化为同分母→分子相加减。通分前先对分母进行因式分解！

🥈 白银级
增根（核心陷阱）

解分式方程时，去分母可能产生使原方程分母为0的"解"——增根。增根必须舍去！检验方法：将解代入原方程各分母，看是否为零。为零 = 增根，舍去。

🥇 黄金级
整数指数幂

引入负整数指数：$a^{-n} = \frac{1}{a^n}\ (a \neq 0)$。规定 $a^0 = 1\ (a \neq 0)$。
运算法则统一：
① $a^m \cdot a^n = a^{m+n}$
② $(a^m)^n = a^{mn}$
③ $(ab)^n = a^n b^n$
④ $a^m \div a^n = a^{m-n}$
这些法则对任意整数指数都成立（包括负整数），不再限制 $m \geq n$。
例：$2^{-3} = \frac{1}{2^3} = \frac{1}{8}$；$x^{-2} = \frac{1}{x^2}$。

🥇 黄金级
📐 公式速查
分式运算

乘法：$\frac{a}{b} \cdot \frac{c}{d} = \frac{ac}{bd}$

除法：$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \cdot \frac{d}{c}$

加减：同分母 $\frac{a}{c}\pm\frac{b}{c}=\frac{a\pm b}{c}$；异分母先通分

🥈 白银级
分式方程

步骤：①找最简公分母 → ②去分母 → ③解整式方程 → ④检验（必做！）

增根：使最简公分母为0的解

🥇 黄金级
💡 解题技巧

1 分式化简的标准流程

① 分子分母分别因式分解 → ② 约分（消去公因式）→ ③ 需要加减时，先通分再运算。核心纪律：先分解，再约分；先通分，再加减。

例题：化简 $\frac{x^2-1}{x^2+x} \div \frac{x-1}{x}$
解：$=\frac{(x+1)(x-1)}{x(x+1)} \cdot \frac{x}{x-1} = 1$（先分解再约分，每步消一个公因式）

⚠️ 易错：$x$ 约分的前提是 $x \neq 0$，$x+1$ 约分的前提是 $x \neq -1$。结果正确但需注意定义域

❌ 错误：约分 $\frac{x^2-1}{x-1}=x+1$ 时忘了标注 $x \neq 1$
🔍 诊断：分式约分隐含了定义域条件——$x-1 \neq 0$ 即 $x \neq 1$。考试中如果题目问"化简并写定义域"，漏写条件直接扣分。养成习惯：约分前先写"定义域"。
🌱 基础变式：化简 $\frac{x^2-4}{x^2-2x} \cdot \frac{x}{x+2}$
解：$=\frac{(x+2)(x-2)}{x(x-2)} \cdot \frac{x}{x+2} = 1$（$x \neq 0, \pm 2$）。

2 ⚠️ 增根安检清单（解分式方程必做）

步骤①：识别所有分母→步骤②：列出使各分母为0的x值→步骤③：正常去分母、解方程→步骤④：检查解是否在"禁用名单"中→步骤⑤：在名单中=增根，舍去并标"原方程无解"；不在名单中=真解

例题：解方程 $\frac{1}{x-1} = \frac{2}{x^2-1}$
解：分母 $x-1$ 和 $x^2-1=(x+1)(x-1)$。禁用：$x=1$ 或 $x=-1$。去分母：$x+1=2$，$x=1$。但 $x=1$ 使分母为0→增根，原方程无解。

⚠️ 易错：解完方程不检验直接写答案=0分！中考扣分重灾区

❌ 错误：解分式方程后直接写答案不检验——中考扣分重灾区
🔍 诊断：分式方程解完后必须写"检验"步骤：把解代入原方程各分母，若分母=0则为增根。检验不是可选步骤——是必写步骤。格式："经检验，$x=1$ 使分母 $x-1=0$，是增根，舍去。"

📋 中考怎么考：分式方程通常4-6分，不检验直接扣2分。写了检验过程但检验结果没判断对也扣分。

🌿 综合变式：解方程 $\frac{2}{x+2} + \frac{1}{x-2} = \frac{4}{x^2-4}$
解：分母 $x \neq \pm 2$。$2(x-2)+(x+2)=4$ → $3x-2=4$ → $x=2$。检验：$x=2$ 使分母为 0 → 增根，无解。
❌ 错误：方程 $\frac{x}{x-1} = \frac{2}{x-1}$ → 两边同乘 (x-1)，得 x=2。但忘记分母 x-1≠0。
🔍 诊断：此题两边同乘(x-1)后得到 x=2，代入原方程各分母：x-1=1≠0，检验通过。但如果题目中 x=1 就需要舍去。注意：即使解出的答案正确，没写"检验"步骤仍会被扣分！

解方程：$\frac{x}{x-1} - 1 = \frac{3}{(x-1)(x+2)}$

🔑 关键思路：复杂分式方程先去分母（找最简公分母），解整式方程后必须检验——解出的 $x$ 若使任何分母为零即为增根，应舍去。

  1. 确定定义域：分母 $x-1\neq0$ → $x\neq1$；$(x-1)(x+2)\neq0$ → $x\neq1,-2$。禁用值：$x=1$ 和 $x=-2$。
  2. 找最简公分母：各分母为 $x-1$、$1$、$(x-1)(x+2)$，最简公分母 $(x-1)(x+2)$。
  3. 去分母：两边乘以 $(x-1)(x+2)$：$x(x+2)-(x-1)(x+2)=3$。
  4. 展开并化简：$x^2+2x-(x^2+2x-x-2)=3$ → $x^2+2x-(x^2+x-2)=3$ → $x^2+2x-x^2-x+2=3$ → $x+2=3$ → $x=1$。

🔄 替代方法：先将减号左边的分式通分：$\frac{x}{x-1}-1=\frac{x-(x-1)}{x-1}=\frac{1}{x-1}$，方程化为 $\frac{1}{x-1}=\frac{3}{(x-1)(x+2)}$，交叉相乘得 $x+2=3$，$x=1$。步骤更简但结果相同。

⚠️ 常见错误1：解出 $x=1$ 后不检验直接写"解为 $x=1$"。分式方程的解必须排除增根——$x=1$ 使原方程无意义，这不是真解。

⚠️ 常见错误2：去分母时漏乘某一项。方程左边有 $\frac{x}{x-1}$ 和 $-1$ 两项，去分母时 $-1$ 也要乘以 $(x-1)(x+2)$。漏乘导致方程变形错误。

✅ 检验：将 $x=1$ 代入原方程各分母：$x-1=0$（无意义），$(x-1)(x+2)=0$（无意义）。∴ $x=1$ 是增根，原方程无解 ✓。

第16章 · 二次根式

🎯 学习目标

理解二次根式的概念和最简二次根式；掌握二次根式的四则运算；熟练进行分母有理化

🔗 前置知识：平方根与算术平方根（第6章）· 实数运算。本章是七年级实数章节的深化——从"认识根号"到"运算根号"的跃迁。
🧭 化归思想：二次根式的加减→先化简为同类二次根式→再合并（与整式的合并同类项一模一样）。乘除→利用公式 $\sqrt{a}\cdot\sqrt{b}=\sqrt{ab}$——把复杂根式化为最简。核心思维：一切运算前先化简！
🧠 思维导图
二次根式
  • 概念 $\sqrt{a}\ (a \geq 0)$ · $(\sqrt{a})^2 = a$ · $\sqrt{a^2} = |a|$
  • 最简二次根式 被开方数不含分母 · 不含能开尽方的因数
  • 同类二次根式 化简后被开方数相同的二次根式
  • 运算
    • 乘法 $\sqrt{a}\cdot\sqrt{b}=\sqrt{ab}$
    • 除法 $\frac{\sqrt{a}}{\sqrt{b}}=\sqrt{\frac{a}{b}}$
    • 加减 先化简→合并同类
  • 分母有理化 $\frac{1}{\sqrt{a}} \to \frac{\sqrt{a}}{a}$ · $\frac{1}{\sqrt{a}+\sqrt{b}} \to \frac{\sqrt{a}-\sqrt{b}}{a-b}$
📖 核心概念
🔍 先行组织者：$\sqrt{4}=2$ 你知道，但 $\sqrt{2}$ = 1.414... 是个无限不循环小数（无理数）。二次根式就是带根号的代数式。核心操作：化简（$\sqrt{12}=2\sqrt{3}$）、合并（同类根式相加减）、分母有理化（不让分母有根号）。根号下的数必须≥0——这是根式存在的第一前提。
二次根式的定义

形如 $\sqrt{a}\ (a \geq 0)$ 的式子。$a$ 称为被开方数。$\sqrt{a}$ 表示 $a$ 的算术平方根（结果 $\geq 0$）。$(\sqrt{a})^2 = a$，$\sqrt{a^2} = |a|$。

🥉 青铜级
最简二次根式

满足两个条件：① 被开方数不含分母；② 被开方数中不含能开得尽方的因数或因式。例：$\sqrt{8} = 2\sqrt{2}$，$\sqrt{\frac{1}{2}} = \frac{\sqrt{2}}{2}$。

🥈 白银级
同类二次根式

⚡ 1 化为最简二次根式

方法：① 被开方数含分母→分子分母同乘使其开出；② 被开方数含平方因子→$\sqrt{a^2b}=a\sqrt{b}$。核心：把根号内的"平方数"移到根号外。

解：$\sqrt{48} = \sqrt{16 \times 3} = \sqrt{16} \cdot \sqrt{3} = 4\sqrt{3}$。

⚠️ 易错：$\sqrt{12}+\sqrt{3} \neq \sqrt{15}$！根式加减不能把被开方数直接相加

❌ 错误：$\sqrt{12} + \sqrt{3} = \sqrt{15}$
🔍 诊断：根式加减不能将被开方数直接相加！$\sqrt{12}=2\sqrt{3}$ → $\sqrt{12}+\sqrt{3}=2\sqrt{3}+\sqrt{3}=3\sqrt{3}$。根式的加减类似于合并同类项——只有同类二次根式才能合并。

💡 原理：$\sqrt{a^2b}=a\sqrt{b}$ 是因为 $\sqrt{a^2b}=\sqrt{a^2}·\sqrt{b}=|a|\sqrt{b}$。根号内可以"拆出平方因子"——找到被开方数中最大的完全平方数因子，把它开出根号。$\sqrt{48}=\sqrt{16×3}=\sqrt{16}·\sqrt{3}=4\sqrt{3}$。

🌱 基础变式：化简 $\sqrt{72}$
解：$\sqrt{72} = \sqrt{36 \times 2} = \sqrt{36} \cdot \sqrt{2} = 6\sqrt{2}$。

2 分母有理化技巧

分母含 $\sqrt{a}$→同乘$\sqrt{a}$；分母含 $\sqrt{a}\pm\sqrt{b}$→用平方差公式，同乘共轭式。有理化后分母变为整数。

例题：化简 $\frac{2}{\sqrt{5}-\sqrt{3}}$
解：$=\frac{2(\sqrt{5}+\sqrt{3})}{(\sqrt{5}-\sqrt{3})(\sqrt{5}+\sqrt{3})} = \frac{2(\sqrt{5}+\sqrt{3})}{5-3} = \sqrt{5}+\sqrt{3}$。

⚠️ 易错：有理化后分母为 $(\sqrt{a})^2-(\sqrt{b})^2 = a-b$，忘记处理符号

❌ 错误：$\frac{1}{\sqrt{5}-\sqrt{3}}$ 分母有理化时只乘了分母——分子忘了乘
🔍 诊断：分母有理化必须分子分母同时乘以共轭式。只乘分母不乘分子，分数值变了！$\frac{1}{\sqrt{5}-\sqrt{3}}=\frac{\sqrt{5}+\sqrt{3}}{(\sqrt{5}-\sqrt{3})(\sqrt{5}+\sqrt{3})}=\frac{\sqrt{5}+\sqrt{3}}{2}$。

💡 原理：分母有理化的数学依据——平方差公式。$(\sqrt{a}+\sqrt{b})(\sqrt{a}-\sqrt{b}) = (\sqrt{a})^2-(\sqrt{b})^2 = a-b$（一个整数！）。所以 $\frac{1}{\sqrt{a}+\sqrt{b}}$ 的"共轭式"就是 $\sqrt{a}-\sqrt{b}$，同乘后分母变成整数。

🌿 综合变式：化简 $\frac{6}{\sqrt{7}+\sqrt{5}}$
解：$=\frac{6(\sqrt{7}-\sqrt{5})}{(\sqrt{7}+\sqrt{5})(\sqrt{7}-\sqrt{5})} = \frac{6(\sqrt{7}-\sqrt{5})}{7-5} = 3(\sqrt{7}-\sqrt{5})$。

3 勾股定理的面积法证明

核心思想：用同一个正方形的面积，从两种角度计算——一是直接算正方形面积，二是把正方形拆成4个直角三角形+1个小正方形，两者相等即得勾股定理。

面积法（弦图）：取4个全等的直角三角形（直角边a、b，斜边c），拼成一个大正方形（边长为 $a+b$）。大正方形面积 = $4\times\frac{1}{2}ab + c^2 = 2ab + c^2$。同时大正方形面积 = $(a+b)^2 = a^2 + 2ab + b^2$。两式相等 → $2ab + c^2 = a^2 + 2ab + b^2$ → $c^2 = a^2 + b^2$。

⚠️ 易错：面积法证明的关键步骤是两种方式算同一个面积，建立等式。忘记小正方形面积（$c^2$ 或 $(a-b)^2$）是常见错误。

💡 原理：中国古代数学家赵爽使用"弦图"（即上述面积法）证明了勾股定理，比西方早一千年。这种证明方法的妙处在于：不依赖任何三角形相似或三角比，只用面积加减就完成了证明——是最直观、最易理解的证法。

⚠️ 常见错误：混淆"斜边上的高"和"任意一条直角边"——斜边上的高是从直角顶点向斜边所作的垂线段，不是给定的直角边 5 或 12。另一个常见错误是忘记面积公式中的 $\frac{1}{2}$。

一根旗杆在离地面 $6$ 米处折断，旗杆顶部落在离旗杆底部 $8$ 米处。旗杆原来有多高？
答案：16 米

🔑 关键思路：旗杆折断后形成一个直角三角形——未折断的竖直部分（6米）为一直角边，地面距离（8米）为另一直角边，折断倒下的部分为斜边。先求斜边长，再加上未折断部分即得原高度。

  1. 建立几何模型：旗杆底部为 A，折断处为 B，旗杆顶部落地处为 C。AB = 6 m（竖直未断部分），AC = 8 m（地面水平距离），∠A = 90°。
  2. 勾股定理求BC（折断部分长）：$BC = \sqrt{AB^2 + AC^2} = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10$ 米。
  3. 求原高度：原高度 = 未断部分 AB + 折断部分 BC = $6 + 10 = 16$ 米。
  4. 验证数据合理性：6-8-10 是 3-4-5 的 2 倍，符合勾股数关系 ✓。

⚠️ 常见错误：① 把 6 当作斜边——6 是竖直的未断部分，折断部分 BC 从折点斜着倒下去才是斜边。判断依据：直角在旗杆底部，斜边必然是最长边。② 只算折断部分 10 米，忘了加上未断的 6 米——题目问"原来多高"，不要把两部分加起来等于答案。

🔄 替代方法：若用方程法——设折断部分长为 $x$ 米，原高 $= 6 + x$，由勾股定理 $x^2 = 6^2 + 8^2$ 解 $x = 10$，原高 $= 16$ 米。思路一致但表述不同。

✅ 检验：6² + 8² = 36 + 64 = 100 = 10² ✓。原高 16 = 6 + 10 ✓。

💡 解题技巧

1 判定平行四边形的"黄金钥匙"

五种判定中，"一组对边平行且相等"是最实用的——只需要两个条件（∥和=）而非四个条件。遇到"中点"优先想"对角线互相平分"判定法。

例题：四边形ABCD中，AB=CD，AB∥CD，证ABCD是平行四边形。
解：一组对边平行且相等→平行四边形。一步到位。

⚠️ 易错：一组对边平行+另一组对边相等≠平行四边形（反例：等腰梯形）

❌ 错误：一组对边平行 + 另一组对边相等 → 学生直接判定为平行四边形
🔍 诊断：这是判定陷阱！一组对边平行且另一组对边相等 $\neq$ 平行四边形。反例：等腰梯形——上下底平行，两腰相等，但不是平行四边形。正确的是"一组对边平行且相等"。

💡 原理："一组对边平行且相等"之所以是万能判定法，是因为它用最少的条件得出最强的结论——由平行得内错角相等，由相等和角相等证另一对边平行（全等三角形），两步推出平行四边形。其他判定法要么需要4个条件（两组对边），要么需要隐含条件。

📋 中考怎么考：平行四边形判定是8年级几何证明的核心考点（6-8分），"一组对边平行且相等"是最常选的判定方法。

🌱 基础变式：四边形 $ABCD$ 中，$AB \parallel CD$，$AB=CD=8$。证 $ABCD$ 是平行四边形。
解：一组对边平行且相等 → 平行四边形。一步到位。

2 菱形面积的双公式

菱形面积有两种求法：① $S = ah$（底×高，通用公式）；② $S = \frac{1}{2}d_1 d_2$（对角线乘积的一半，菱形专有）。给对角线长度优先用②。

例题：菱形对角线长分别为 6 和 8，求面积和边长。
解：$S = \frac{1}{2}×6×8 = 24$。对角线互相垂直平分→边长$=\sqrt{3^2+4^2}=5$。

⚠️ 易错：菱形对角线互相垂直且平分，但不一定相等（矩形才是对角线相等）

❌ 错误：菱形对角线长 6 和 8，学生用 $S=ah$ 算面积但找不到高
🔍 诊断：菱形面积两个公式灵活用——$S=ah$（通用）+ $S=\frac{1}{2}d_1d_2$（专用）。给对角线长时直接用 $\frac{1}{2}d_1d_2$，无需找高。走弯路多耗时间还容易出错。

💡 原理：菱形对角线互相垂直——这是因为菱形是特殊的平行四边形（邻边相等→对角线垂直）。把菱形看作4个全等直角三角形（对角线分割而成），面积=4×半个小三角形面积=$\frac{1}{2}d_1d_2$。两个公式（$ah$和$\frac{1}{2}d_1d_2$）等价，但给的已知量不同。

🌿 综合变式：菱形对角线长分别为 10 和 24，求菱形的周长。
解：边长 $=\sqrt{5^2+12^2}=\sqrt{169}=13$（对角线互相垂直平分）。周长 $=4 \times 13 = 52$。

3 辅助线：作平行线或连接中点

遇到中点→连接两点构造中位线。需要证明平行四边形→作平行线或利用对角线互相平分。四边形问题常转化为三角形问题。

例题：四边形ABCD，E、F、G、H分别为各边中点。证EFGH是平行四边形。
解：连AC，EH是△ABD中位线→EH∥BD且EH=BD/2；同理FG∥BD。EH∥FG且EH=FG→EFGH为平行四边形。

⚠️ 易错：区分"三角形中位线"和"梯形中位线"——三角形中位线∥第三边，梯形中位线∥两底

❌ 错误：三角形中位线长度误写为等于第三边（忘了 $\frac{1}{2}$）
🔍 诊断：中位线 = 第三边的一半，不是相等。$DE = \frac{1}{2}BC$ 且 $DE \parallel BC$。记住口诀："中位线缩一半，平行第三边"。

💡 原理：三角形中位线的本质——连接两边中点的线段，相当于把三角形"缩小一半"。证明：用平行四边形（取中点作平行→构造平行四边形→证中位线∥第三边且等于一半）。核心作用：看到中点→想到中位线→想到平行+一半的关系。

🌳 拓展变式：四边形 $ABCD$ 中，$E$、$F$、$G$、$H$ 分别为 $AB$、$BC$、$CD$、$DA$ 中点。求证 $EG$ 和 $FH$ 互相平分。
解：先证 $EFGH$ 为平行四边形 → 平行四边形对角线互相平分。

4 ⚡ 四边形判定路径模板

看到结论要求 → 选最简判定路径：
① 证平行四边形 → 优先用"一组对边平行且相等"（条件最少，只需2条）
② 证矩形 → 先证平行四边形 + 再加一个直角 或 对角线相等
③ 证菱形 → 先证平行四边形 + 再加一组邻边相等 或 对角线垂直
④ 证正方形 → 先证矩形 + 再证菱形（或反过来）

口诀：平四先判∥且=，矩形加直角/对角等，菱形加邻等/对角垂，正方形=矩形+菱形。

例题：ABCD中，AB=CD，AB∥CD，∠A=90°。证ABCD是矩形。
解：① AB=CD且AB∥CD→ABCD是▱（一组对边平行且相等）；② 加上∠A=90°→有一个角是直角的▱是矩形。

★★☆
菱形 $ABCD$ 中，对角线 $AC=6$，$BD=8$，则菱形的面积为____。
答案：24

🔑 关键思路：菱形面积有两个公式——通用公式 $S = ah$（底×高）和专用公式 $S = \frac{1}{2}d_1 d_2$（对角线乘积的一半）。题目直接给了两条对角线长度，用专用公式一步到位。

  1. 识别已知条件：给了对角线长 6 和 8，而非边长与高 → 应使用菱形对角线面积公式。
  2. 公式依据：菱形对角线互相垂直平分，将菱形分成 4 个全等的直角三角形，每个小三角形面积 = $\frac{1}{2} \times \frac{d_1}{2} \times \frac{d_2}{2} = \frac{d_1 d_2}{8}$，4 个合计 $4 \times \frac{d_1 d_2}{8} = \frac{d_1 d_2}{2}$。
  3. 代入计算：$S = \frac{1}{2} \times 6 \times 8 = \frac{48}{2} = 24$。

⚠️ 常见错误：① 混淆公式——用了矩形面积公式 $S = ab$（直接乘对角线得 48）而忘了除以 2。② 混淆菱形和矩形性质——菱形对角线互相垂直但不一定相等，矩形对角线相等但不一定垂直。

★★☆
$\triangle ABC$ 中，$D$、$E$ 分别是 $AB$、$AC$ 的中点，$BC=10$，则 $DE=$____。
答案：5

🔑 关键思路：连接三角形两边中点的线段是中位线。中位线有两个核心结论：① 平行于第三边；② 等于第三边的一半。看到"两个中点"就条件反射想到中位线定理。

  1. 识别中位线：D 是 AB 中点，E 是 AC 中点 → DE 是 $\triangle ABC$ 的中位线（连接两边中点的线段）。
  2. 应用中位线定理：$DE \parallel BC$ 且 $DE = \frac{1}{2}BC$。
  3. 代入计算：$DE = \frac{1}{2} \times 10 = 5$。

⚠️ 常见错误：① 记成中位线等于第三边（忘了乘 $\frac{1}{2}$），错答 10。② 把中位线当成仅平行第三边而忽略了长度关系——中位线同时具备"平行"和"一半"两个结论。

2023重庆
★★★
矩形 $ABCD$，$AB=3$，$BC=4$，则对角线 $AC=$____。
答案：5

🔑 关键思路：矩形对角线将矩形分成两个全等的直角三角形，每条对角线都是该直角三角形的斜边。利用矩形的角都是直角的性质，用勾股定理直接求对角线长。

  1. 识别几何结构：矩形 ABCD 中，∠B = 90°（矩形四个角都是直角），AB = 3 和 BC = 4 是矩形的两条邻边。
  2. 将对角线问题转化为直角三角形问题：连接 AC，△ABC 为直角三角形（∠B = 90°），AB 和 BC 为两条直角边，AC 为斜边。
  3. 勾股定理求斜边：$AC = \sqrt{AB^2 + BC^2} = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$。
  4. 验证矩形对角线性质：矩形对角线相等，若题目同时问 BD，答案也为 5。

⚠️ 常见错误：① 混淆矩形对角线和菱形对角线——矩形对角线相等但不一定垂直，菱形对角线垂直但不一定相等。② 记错矩形对角线长度公式——不是 $\sqrt{AB \times BC}$（别把面积公式和勾股定理搞混）。

🔄 替代方法：已知 3-4-5 是经典勾股数，AB = 3, BC = 4 → 直接得出 AC = 5，无需计算。熟记勾股数可以大幅提升解题速度。

✅ 检验：3² + 4² = 9 + 16 = 25 = 5² ✓。矩形对角线相等的性质：BD 同样 = 5，用勾股定理 $BD = \sqrt{3^2 + 4^2} = 5$ ✓。

🧭 函数思维：函数是研究"变化"的工具——一个量变了，另一个量怎样跟着变。$y=kx+b$ 中，$k$ 是变化率（斜率），$b$ 是初始值（截距）。看到一次函数就想到"一条直线"——把代数翻译成几何。这是高中数学的思维起点。

📋 函数"三语互译"训练

  1. 解析式→图象：$k>0$ 上升，$k<0$ 下降。$|k|$ 越大越陡。$b>0$ 交 $y$ 轴正半轴，$b<0$ 交负半轴
  2. 图象→解析式：找两个点 $(x_1,y_1)$、$(x_2,y_2)$，$k = \frac{y_2-y_1}{x_2-x_1}$（斜率），再代入一点求 $b$
  3. 表格→解析式：看 $x$ 每加 $1$，$y$ 的变化量就是 $k$。$b$ = 当 $x=0$ 时的 $y$ 值

一次函数 y=kx+b 图像解析
函数的概念

在一个变化过程中，如果有两个变量 $x$ 和 $y$，对于 $x$ 的每一个确定的值，$y$ 都有唯一确定的值与其对应，那么 $y$ 是 $x$ 的函数。$x$ 是自变量，$y$ 是因变量。函数的图象是一条平滑的线（一次函数为直线）。

🥉 青铜级
正比例函数 $y=kx$

过原点 $(0,0)$ 的直线。$k>0$：图象经过一、三象限，$y$ 随 $x$ 增大而增大。$k<0$：图象经过二、四象限，$y$ 随 $x$ 增大而减小。$|k|$ 越大，直线越陡。

🥈 白银级
一次函数 $y=kx+b$

$k$（斜率）决定方向和陡峭程度；$b$（截距）决定与 $y$ 轴的交点位置。$y=kx+b$ 的图象是 $y=kx$ 向上平移 $b$ 个单位（$b>0$ 时）。

🥈 白银级
一次函数与方程、不等式

与方程：$kx+b=0$ 的解 = 函数图象与 $x$ 轴交点的横坐标。两直线 $y=k_1x+b_1$ 与 $y=k_2x+b_2$ 的交点 = 方程组 $\begin{cases} y=k_1x+b_1 \\ y=k_2x+b_2 \end{cases}$ 的解。
与不等式：$kx+b>0$ 的解 = 图象在 $x$ 轴上方的 $x$ 范围。

🥇 黄金级
自变量取值范围

求函数自变量取值范围的三种情况：
① 整式：自变量取全体实数（如 $y=x^2+2x+1$）。
② 分式：自变量使分母不为0（如 $y=\frac{1}{x-2}$ 中 $x \neq 2$）。
③ 根式：自变量使被开方数≥0（如 $y=\sqrt{3-x}$ 中 $3-x \geq 0$，$x \leq 3$）。
④ 实际问题：自变量须满足实际意义（如人数必须是非负整数）。
综合题需同时满足多个条件，取交集。

💡 解题技巧

⚡ 1 待定系数法求解析式

正比例：知一点代入 $y=kx$ 求 $k$。
一次函数：知两点→列方程组求 $k$ 和 $b$。知 $k$ 和一点→直接代入求 $b$。

例题：一次函数过点 $(1,3)$ 和 $(2,5)$，求解析式。
解：$k=\frac{5-3}{2-1}=2$，代入$(1,3)$：$3=2×1+b$，$b=1$，∴ $y=2x+1$。

⚠️ 易错：求 $k$ 时坐标为 $(x,y)$ 顺序对，$k=\frac{y_2-y_1}{x_2-x_1}$，分子分母不能反

❌ 错误：已知两点 $(1,3)$ 和 $(2,5)$，求 $k$ 时写 $k=\frac{1-2}{3-5}=\frac{-1}{-2}=0.5$
🔍 诊断：斜率公式 $k=\frac{y_2-y_1}{x_2-x_1}$，分子是 y 的差，分母是 x 的差。$k=\frac{5-3}{2-1}=2$。不要把分子分母顺序弄反。验算：$0.5$ 表示 x 每增加 1，y 只增加 0.5——显然不对。

💡 原理：确定一次函数 $y=kx+b$ 需要确定 $k$ 和 $b$ 两个未知数 → 需要两个独立条件（两个点或一个点和$k$）。两个点→联立方程组消元求解；知道$k$和一个点→直接代入求$b$。核心：未知数个数=条件个数。

🌿 综合变式（表格→解析式）：已知 $y$ 是 $x$ 的一次函数，当 $x=1$ 时 $y=3$，当 $x=3$ 时 $y=7$，求解析式。
解：$k=\frac{7-3}{3-1}=2$，代入$(1,3)$：$3=2×1+b$，$b=1$。∴ $y=2x+1$。

📋 中考怎么考：一次函数求解析式是中考必考题（通常6分），给图表或给两个点求$k$和$b$。$k=\frac{\Delta y}{\Delta x}$ 是第一步，求$b$是第二步。

2 画一次函数图象

最简单：找两个点连线。通常选与 $x$ 轴交点（令 $y=0$）和与 $y$ 轴交点 $(0,b)$。正比例函数只需一个点+原点。

例题：画 $y=2x-4$ 的图象。
解：$x=0→y=-4$（点A）；$y=0→x=2$（点B）。连AB即为图象。

⚠️ 易错：$b$ 是与 $y$ 轴交点的纵坐标，直接标 $(0,b)$ 即可

❌ 错误：画 $y=2x-4$ 时，标 $(0,-4)$ 为"与 x 轴交点"
🔍 诊断：$(0,b)$ 是与 y 轴的交点（x=0），不是与 x 轴的交点。与 x 轴交点是令 $y=0$ → $(2,0)$。混淆两个轴的交点是最常见的函数图像错误。

💡 原理：一次函数图象是一条直线，两点确定一条直线→只需找两个点。最方便的两个点：与 $y$ 轴交点 $(0,b)$ 和与 $x$ 轴交点 $(-\frac{b}{k}, 0)$。$k>0$ 从左下到右上（上升），$k<0$ 从左上到右下（下降）。$|k|$ 越大越陡。

🌱 基础变式：画出 $y=-3x+6$ 的图象，并标注与坐标轴的交点。
解：$x=0$ → $(0,6)$；$y=0$ → $(2,0)$。连接两点即得。$k=-3<0$，下降。

3 k的正负决定图象走向——分类讨论

k > 0（正比例/一次函数上升）：图象从左下到右上，$y$ 随 $x$ 增大而增大。k < 0（下降）：图象从左上到右下，$y$ 随 $x$ 增大而减小。分类讨论的触发条件：题目说"一次函数 y=kx+b 的 y 随 x 增大而减小"→ 自动得出 k<0；反之亦然。

例题：已知一次函数 $y=(m-2)x+3$ 的 y 随 x 增大而减小，求 m 的取值范围。
解：y随x增大而减小→k<0→$m-2<0$→$m<2$。答案：$m < 2$。

⚠️ 易错：k的正负和b的符号是两个独立的概念——k决定走向（上升/下降），b决定起点（与y轴交点在上方/下方）。不要混淆。

🌱 基础变式：已知一次函数 $y=(2k-1)x+4$ 的图象经过一、三象限，求 $k$。
解：过一三象限→k>0→$2k-1>0$→$k>\frac{1}{2}$。

4 图象法解二元一次方程组

将两个方程分别化为 $y = kx + b$ 的形式 → 在同一坐标系中画出两条直线 → 两直线的交点坐标就是方程组的解。

例题：用图象法解方程组 $\begin{cases} x + y = 3 \\ 2x - y = 0 \end{cases}$
解：化为 $y=-x+3$ 和 $y=2x$。画 $y=-x+3$：过 $(0,3)$ 和 $(3,0)$ 作直线；画 $y=2x$：过 $(0,0)$ 和 $(1,2)$ 作直线。两直线交点 $(1,2)$→解为 $x=1, y=2$。验证：$1+2=3$ ✓，$2×1-2=0$ ✓。

⚠️ 易错：如果两直线平行（$k$相同$b$不同），方程组无解；如果两直线重合，方程组有无穷多解。交点不存在或无唯一交点时要对应写出结论。

💡 原理：二元一次方程 $y=kx+b$ 就是一条直线的解析式，方程组的解就是两条直线的公共点坐标——这就是"数形结合"的完美体现：代数方程组的解=几何交点坐标。

★★☆
一次函数 $y = -2x + 3$ 的图象不经过第几象限？
答案：第三象限

🔑 关键思路：一次函数图象经过哪些象限，由 $k$ 和 $b$ 的正负号共同决定。$k$ 决定直线的升降方向，$b$ 决定与 y 轴的交点位置。先判断 $k<0$ 还是 $k>0$，再判断 $b$ 的符号，就能确定直线穿过哪几个象限。

  1. 判断 $k$ 的符号：$k = -2 < 0$，图象从左向右下降（↘），穿过二、四象限。
  2. 判断 $b$ 的符号：$b = 3 > 0$，与 y 轴交于正半轴 $(0, 3)$，即经过第一象限。
  3. 综合分析图象走向：从 y 轴正半轴 $(0, 3)$ 出发，向右下方（k<0）延伸 → 经过第一象限（含交点）、第四象限。向左上方延伸 → 经过第二象限。永远不会进入第三象限（因为 $b>0$ 且直线下降，不可能进入左下区域）。
  4. 快速判断法——口诀：$k<0$, $b>0$ → 过一、二、四象限，不过三。

⚠️ 常见错误：① 混淆 $k$ 和 $b$ 的作用——$k$ 决定升降方向，$b$ 决定起点高低。$k<0$ 不意味着"不过第一象限"，还要看 $b$。② 口诀死记硬背而不会推导——关键是自己画出位置再判断。

★★☆
一次函数过 $(0,2)$ 和 $(3,8)$，求解析式。
答案：$y = 2x + 2$

🔑 关键思路：用待定系数法——先求斜率 $k = \frac{\Delta y}{\Delta x}$，再利用已知点代入求 $b$。注意点 $(0, 2)$ 的 $x=0$，意味着它恰好在 y 轴上，直接给出截距 $b=2$。

  1. 求斜率 $k$：$k = \frac{y_2 - y_1}{x_2 - x_1} = \frac{8 - 2}{3 - 0} = \frac{6}{3} = 2$。
  2. 求截距 $b$（利用点 $(0,2)$）：$y = kx + b$，代入 $x=0, y=2$：$2 = 2 \times 0 + b$，$b = 2$。（$b$ 的几何意义：点 $(0,b)$ 就是与 y 轴的交点，此题直接给出来了。）
  3. 验证另一个点：将 $x=3$ 代入 $y = 2x + 2$，得 $y = 2 \times 3 + 2 = 8$ ✓。

⚠️ 常见错误：① 斜率公式分子分母颠倒——写成 $k = \frac{x_2-x_1}{y_2-y_1} = \frac{3}{6} = 0.5$（错误）。正确公式 $k = \frac{y_2-y_1}{x_2-x_1}$，分子是 y 的差。② 忽略点 $(0,2)$ 直接给了 $b$ 的便利，又去列方程组算一遍，浪费时间。

★★★
已知一次函数 $y = kx + b$ 的图象与 $y = 2x - 1$ 平行，且过点 $(1, 4)$，求解析式。
答案：$y = 2x + 2$

🔑 关键思路：两条直线平行 → 斜率相等（$k_1 = k_2$）。直接从已知直线读出 $k=2$，然后代入点 $(1,4)$ 求出 $b$。平行关系把两个未知量 $(k,b)$ 减少为一个，只需一个条件（过某点）即可确定。

  1. 由平行确定 $k$：目标直线与 $y = 2x - 1$ 平行，两直线平行 $\implies$ 斜率相同，$k = 2$。
  2. 代入点求 $b$：$y = 2x + b$ 过点 $(1, 4)$，代入：$4 = 2 \times 1 + b$。
  3. 解出 $b$：$b = 4 - 2 = 2$。
  4. 写出解析式：$y = 2x + 2$。

⚠️ 常见错误：① 把"平行"理解成 $b$ 相同——两条直线平行要求 $k$ 相同、$b$ 不同。若 $k$ 和 $b$ 都相同则两条直线重合，不是平行。② 读题时将已知直线 $y=2x-1$ 中的 $b=-1$ 误用为目标直线的 $b$。

🔄 替代方法：也可以设解析式为 $y=2x+b$，然后直接代入 $(1,4)$ 得 $b=2$。方法本质上与 $k = \frac{y - y_1}{x - x_1}$ 的"点斜式"思路相同——知道斜率和一点，即可唯一确定直线。

✅ 检验：代入 $x=1$：$y=2\times1+2=4$ ✓。验证平行性：目标 $k=2$ = 已知直线 $k=2$ ✓。

第20章 · 数据的分析

🎯 学习目标

掌握平均数、中位数、众数的概念和计算；理解加权平均数的意义；会用方差衡量数据的波动程度

🔗 前置知识：数据的收集与整理（第10章）。本章从"描述数据"跃迁到"分析数据"——不只是画出统计图，更要理解数据的集中趋势和离散程度。
🧭 数据思维：看数据不能只看一个数。平均数告诉你"一般水平"，中位数告诉你"中间水平"，方差告诉你"稳定程度"。三个指标分别回答三个不同问题——选错指标，结论就错。这也是为什么比赛要去掉最高最低分——用平均数的思想优化平均数。
🧠 思维导图
数据的分析
  • 平均数（集中趋势）
    • 算术平均数 $\bar{x} = \frac{x_1+x_2+\cdots+x_n}{n}$
    • 加权平均数 $\bar{x} = \frac{w_1x_1+w_2x_2+\cdots}{w_1+w_2+\cdots}$
  • 中位数 排序后中间的数（不受极端值影响）
  • 众数 出现次数最多的数（可多个或没有）
  • 方差（离散程度） $s^2 = \frac{1}{n}\sum(x_i-\bar{x})^2$ · 越小越稳定
📖 核心概念
🔍 先行组织者：一个班考试成绩出来了，如何描述？平均数=总分÷人数（受极端值影响大），中位数=中间那个人的分数（不受极端值影响），众数=出现最多的分数，方差=分数的波动程度（越小越稳定）。选哪个统计量代表全班水平？这取决于你想强调什么——统计是有观点的。一个亿富翁进入村里，平均收入飙升但中位数不变——这就是为什么"人均"有时候不靠谱。
平均数

所有数据之和除以数据个数。公式：$\bar{x} = \frac{1}{n}(x_1+x_2+\cdots+x_n)$。平均数能反映数据的一般水平，但受极端值影响大。

🥉 青铜级
加权平均数

各数据乘以权重的和除以权重之和。$\bar{x} = \frac{w_1x_1+w_2x_2+\cdots+w_nx_n}{w_1+w_2+\cdots+w_n}$。权越大，数据对平均数的影响越大。

🥈 白银级
中位数与众数

中位数：数据排序后最中间的数。不受极端值影响，反映"中等水平"。
众数：出现次数最多的数。可能有多个，反映"多数水平"。

🥈 白银级
方差

衡量数据波动程度的指标。$s^2 = \frac{1}{n}[(x_1-\bar{x})^2+(x_2-\bar{x})^2+\cdots+(x_n-\bar{x})^2]$。方差越小，数据越稳定。标准差 $s = \sqrt{s^2}$。

🥇 黄金级
用样本估计总体

在抽样调查中，用样本的统计量来推断总体的对应参数：
① 用样本平均数估计总体平均数。
② 用样本方差估计总体方差。
③ 用样本比例估计总体比例。
关键原则：样本容量越大、抽样越随机，估计越准确。抽样调查的最终目的是得到对总体参数的可靠估计，而不是精确计算。

💡 解题技巧

1 平均数vs中位数vs众数——选谁说话

数据有极端值时，平均数会被拉偏（如一个亿万富翁拉高全村平均工资）→用中位数更客观。比赛评分去掉最高最低再平均——就是为了消除极端值的影响。众数用于知道"大多数是什么情况"。

例题：某公司员工工资：3k, 4k, 4k, 5k, 6k, 100k。平均=20.3k，中位数=4.5k。
解：平均数被CEO的100k拉到了20.3k，不代表一般情况。中位数4.5k才是真实水平。

⚠️ 易错：数据个数为偶数时，中位数=中间两个数的平均数

❌ 错误：数据 1, 2, 3, 4, 100 求中位数 → 学生取中间两个的平均 $(3+4)/2=3.5$
🔍 诊断：数据已排序（5 个），中位数是第 3 个 → 3（不是中间两个的平均）。只有偶数个数据时才取中间两个的平均。记忆：奇数取中间，偶数取两个平均。

💡 原理：三个统计量回答三个不同问题。平均数→"总和平均后每人有多少"（受极端值影响）；中位数→"中间那个人有多少"（不受极端值影响）；众数→"大多数人有多少"。选对指标=选对故事。公司薪资用中位数描述比平均数更诚实。

📋 中考怎么考：平均值/中位数/众数选择填空必考（2-4分），解答题也可能给数据让你算统计量。关键是会选谁来说话。

🌱 基础变式：数据：2, 2, 3, 8, 10。求平均数、中位数、众数。哪个最能代表这组数据？
解：平均数=5，中位数=3，众数=2。中位数 3 最能代表（平均数被 10 拉高）。

⚡ 2 方差计算与稳定性判断

步骤：① 求平均数→② 求每个数与平均数的差→③ 差的平方→④ 求平方的平均数。方差越小越稳定。

例题：甲乙两人射击成绩：甲(8,9,8,9)、乙(10,5,10,7)。谁更稳定？
解：甲 $\bar{x}=8.5$，$s^2=\frac{(8-8.5)^2+(9-8.5)^2+(8-8.5)^2+(9-8.5)^2}{4}=0.25$。乙同理得 $s^2=4.25$。甲方差小→更稳定。

⚠️ 易错：方差单位是原数据单位的平方（如"斤²"），标准差单位与原数据相同

❌ 错误：方差越大越稳定——学生把"方差"和"成绩高"混淆
🔍 诊断：方差衡量偏离程度，越小越稳定。记忆口诀："方差小→数据齐→成绩稳"。不要把"方差大"当成好成绩——方差大意味着成绩忽高忽低。

💡 原理：方差衡量"偏离中心的程度"。每个数据减去平均数（偏离量）→平方（消除正负抵消）→求平均（得到一个统一的偏离指标）。方差大→数据分散（不稳定）；方差小→数据集中（稳定）。方差值本身不重要，重要的是比较两个方差的相对大小。

📋 中考怎么考：统计题最后一小问（1-2分），经常问"哪组更稳定"用方差判断。记住：方差小 = 稳定。

🌿 综合变式：两组数据：A(3,5,5,7)、B(2,4,6,8)。哪组更稳定？
解：$ \bar{x}_A=5$，$s^2_A=\frac{(3-5)^2+(5-5)^2+(5-5)^2+(7-5)^2}{4}=\frac{4+0+0+4}{4}=2$。$ \bar{x}_B=5$，$s^2_B=\frac{9+1+1+9}{4}=5$。$2<5$ → A 组更稳定。

★☆☆
数据：2, 4, 4, 6, 8 的平均数是____，众数是____。
答案：平均数 4.8，众数 4

🔑 关键思路：平均数是所有数据之和除以数据个数；众数是出现频次最高的数。两个统计量分别反映数据的"一般水平"和"多数情况"。

  1. 求平均数：$\bar{x} = \dfrac{2 + 4 + 4 + 6 + 8}{5} = \dfrac{24}{5} = 4.8$。
  2. 求众数：统计频次：2（1次）、4（2次）、6（1次）、8（1次）→ 出现最多的是 4，众数 = 4。
2022广州
★★☆
数据：3, 5, 7, 7, 9 的中位数是____。
答案：7

🔑 关键思路：中位数是数据排序后位于最中间的数，不受极端值影响。数据个数为奇数时，取正中间那个数；偶数时，取中间两个数的平均数。

  1. 确认数据已排序：数据 3, 5, 7, 7, 9 已按从小到大的顺序排列。
  2. 确定中位数位置：共 5 个数据（奇数个），中位数是第 $\frac{5+1}{2} = 3$ 个。
  3. 取出中位数：第 3 个数据是 7。

⚠️ 常见错误：① 忘记先排序就直接取"看起来中间"的数——如果数据是乱序（如 9, 3, 7, 5, 7），必须先排序才能找中位数。② 数据个数为奇数时误用"中间两个取平均"——只有偶数个数据时才取中间两个的平均数。

2023杭州
★★★
甲组数据：1, 2, 3, 4, 5；乙组数据：1, 1, 5, 9, 9。哪组数据的方差大？
答案：乙组方差大

🔑 关键思路：方差衡量数据偏离平均数的程度——方差越小数据越稳定（越集中），方差越大数据越分散。定性判断：观察两组数据的分布特点，甲组均匀递增（每个数差距≈1），乙组两极分化（1和1 vs 9和9），显然乙组更分散 → 方差更大。

  1. 求甲组平均数与方差：$\bar{x}_甲 = \frac{1+2+3+4+5}{5} = 3$。$s^2_甲 = \frac{(1-3)^2+(2-3)^2+(3-3)^2+(4-3)^2+(5-3)^2}{5} = \frac{4+1+0+1+4}{5} = \frac{10}{5} = 2$。
  2. 求乙组平均数与方差：$\bar{x}_乙 = \frac{1+1+5+9+9}{5} = \frac{25}{5} = 5$。$s^2_乙 = \frac{(1-5)^2+(1-5)^2+(5-5)^2+(9-5)^2+(9-5)^2}{5} = \frac{16+16+0+16+16}{5} = \frac{64}{5} = 12.8$。
  3. 比较：$s^2_甲 = 2$，$s^2_乙 = 12.8$。$12.8 > 2$ → 乙组方差大。
  4. 直观验证：甲组数据集中在 1~5，离平均数 3 的偏差在 ±2 以内；乙组数据在 1 和 9 处分别有 2 个值，离平均数 5 的偏差达 ±4——乙组数据波动剧烈得多。

⚠️ 常见错误：① 认为"方差越大越好"——方差衡量的是波动程度，不是数据大小。比赛成绩中方差小意味着发挥稳定。② 计算方差时忘记先求平均数，直接用数据套公式——方差的核心是"与平均数的偏差"。

🔄 替代方法：对于选择题，无需精确计算即可判断。甲组每个数间隔均匀（1,2,3,4,5），方差必小；乙组数据出现明显的两端集中（1,1 vs 9,9），方差必大。定性分析可秒杀结论。

✅ 检验：甲组标准差 $s_甲 \approx 1.41$，乙组标准差 $s_乙 \approx 3.58$。标准差的量级差异也验证了乙组方差更大 ✓。
