第21章 信息的传递

🎯 学习目标

① 理解电话的工作原理——话筒将声信号变为电信号，听筒将电信号变为声信号

② 知道电磁波的产生条件（变化的电流），掌握波速公式 $c = \lambda f$ 并进行计算

③ 了解电磁波谱的分类及各种通信方式（无线电广播、电视、移动通信、光纤通信）的特点

⛳ 零基础路线： 先读①电磁波的产生与传播 → ② $c=\lambda f$ 计算 → ③通信方式，跳过电话交换机的细节。重点：掌握电磁波与声波的本质区别（电磁波不需介质、真空中也可传播），会进行波长、频率、波速的相互换算，能按频率大小排列电磁波谱各波段。
📎 前置知识关联： 第20章·电与磁——电磁感应（变化的电流产生变化的磁场，变化的磁场产生电场，形成电磁波）； 八年级·声现象——波的基本概念（波长、频率、波速的关系），声波不能在真空中传播（对比电磁波）； 第15章·电流和电路——电流的磁效应，变化的电流是产生电磁波的根本原因； 生活经验·使用手机、Wi-Fi、蓝牙、收音机和电视的直观体验。
🔬 物理学思维方法：类比法 将抽象的电磁波与直观的水波、声波进行类比，通过"波"的共同特征——波长、频率、波速——来理解电磁波的传播规律。水波振动的是水分子，声波振动的是空气分子，而电磁波不需要介质。这种"同中求异"的比较思维，既是本章的重要学习方法，也是物理学研究的基本思路。

📖 核心概念
🔍 先行组织者：你拿起手机拨打电话，声音瞬间传到千里之外——这是怎么做到的？声音本身传不远，但把它"编码"成电信号，加载到电磁波上，以光速传播，再"解码"还原出来。本章揭开通信的底层魔法：话筒如何把你的声音变成变化电流？电磁波如何"驮着"信息在真空中飞驰？为什么数字通信比模拟通信更清晰？信息之路从电话线走向光纤、卫星，越来越宽。
电话的原理

话筒：声音使膜片振动 → 压紧或放松碳粒 → 电阻变化 → 电流相应变化（声信号→电信号）。

听筒：变化的电流通过电磁铁 → 磁性强弱变化 → 吸引膜片振动 → 还原声音（电信号→声信号）。

电话交换机：将多部电话机连接成网络，通过交换设备实现任意两部电话之间的通话转接。

🥈 白银级
电磁波的产生

条件：变化的电流（振荡电流）在其周围空间产生变化的磁场，变化的磁场又产生变化的电场，交替向外传播形成电磁波。

即使只有一根导线中通过高频振荡电流，也能向空间发射电磁波。

理论和实验都证明：任何变化的电场或磁场都会在周围空间产生电磁波。

🥇 黄金级
电磁波的传播

不需要介质：电磁波在真空中也能传播——这是电磁波与声波最本质的区别。

真空中波速恒定：$c = 3 \times 10^8\ \text{m/s}$（光速）。

在介质中传播时速度小于真空中的速度：$v = \dfrac{c}{n}$（$n$ 为介质折射率）。

传播速度与频率无关（真空中），但不同频率的电磁波在介质中速度略有不同。

🥇 黄金级
波长、频率与波速

波长 $\lambda$：波在一个周期内传播的距离，单位 $\text{m}$。

频率 $f$：每秒振动的次数，单位 $\text{Hz}$（赫兹）。

波速 $c$：波在单位时间内传播的距离。

三者关系：$c = \lambda f$（波速恒定，波长与频率成反比）。

频率越高，波长越短。例如：中波广播 $f=1\ \text{MHz}$ → $\lambda = 300\ \text{m}$；Wi-Fi $f=2.4\ \text{GHz}$ → $\lambda \approx 0.125\ \text{m}$。

🥇 黄金级
电磁波谱

按频率从低到高（波长从长到短）排列：

无线电波（$10^3$–$10^{12}\ \text{Hz}$）→ 通信/广播

微波 → 卫星通信/微波炉（$3\times10^8$–$3\times10^{11}\ \text{Hz}$）

红外线 → 遥控/夜视（$10^{12}$–$4\times10^{14}\ \text{Hz}$）

可见光 → 人眼可见（$4\times10^{14}$–$8\times10^{14}\ \text{Hz}$）

紫外线 → 消毒/验钞（$8\times10^{14}$–$10^{16}\ \text{Hz}$）

X 射线 → 医学拍片（$10^{16}$–$10^{20}\ \text{Hz}$）

$\gamma$ 射线 → 医疗/探测（$>10^{20}\ \text{Hz}$）

🥇 黄金级
模拟通信 vs 数字通信

模拟通信：传输的信号在时间和幅度上都是连续变化的。例如普通电话将声音的振动波形直接转换为连续变化的电流波形。缺点是信号易受干扰，传输过程中会失真。

数字通信：将信息编码成0 和 1 两个二进制数字进行传输。抗干扰能力强（只需判断 0/1，无需精确还原波形），信息加密方便，适合计算机处理。

现代通信以数字方式为主，但仍保留模拟电话线的兼容。

🥈 白银级
光纤通信

利用激光在光导纤维中经过多次全反射来传递信息。

激光频率极高（$10^{14}\ \text{Hz}$ 级别），通信容量巨大——一根光纤可同时传输数百万路电话。

优点：传输容量大、损耗小、抗电磁干扰、保密性好。

🥈 白银级
移动通信与卫星通信

移动通信：手机通过基站（基地台）转发信号。通话过程中，手机会自动切换到信号最强的邻近基站（越区切换）。

卫星通信：利用通信卫星作为中继站，将地面站发来的微波信号转发到另一地面站。三颗同步卫星可基本实现全球通信。

🥉 青铜级
📐 公式速查
波速公式

\[ c = \lambda f \]

$c$ — 波速（$\text{m/s}$），$f$ — 频率（$\text{Hz}$），$\lambda$ — 波长（$\text{m}$）
真空中 $c = 3 \times 10^8\ \text{m/s}$（恒定不变）
变形：$\lambda = \dfrac{c}{f}$ $f = \dfrac{c}{\lambda}$

🥇 黄金级
电磁波在介质中的波速

\[ v = \frac{c}{n} \]

$v$ — 介质中的波速，$c$ — 真空中的波速，$n$ — 介质的折射率（$n > 1$）
因此电磁波在介质中传播速度小于真空中的速度。

🥈 白银级
频率与周期关系

\[ f = \frac{1}{T} \]

$T$ — 周期（$\text{s}$），$f$ — 频率（$\text{Hz}$）
结合 $c = \lambda f$ 可得 $\lambda = cT$

🥉 青铜级
✍️ 解题技巧

1 波长与频率的计算

📌 方法：已知其中两个量，用 $c = \lambda f$ 求第三个量。注意频率单位的数量级换算。
例题：某调频广播电台的频率为 $f = 100\ \text{MHz}$，求该电磁波在真空中的波长。
解：已知 $c = 3 \times 10^8\ \text{m/s}$，$f = 100\ \text{MHz} = 10^8\ \text{Hz}$。
$\lambda = \dfrac{c}{f} = \dfrac{3 \times 10^8}{10^8} = 3\ \text{m}$。
⚠️ 易错点：频率单位换算！$1\ \text{MHz} = 10^6\ \text{Hz}$，$1\ \text{GHz} = 10^9\ \text{Hz}$。代入公式前必须统一为 $\text{Hz}$。
❌ 错误：看到 $100\ \text{MHz}$ 直接代入 $f=100$，得到 $\lambda = 3 \times 10^6\ \text{m}$
🔍 诊断：$100\ \text{MHz} = 100 \times 10^6\ \text{Hz} = 10^8\ \text{Hz}$，正确的波长是 $3\ \text{m}$ 而不是 $3$ 百万米。注意处理 $\text{M}$（百万）、$\text{k}$（千）、$\text{G}$（十亿）等数量级前缀。
📐 原理依据：对于电磁波，在真空中波速 $c$ 是常数。$c = \lambda f$ 表明波长和频率成反比——频率越高，波长越短。
🌱 基础变式：一个手机 Wi-Fi 信号频率为 2.4 GHz，求其在真空中的波长。（$f=2.4 \times 10^9\ \text{Hz}$，$\lambda = 3\times10^8 / 2.4\times10^9 = 0.125\ \text{m} = 12.5\ \text{cm}$）
🏆 考场提示：中考中一般直接用 $c = \lambda f$ 进行简单计算，重点在于单位的数量级换算。偶尔会结合介质波速出对比题。

2 电磁波与声波的对比

📌 方法：抓住核心区别——是否需要介质。
例题：宇航员在月球上面对面交谈，能否直接听到对方的声音？为什么？他们如何实现通信？
解：不能直接听到。月球表面是真空，声波需要介质传播，真空不能传声。宇航员通过无线电（电磁波）通信，因为电磁波可以在真空中传播。
⚠️ 易错点：混淆电磁波和声波的速度。光（电磁波）在空气中速度约 $3\times10^8\ \text{m/s}$，而声波在空气中约 $340\ \text{m/s}$——相差近一百万倍。打雷时先看到闪电后听到雷声就是这个原因。
📐 原理依据：电磁波（包括光）是横波，本质是变化的电场和磁场，不依赖介质传播。声波是机械波（纵波），依靠介质中质点的振动传播。
🌱 基础变式：一人在距山崖 680 m 处鸣枪，听到回声的时间为 4 s。若在山顶用无线电发射信号，信号到达 300 km 外接收站需要多长时间？（$t_{\text{声}} = 2s/v_{\text{声}} = 1360/340 = 4\ \text{s}$；$t_{\text{电}} = s/c = 3\times10^5 / 3\times10^8 = 0.001\ \text{s}$）
🏆 考场提示：中考常以"太空对话""深海通信"等情境考查电磁波与声波的传播条件对比，属于基本概念题。

3 电磁波谱排序与识别

📌 方法：按频率从低到高（波长从长到短）记忆：无→微→红→可→紫→X→γ
例题：下列电磁波中，频率最高的是（ ）
A. 红外线　B. 紫外线　C. X 射线　D. 无线电波
解：电磁波谱中频率从低到高为：无线电波 < 红外线 < 可见光 < 紫外线 < X 射线 < $\gamma$ 射线。四个选项中 X 射线频率最高。选 C。
⚠️ 易错点：可见光只是电磁波谱中很窄的一段。红外线的频率低于可见光（红光外侧），紫外线的频率高于可见光（紫光外侧）。另外，所有电磁波在真空中传播速度相同！
📐 原理依据：整个电磁波谱按照频率/波长统一排列，不同波段的电磁波产生方式不同、性质不同、应用各异，但在真空中传播速度完全一样。
🌱 基础变式：下列器件或应用中，利用红外线工作的是（ ）A. 验钞机　B. 遥控器　C. 消毒柜　D. 拍胸片。——选 B（遥控器利用红外线传递信号）。
🏆 考场提示：中考常以"下列应用中利用了××电磁波的是"的形式考查电磁波谱各波段的应用。属于记忆型题目。

2022 广州
★★☆
题 2
某电视台发射的电磁波频率为 $f = 5.0 \times 10^8\ \text{Hz}$。求：
（1）该电磁波的波长；
（2）若该电磁波从电视台传播到 30 km 外的接收站需要多长时间。
（真空中电磁波速度 $c = 3.0 \times 10^8\ \text{m/s}$）

（1）由 $c = \lambda f$ 得 $\lambda = \dfrac{c}{f} = \dfrac{3.0 \times 10^8}{5.0 \times 10^8} = 0.6\ \text{m}$。

📝 三级闯关 · 课后练习

🥉 基础关 · 概念识记

基础 ★☆☆
电磁波在真空中的传播速度是多少？
解析：电磁波在真空中的传播速度等于光速，$c = 3 \times 10^8\ \text{m/s}$。这是宇宙中最快的速度，也是恒定不变的常数。电磁波在介质中传播时速度会变慢（$v = c/n$，$n$ 为介质折射率）。
🏆 得分点：答出 $3 \times 10^8\ \text{m/s}$；说明真空中速度等于光速且恒定。
基础 ★☆☆
波速公式 $c=\lambda f$ 中每个符号的含义是什么？
解析：$c$ ——波速，电磁波在单位时间内传播的距离（真空中 $c=3\times10^8\ \text{m/s}$）；$\lambda$ ——波长，波在一个周期内传播的距离（单位 $\text{m}$）；$f$ ——频率，每秒振动的次数（单位 $\text{Hz}$）。三者关系：波速 = 波长 × 频率。在真空中 $c$ 恒定时，波长与频率成反比。
🏆 得分点：正确写出每个符号的物理含义和单位；说明真空中 $c$ 恒定、$\lambda$ 与 $f$ 成反比。
基础 ★☆☆
电话话筒和听筒分别将什么信号转换成什么信号？
解析：话筒将声信号转化为电信号（变化的电流）——声音使膜片振动，压紧或放松碳粒，使电阻变化，电流相应变化。听筒将电信号转化为声信号——变化的电流通过电磁铁，磁性强弱变化吸引膜片振动，还原出声音。话筒和听筒合在一起就构成了电话的收发装置。
🏆 得分点：正确写出话筒：声→电；听筒：电→声；简要说明各自的转化原理。

🥈 提升关 · 中考应用

提升 ★★☆
某电磁波频率 $100\ \text{MHz}$，求波长（$c=3\times10^8\ \text{m/s}$）
解析：由 $c = \lambda f$ 得 $\lambda = \dfrac{c}{f}$。先将频率单位统一为 $\text{Hz}$：$f = 100\ \text{MHz} = 100 \times 10^6\ \text{Hz} = 10^8\ \text{Hz}$。代入公式：$\lambda = \dfrac{3 \times 10^8}{10^8} = 3\ \text{m}$。
🏆 得分点：正确进行单位换算（$\text{MHz} \to \text{Hz}$）；正确代入公式 $c = \lambda f$ 求解；答案和单位正确。
提升 ★★☆
模拟通信和数字通信的区别是什么？哪种抗干扰能力更强？
解析：模拟通信传输的信号是连续变化的，直接模拟声波或图像的波形，信号易受干扰、传输过程中会失真。数字通信将信息编码成0 和 1 两个二进制数字进行传输，接收端只需判断"有信号/无信号"，无需精确还原波形。数字通信抗干扰能力更强——即使传输过程中信号有所衰减或变形，仍能准确判断 0 和 1，还原出原始信息。因此现代通信已全面转向数字方式。
🏆 得分点：清楚说明模拟通信（连续变化）和数字通信（0/1编码）的本质区别；正确判断数字通信抗干扰更强；解释原因。
提升 ★★☆
光纤通信利用了什么原理？有什么优点？
解析：光纤通信利用激光在光导纤维中经过多次全反射来传递信息。激光频率极高（约 $10^{14}\ \text{Hz}$ 级别），通信容量巨大。优点：① 传输容量大——一根光纤可同时传输数百万路电话；② 损耗小——光在光纤中传播损耗极低，中继距离长；③ 抗电磁干扰——光纤是绝缘材料，不受电磁干扰；④ 保密性好——光信号在光纤内部传输，不易被窃听。
🏆 得分点：答出全反射原理；至少说出三个优点（容量大、损耗小、抗干扰、保密性好中的三个）。

挑战 ★★★
手机既能发射电磁波也能接收电磁波，它工作时能量如何转化？手机的电磁波是否对人体有害？谈谈你的看法。
解析：手机工作时电能转化为电磁波的能量（发射），同时接收到的电磁波能量转化为电信号（接收），最终电信号转化为声信号（通过听筒）。关于电磁波对人体是否有害：手机发射的是微波（频率约 $0.8$–$2.4\ \text{GHz}$），属于非电离辐射，与 X 射线、$\gamma$ 射线等电离辐射不同，不会直接破坏分子结构。但长时间、近距离使用是否产生热效应或其他健康影响，科学界尚无定论。建议尽量使用耳机、减少通话时间以降低辐射暴露。
🏆 得分点：说出手机工作时电能与电磁波能量的相互转化；能从非电离辐射的角度理性分析；提出合理的防护建议。
挑战 ★★★
同步通信卫星相对地面静止，它绕地球转动的周期是多少？三颗同步卫星能否覆盖全球？
解析：同步通信卫星绕地球转动的周期与地球自转周期相同，即 $T = 24\ \text{h}$（更精确为 $23\ \text{h}\ 56\ \text{min}$，即一个恒星日）。卫星位于地球赤道上空约 $35\,800\ \text{km}$ 的轨道上，从地面看仿佛静止不动。三颗同步卫星等间隔布放在赤道上空（经度相差 $120^\circ$），其信号覆盖范围可基本实现全球通信（两极附近会有盲区）。这就是为什么全球卫星通信系统通常使用至少三颗同步卫星的原因。
🏆 得分点：答出周期为 $24$ 小时（与地球自转周期相同）；答出三颗卫星可基本覆盖全球；能简要说明原理。

麦克斯韦的预言（1865）：英国物理学家麦克斯韦用数学推导预言了电磁波的存在——变化的电场产生磁场，变化的磁场产生电场，二者交替传播形成电磁波。他计算出电磁波的速度正好等于光速！20多年后赫兹用实验证实了电磁波的存在。今天，WiFi、手机、蓝牙都在使用电磁波。
