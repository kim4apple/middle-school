Starter Unit 2

What's this in English?

字母 I–R · 日常物品 · 询问物品名称

📖 话题：Classroom items 💬 功能：Ask about things 📚 语法：字母 I–R · 不定冠词 a/an 📝 写作：描述物品

🎯 我能做到

📖 课文精讲 · In the Classroom
场景：老师在课堂上拿出各种物品，让学生用英语说出名称。
Teacher: What's this in English?
Li Ming: It's a map.
Teacher: And what's that?
Wang Fang: It's an orange.
Teacher: Good! Spell it, please.
Wang Fang: O-R-A-N-G-E.
📖 查看翻译

逐句翻译：

老师：用英语说这个是什么？

李明：它是一张地图。

老师：那个呢？

王芳：它是一个橙子。

老师：很好！请拼写它。

王芳：O-R-A-N-G-E。

🔑 重点讲解：

1. What's this in English? → "这个用英语怎么说？" What's = What is。

2. 回答用 It's a / an + 物品。辅音音素开头的词用 a，元音音素开头的词用 an。

3. Spell it, please. → "请拼写它。" 用大写字母拼读每个字母。

4. this 指"这个"（近处），that 指"那个"（远处）。

🔤 重点句型 · Asking about Things
What's this / that in English?
询问物品的英文名称。
✅ What's this in English? — It's a pen.
✅ What's that in English? — It's an apple.
➡ 问近处的物品用 this，远处的用 that。
It's a / an + 物品名
用 a 或 an 回答。规则：元音音素开头的词用 an，辅音音素用 a。
✅ a map / a pen / a book
✅ an orange / an apple / an egg
➡ 看 发音，不是看字母！
Spell it, please.
请对方拼写单词。
✅ Spell it, please.
➡ B-O-O-K.
➡ 用大写字母读出每个字母。
Is this / that a / an ...?
一般疑问句，确认物品。
✅ Is this a book?
➡ Yes, it is. / No, it isn't.

2️⃣ 不定冠词：a / an

a 和 an 都表示"一个 / 一件 / 一把……"，放在单数可数名词前面。

冠词用法例子
a辅音音素开头的词a pen 一支钢笔
a map 一张地图
a book 一本书
an元音音素开头的词an orange 一个橙子
an apple 一个苹果
an egg 一个鸡蛋

📌 注意：看 发音 而不是字母！如 an hour（h 不发音），a university（发 /juː/ 是辅音）。

⚠️ 中式英语纠错：
❌ This is orange.（一个橙子前需要冠词 a/an）
✅ This is an orange.

❌ It's a apple.（apple 开头是元音 /æ/，应该用 an）
✅ It's an apple.

3️⃣ This / That 对比

thisthat
距离近处（这里/眼前）远处（那边/远处）
例句This is my desk.That is your bag.
📝 三级闯关 · 实战练习

🥉 基础关 · 概念识记

基础 ★☆☆
— What's this in English?
— ____ a book.
📝 查看答案
答案：It's
解析：回答 What's this? 用 It's a/an ...。It's = It is。
基础 ★☆☆
选出正确冠词：____ apple (a / an)
📝 查看答案
答案：an
解析：apple 开头发音是元音 /æ/，所以用 an。
基础 ★☆☆
字母 N 的小写是 ____。
📝 查看答案
答案：n
解析：N 的小写是 n。

🥈 提升关 · 语境运用

提升 ★★☆
— Is this ____ pen?
— Yes, it is. It's ____ pen. (a / an)
📝 查看答案
答案：a; a
解析：pen 开头是辅音 /p/，用 a。两空都是 a pen。
提升 ★★☆
连词成句：this / what's / English / in / ?
📝 查看答案
答案：What's this in English?
解析：What's 开头，注意 in English 为固定搭配，放在 this 后面。
提升 ★★☆
— Spell it, please.
— ____. (book)
📝 查看答案
答案：B-O-O-K
解析：拼写时用大写字母，字母之间加连字符。
挑战 ★★★
改错：— What's that? — It's a orange.
📝 查看答案
答案：— What's that? — It's an orange.
解析：orange 开头是元音 /ɔː/，应该用 an 而不是 a。

✍️ 微写作：描述物品

用英语写出你书包里的 3 件物品。用 What's this? / It's a / an ... 的句式。

It's a pencil case.

It's an eraser.

It's a ruler.

我书包里有什么？它是一个铅笔盒。是一块橡皮。是一把尺子。
